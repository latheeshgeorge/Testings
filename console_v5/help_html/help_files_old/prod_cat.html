Product Categories
Product categories for the sites can be managed from this section.
List Product Categories
Product category groups can be searched using the following options
1. Category Name
2. Parent Category
3. Category Group
Combination of the above 3 can be used to search for categories existing in the site.
Use "Add" link to go the product category group add page. There are 2 ways to go to the edit page of an existing product category group. One method is by clicking the name of rhe product category group and another by tick marking the required category group and pressing the "Edit" icon.
Delete operation can be done to more than one product category groups at a time. Tick mark the product category groups to be deleted and press the "Delete" icon.
Additional options that can be used with the listed categories can be obtained by click on the link "Options". The additional options listed are
1. Change Parent to
This options allows to changes the parent of selected categories to the one set in the dropd own box. Select the categories, select the parent and then click the "Change" button to change the parent.
2. Change hide status to
If the hidden status of categories are to be changed, this option can be used. Select the required categories, select the required status from the drop down and click the "Change" button.
3. Assign to Category Group
To assign one or more categories to a category group at once, this option can be used. Select the categories to be assigned, select the category group to which the categories to be assigned, from the drop down box, and click the "Assign" button. This will mapped the selected categories to the category group selected in the dropdown box. If any of the selected categories already in the selected group then no actions will be done for those categories. Which means one category will be mapped only once with a category group.
4. Unassign from Category Group
To unassign one or more categories from a category group at once, then this option can be used. Select the categories to be unassigned, select the category group from which the categories to be unassigned, from the drop down box, and click the "Unassign" button. This will unassign the selected categories from the category group selected in the drop down box if any mapping exists.
Add Product Categories
Product category groups can be added through this section.
Key Fields
1. Parent Category
The parent category for the current category can be selected from this drop down. By default "-- Root Level -- " will be shown in this drop down box, which mean there will not be any categories to the top of current category in hierarchy.
2. Category Group Selection
The category groups to which the current category is to be mapped with can be done by selecting the required category groups from this list box. Use "Ctrl" button to make multiple selection.
3. Long Description
The long description (if specified) will be displayed in the category details page. Html tags are supported by this section.
4. Display Settings for Subcategories & Products of this category
This section allows to set the display settings options for subcategories and products of current category.
a. Subcategory List
Allows to set where the subcategories of current category (if any) should be displayed. The options available are
i. Show in Middle Area
Shows the subcategories only in the middle area.
ii. Show below Selected Category
Shows the first level subcategories of current category below the current category itself. Subcategories in levels other than the first level will be shown in the middle area only.
iii. Both in middle and in below selected category
Show the subcategories in both middle area as well as below the current category. While viewing the subcategories in levels other than the first level, they will be displayed in the middle area only.
b. Product Display Method
This option helps to set how the products should be displayed in the category details page. The options shown in the dropdown box depends on the type of layout used by the site
c. Product List
This options allows to set where the products should be displayed while viewing the category details page. The options shown in the drop down box depends on the type of the layout used by the site.
d. Fields to be displayed for products
The fields to be included while displaying the products in category details page can be set from here. The fields that are to displayed should be ticked. It is compulsory that atleast one of the field should be ticked. The options available are
1. Product Image
2. Product Title
3. Product Description
4. Product Price

5. Sub Categories
Subcategories of current category will be displayed in this section. Click on "+" icon to view the subcategory list. Click on "-" to hide the subcategory list.
Use "Assign More" in this section to assign more subcategories.
Use "Unassign" to unassign selecte categories
Use "Save Order" to save the sort order for subcategories. To save new sort order, tick mark the selected categories, enter the required sort order value and click the "Save Order" button.
6. Category Images
Images can be assigned to categories from this section. It is possible to assign more than one image to a category.
Use "Assign More" buttom to assign more images. When clicked on this button a page will be shown with the list of images existing in the site. Select the required images and click the "Assign" button.
Images assigned to the category will be listed in this section. It can be viewed when clicked on the "+" icon. Use "-" icon to hide the mapped images.
Sort order and the title can be set for the images assigned to the category. To save these details, tick the images for which the details to be saved and then click the "Save Details" button.
If the option "Show any product image" option is ticked, then the images assigned to the category will be ignored and the system will pick an image from any one of the product assigned to current category.
