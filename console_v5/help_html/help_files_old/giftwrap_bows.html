Giftwrap Bows
Gift wrap bows section allows to define the details of bows that will be used for wrapping the gifts
List Giftwrap Bows
Bows can be searched by giving the name of the bow. Number of bows in each of the page and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the bow add page. There are 2 ways to go to the edit page of an existing bow. One method is by clicking the name of the bow and another by tick marking the required bow and pressing the "Edit" icon.
Delete operation can be done to more than one bows at a time. Tick mark the bows to be deleted and press the "Delete" icon.
Status of bows can be changed directly from the listing page. Tick mark the bows which need status changing. Select the required status from the drop down box provided at the section "Change Status" and press the "Change" button.
Order in which the bows should be displayed in the cart page can be set by giving sort order values in textbox given for each of the bows in the list. Click the "Save Order" button to save the sort order.
Add Giftwrap Bows
Giftwrap Bows can be added through this section.
Key Fields
1. Bow name
The name that will be show for the bow while displayed in the cart page
2. Extra Price
If extra payment have to be made for a bow, it should be specified in this section.
Edit Giftwrap Bows
Giftwrap bows details can be edited through this section.
Key Fields
1. Bow name
The name that will be show for the bow while displayed in the cart page
2. Extra Price
If extra payment have to be made for a bow, it should be specified in this section.
3. Bow Images
This section allows to map the images for the bow. Use "Assign More" button in this section to go to the image gallery section to assign the image(s) for the bow. Once the images are assigned, option to set the order in which the images will be displayed to the customer and also the title for the images will be displayed. Specify these details and press the "Save Details" button to save the changes. One thing to be noted is that, the image details will be saved only for those images for which the checkbox is ticked.
Use "Unassign" button to unassign any assigned image(s) set for the bow.
