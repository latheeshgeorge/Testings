Static Pages
Static Pages of the site can be managed from this section
List Static Pages
Static pages can be searched by giving the title. Number of static pages in each of the page and also the sort order to be used can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the static page add page. There are 2 ways to go to the edit page of an existing static page. One method is by clicking the name of the page and another by tick marking the required static page and pressing the "Edit" icon.
Delete operation can be done to more than one static pages at a time. Tick mark the static pages to be deleted and press the "Delete" icon.
Status of static pages can be changed directly from the listing page. Tick mark the page which need status changing. Select the required status from the drop down box provided at the section "Change Hide Status" and press the "Change" button.
Add Static Pages
Static pages can be added through this section.
Key Fields
1. Page Groups
The list of static page groups added to the site will be displayed in this list box. User should select the static page groups to which the current static page will be linked to. Use Ctrl button to make multiple selections.
2. Page Type
Console user can decide whether the static page represents a content from the site itself or should be a link to some other page in some other area. The page type field allows to make this decision. If "Page" is selected, then the option to specify the content for the page will be displayed to the user. If the option "Link" is selected, then a new option named "Page Link" will be displayed. In this textbox, user can specify the link to the page where the static page is pointing. Another option to decide whether the link should be open in a new window or in same window also will be displayed if the option "Link" is selected.
