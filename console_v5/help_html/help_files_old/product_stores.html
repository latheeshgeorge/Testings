Product Stores
Product Stores section allows to manage the stores for the sites existing in various places. Once the store is created the products which are to be maintained in the stores and also the stock to be kept in various stores can be set from the product edit page in console. While creating a console user the product stores created through this section will be shown in a drop down box. This is done to create console users specifically for a particular product store.
List Product Stores
The product stores created by console users will be listed in this section. Stores can be searched by giving the store name. Number of stores to be shown in in each of the page and also the sort order to be used can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the product store add page. There are 2 ways to go to the edit page of an existing product store. One method is by clicking the name of the store and another by tick marking the required stores and pressing the "Edit" icon.
Delete operation can be done to more than one stores at a time. Tick mark the stores to be deleted and press the "Delete" icon.
Status of product stores can be changed directly from the listing page. Tick mark the stores which need status changing. Select the required status from the drop down box provided at the section "Change Hidden Status" and press the "Change" button.
Sort order for stores can be set from the listing page itself by specifying the required sort order in the textboxes provided with each of the prodcut stores in the list. Once the order is set for all the required stores, press the "Save Order" button to save the sort orders.
Add/Edit Product Stores
New product stores can be created through this section.
Key Fields
1. Store name
This is the name by which the store will be identified through out the console area.
2. Contact Person
This field helps to identify the person who manages the stores. This is just for informative purpose.
3. Active Store
Product store can be made active or inactive using this option.
4. Order
This option allows to specify the sort order in which the stores should be displayed in various sections in console area.
