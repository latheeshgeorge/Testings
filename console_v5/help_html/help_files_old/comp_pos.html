Edit website layout
Edit website layout section allows to manipulate the positions of various features in the site. The position can be set for featuresby dragging and droping them to required positions. The sections in website layout page may differ a bit with respect to the template used in the site. In order to get a better understanding of this section, the concept of templates and layout should be clear. Bshop4 is packed with a number of predefined templates which can be used for various sites. Templates decides the color theme for the site and also the layouts for the site. Layouts are the diffent types of arrangements. That is, the layout decides the way in which the features in the site will be displayed while viewing different pages. One template can have more than one layout. In such a case different pages in the site will be displayed in a different arrangement to the customer. Using the Website layout edit section we are giving the console user the flexibility to arrange the features existing in the site in required positions in different layouts.
Key fields
1. Select Layout
The select layout is a drop down box which show the layouts for the template used by the site. By default the first layout will remain selected in this drop down box. If there are more than one layout for a site, then selecting a layout in this drop down box will reload the page and will show the features set for that layout. The right hand side of the page will show a graphical representation of the arrangements of various components in the layout selected in the "Select Layout " dropdown box. The positions in graphical representation will be different for different layouts for same template and will be different for different templates.
2. Available Components
This is the area where all the components will be listed. Console user can drag and drop the components from this section to the required position.
In the graphical representation of components, each component is shown as a rectangular box with its name embedded in it. Various positions to which the components can be assigned are marked by a border line. To the top left side of each of such boxes, the name of position is displayed in the format "Position: [position name]", where [position name] will be values like left, right, top etc. The position name may vary for different template
In order to position a component at a particular location, drag and drop the component to the required position from the "Available Components" section. When a component is dragged and dropped from the "Available Component" section to any of the positons, that component will be removed form the "Available Component" section and will only be visible in the position to which it has been dragged and droppped. Once the components are set to required positions, click on "Save Positions" to save the changes. When clicked on "Save Positions" button the page will get reloaded and the section "Available Components" will again show all the components, including the one's which have been dragged and dropped to various positions. So if a component is to be assigned to more than one position, then first drag and drop the component to one of the positions, then save the changes and again drag and drop the same component to the second position and so on.
In addition to dragging and dropping components from "Available Components" section to various positons, components can be dragged and dropped between various positions also. That is from position "top" to "botom", "left to right" etc.
The components set to various positions will be displayed in the order in which the rectangular boxes are placed in each of the positions. The sort order should be set by dragging and dropping the rectangular boxes within each position.
If the headings displayed for each of the components in various positions are to be different, then this can be done by clicking on the icon provided to the top of each of the position. Since the edit heading icon is provided in each of the positions, it is possible to set different headings for same components in different positions.
In component title editing section, the following key columns are displayed
1. Component Name
This is the exact name of components
2. Show in site as
This will be a text box and will shown the heading which will be displayed in the site.
3. Original Name
This is the original name for the component.
The original name and component name will be same for some of the components, for e.g. "Best Sellers". But this may be different for certain components like adverts, survey, combo deal etc. For e.g. in case of adverts, the component name will be "Adverts" and the original name will be the actual name given to advert by the console user while it has been created.
Always the "Original Nanme" of component will be displayed withIn the rectangular representation of components and when mouse over each of the rectangular boxes the "Component Name" will be displayed. This will help user to identify each of the components in various sections in the graphincal representation of components.
