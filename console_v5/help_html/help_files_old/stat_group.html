Static Page Groups
Static Page groups holds the static page to be shown in various sections of the site. Same static page same be mapped with more than one static page groups.
List Static Page Groups
Static page groups can be searched by giving the name of the group. Number of groups in each of the page and also the sort order to be used can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the static page group add page. There are 2 ways to go to the edit page of an existing static page group. One method is by clicking the name of the group and another by tick marking the required static page group and pressing the "Edit" icon.
Delete operation can be done to more than one groups at a time. Tick mark the groups to be deleted and press the "Delete" icon.
Status of static page groups can be changed directly from the listing page. Tick mark the group which need status changing. Select the required status from the drop down box provided at the section "Change Hide Status" and press the "Change" button.
The field "Layout(position)(order)" in the group lising is shown as a drop down box. This is because same group can be set to show in more than one layout/position. This drop down box is just for an informative purpose. No operation is associated with this drop down box.
Add Static Page Groups
Static page groups can be added through this section.
Key Fields
1. Hide name
This field decides whether the name of the group is to be displayed to the customer. If ticked, then the name will be hidden in the site. When groups are shown in the top or bottom, the group name will be hidden always irrespective of this setting.
2. Display Location
This field allows to select the areas in which the group is to be displayed. The list box shows the list of layouts and its positions to which the static page groups can be assigned. The values in this list box have the format "Layout Name (position)". There can be more than one layout for a site. "Layout name" refers to the layouts existing for the site. "Position" refers to the positions to which static page groups can be set for the site (eg. right, left, etc). It is possible to show same group in more than one layout/position. Use "ctrl" button to make multiple selection in this list box.
3. Show in all Pages
Groups can be set to show in all pages or only for selected product or categories or static pages. From the "Add" page it is possible only to specify whether it is to be shown for all pages or not. Mapping to certain product or categories or static pages can be done from the static page group edit page.
4. Show home page link
If the home link is to be included in the static page group this option should be selected.
5. Show sitemap link
If the sitemap link is to be included in the static page group this option should be selected.
6. Show help link
If the helpl link is to be included in the static page group this option should be selected.
7. Show saved search link
If the saved search link is to be included in the static page group this option should be selected.
Edit Static PageGroups
Static page group details can be edited through this section.
Key Fields
1. Hide name
This field decides whether the name of the group is to be displayed to the customer. If ticked, then the name will be hidden in the site. When groups are shown in the top or bottom, the group name will be hidden always irrespective of this setting.
2. Display Location
This field allows to select the areas in which the group is to be displayed. The list box show the list of layouts and its positions to which the static page groups can be assigned. The values in this list box have the format "Layout Name (position)(order)". There can be more than one layout for a site. "Layout name" refers to the layouts existing for the site. "Position" refers to the positions to which groups can be set for the site (eg. right, left, etc). "Order" is the sort order in which the group will be listed in the site. The order is important if more than one component is placed in same position. Components set to each of the positions are displayed in the site in the sort order shown in "Order". It is possible to set the same group to more than one area. Use "ctrl" button to make multiple selection in this list box. By default the value of "Order" will be set to 0. It will be changes automatically when the position of each of the components are set from the "Component Position" section in the site console.
3. Show in all Pages
Groups can be set to show in all pages or only for selected product or categories or static pages. From the "Add" page it is possible only to specify whether it is to be shown for all pages or not. Mapping to certain product or categories or static pages can be done from the static page group edit page.
4. Show home page link
If the home link is to be included in the static page group this option should be selected.
5. Show sitemap link
If the sitemap link is to be included in the static page group this option should be selected.
6. Show help link
If the helpl link is to be included in the static page group this option should be selected.
7. Show saved search link
If the saved search link is to be included in the static page group this option should be selected.
8. Pages in Group
Static pages which are mapped to the static page group being editing will be show in the area. To view the list, click on the "+" icon in this section. When the list is displayed, the "+" sign will be replaced with "-" sign. Click "-" to hide the section.
To assign more static page to the group, click on "Assign more" button under this section. This will open up a page which shows the list of static pages which are not yet mapped to the current group. Tick the static pages to be mapped and click the "Done" button to make the assignment.
To unassign static pages from the group, tick the static pages to be unassigned and click the "Unassign" button in this section.
While listing the static pages mapped with the group, it is possible to set the order in which the static pages will be displayedin the group. Click the "Save Details" button to save the changes
6. Categories for which this static page group will be displayed
If the group is to be shown while viewing the details page of particular category, then those categories should be set from this section. This setting will be ignored if the "Show in all pages" option is set for the group. Click "+" icon to view the products mapped to this section and click on "-" to hide the list.
Use "Assign More" button to assign more categories to this section
Categories assigned to this section can be unassigned by tick marking the required categories and clicked the "Unassign" button in this section.
7. Products for which this static page group will be displayed
If the static page group is to be shown while viewing the details page of particular product, then those products should be set from this section. This setting will be ignored if the "Show in all pages" option is set for the group. Click "+" icon to view the products mapped to this section and click on "-" to hide the list.
Use "Assign More" button to assign more products to this section
Products assigned to this section can be unassigned by tick marking the required categories and clicked the "Unassign" button in this section.
8. Static Pages for which this static page group will be displayed
If the static page group is to be shown while viewing particular static pages, then those static pages should be set from this section. This setting will be ignored if the "Show in all pages" option is set for the group. Click "+" icon to view the static pages mapped to this section. and click on "-" to hide the list.
Use "Assign More" button to assign more static pages to this section
Static Pages assigned to this section can be unassigned by tick marking the required static pages and clicked the "Unassign" button in this section.
