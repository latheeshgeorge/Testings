Product Category Groups
Product category groups holds the categories to be shown in various sections of the site. Same category same be mapped with more than one product category groups.
List Product Category Groups
Product category groups can be searched by giving the name of the category group. Number of categories in each of the page and also the sort order to be used can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the product category group add page. There are 2 ways to go to the edit page of an existing product category group. One method is by clicking the name of rhe product category group and another by tick marking the required category group and pressing the "Edit" icon.
Delete operation can be done to more than one product category groups at a time. Tick mark the product category groups to be deleted and press the "Delete" icon.
Status of product category groups can be changed directly from the listing page. Tick mark the product category groups which need status changing. Select the required status from the drop down box provided at the section "Change Hide Status" and press the "Change" button.
The field "Layout(position)(order)" in the group lising is shown as a drop down box. This is because same group can be set to show in more than one layout/position. This drop down box is just for an informative purpose. No operation is associated with this drop down box.
Add Product Category Groups
Product category groups can be added through this section.
Key Fields
1. Hide Group name
This field decides whether the name of the product category group is to be displayed to the customer. If ticked, then the name will be hidden in the site. When category groups are shown in the top or bottom, the group name will be hidden always irrespective of this setting.
2. Display Location
This field allows to select the areas in which the group is to be displayed. The list box shows the list of layouts and its positions to which the product category groups can be assigned. The values in this list box have the format "Layout Name (position)". There can be more than one layout for a site. "Layout name" refers to the layouts existing for the site. "Position" refers to the positions to which category groups can be set for the site (eg. right, left, etc). It is possible to show same product category group in more than one layout/position. Use "ctrl" button to make multiple selection in this list box.
3. Group List Type
Product category groups can be displayed in 2 formats. First is as "Menu" and the second is as "Drop Down". If the "Menu" Option is selected, then all the categories assigned to the category group will be displayed one after the other. If the "Drop Down" options is selected, then the assigned categories will be shown in a selectable drop down box.
4. Show in all Pages
Product category groups can be set to show in all pages or only for selected product or categories or static pages. From the "Add" page it is possible only to specify whether it is to be shown for all pages or not. Mapping to certain product or categories or static pages can be done from the product category edit page.
Edit Product Category Groups
Product category group details can be edited through this section.
Key Fields
1. Hide Group name
This field decides whether the name of the product category group is to be displayed to the customer. If ticked, then the name will be hidden in the site. When category groups are shown in the top or bottom, the group name will be hidden always irrespective of this setting.
2. Display Location
This field allows to select the areas in which the group is to be displayed. The list box show the list of layouts and its positions to which the product category groups can be assigned. The values in this list box have the format "Layout Name (position)(order)". There can be more than one layout for a site. "Layout name" refers to the layouts existing for the site. "Position" refers to the positions to which category groups can be set for the site (eg. right, left, etc). "Order" is the sort order in which the category group will be listed in the site. The order is important if more than one component is placed in same position. Components set to each of the positions are displayed in the site in the sort order shown in "Order". It is possible to set the same product category group to more than one area. Use "ctrl" button to make multiple selection in this list box. By default the value of "Order" will be set to 0. It will be changed automatically when the position of each of the components are set from the "Component Position" section in the site console.
3. Group List Type
Product category groups can be displayed in 2 formats. First is as "Menu" and the second is as "Drop Down". If the "Menu" Option is selected, then all the categories assigned to the category group will be displayed one after the other. If the "Drop Down" options is selected, then the assigned categories will be shown in a selectable drop down box.
4. Show in all Pages
Product category groups can be set to show in all pages or only for selected product or categories or static pages. From the "Add" page it is possible only to specify whether it is to be shown for all pages or not. Mapping to certain product or categories or static pages can be done from the product category edit page.
5. Categories under this Category Group
Categories which are mapped to the product category group being editing will be show in the area. To view the list, click on the "+" icon in this section. When the list is displayed, the "+" sign will be replaced with "-" sign. Click "-" to hide the section.
To assign more categories to the group, click on "Assign more" button under this section. This will open up a page which shows the list of categories which are not yet mapped to the current group. Tick the categories to be mapped and click the "Done" button to make the assignment.
To unassign categories from the group, tick the categories to be unassigned and click the "Unassign" button in this section.
While listing the categories mapped with the group, it is possible to set the following things
a. Order
The order in which the categories are to be displayed in the group
b. Display Type
While displaying categories it is possible to show it as a normal category or as a heading. This option allows to decide this. By default "Normal" will be selected.
c. Link Req
This section allows to set whether the categories listed in the category group should be a link or not. By default "Yes" will be selected.
d. Hidden
The hidden status of the mapped category will be shown in this section.
In order to save the changes done to the above things, click the "Save Details" button.
6. Categories for which this category group will be displayed
If the group is to be shown while viewing the details page of particular category, then those categories should be set from this section. This setting will be ignored if the "Show in all pages" option is set for the group. Click "+" icon to view the products mapped to this section and click on "-" to hide the list.
Use "Assign More" button to assign more categories to this section
Categories assigned to this section can be unassigned by tick marking the required categories and clicked the "Unassign" button in this section.
7. Products for which this category group will be displayed
If the category group is to be shown while viewing the details page of particular product, then those products should be set from this section. This setting will be ignored if the "Show in all pages" option is set for the group. Click "+" icon to view the products mapped to this section and click on "-" to hide the list.
Use "Assign More" button to assign more products to this section
Products assigned to this section can be unassigned by tick marking the required categories and clicked the "Unassign" button in this section.
8. Static Pages for which this category group will be displayed
If the category group is to be shown while viewing particular static pages, then those static pages should be set from this section. This setting will be ignored if the "Show in all pages" option is set for the group. Click "+" icon to view the static pages mapped to this section. and click on "-" to hide the list.
Use "Assign More" button to assign more static pages to this section
Static Pages assigned to this section can be unassigned by tick marking the required static pages and clicked the "Unassign" button in this section.
