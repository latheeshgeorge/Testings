Product Labels
Product labels can be used to define the additional characteristics for products managed in the site. The labels created in this section will be displayed in the product add/ edit page to specify the values for those labels for the product. In product add/edit page, specify values only for those labels which are relevant for the product. A product label will be displayed in the product details page in site only if value is set for that label for that particular product.
List Product Labels
Product labels can be searched by giving the label name. Number of product labels to be show in each of the page and also the sort order to be used can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the product labels add page. There are 2 ways to go to the edit page of an existing product labels. One method is by clicking the name of the label and another by tick marking the required product labels and pressing the "Edit" icon.
Delete operation can be done to more than one product labels at a time. Tick mark the product labels to be deleted and press the "Delete" icon.
Status of product labels can be changed directly from the listing page. Tick mark the label which need status changing. Select the required status from the drop down box provided at the section "Change Status" and press the "Change" button.
Add/Edit Product Labels
Product labels can be added or edited through this section.
Key Fields
1. Label Name
This is the name of the label. This is the caption which will be displayed when labels are show in the product add/edit page and also in the product details page in the site.
2. Show in Search
This field decides whether current label is the shown in the advanced search page.
3. Type
The product labels can be of 2 types
1. Textbox
2. Drop down box
If the option drop down box is selected, then additional textboxes to specify the values to be shown in the dropdown box also will be shown in the page.
