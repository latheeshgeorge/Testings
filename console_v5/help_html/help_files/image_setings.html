Image On/Off
This section shows the checkboxes which decides whether the images are to displayed in various pages in website.
1. Display image for each product item on the view cart page
If product images are to be displayed in the view cart page, then this checkbox should be ticked.
2. Display image for each product item on the wishlist page
If product images are to be displayed in the wishlist page, then this checkbox should be ticked.
3. Display image for each product item on the enquiry page
If product images are to be displayed in the product enquiry page, then this checkbox should be ticked.
4. Turn off the category image
If category image is to be made hidden through out the site, then this checkbox should be ticked.

Managing SSL Images
Images to be displayed in the ssl section of your website can be managed from this section. The options displayed in this section will depend on the payment types and payment methods supported by your website.
The options that will be available with each of the block in this section are the following
1. Image
The options available are "Default" and "Custom". If the default option is selected, the default image set for the payment type of payment method will be displayed in the website. If "Custom" option is selected, then a button will be displayed which allows to select the image to be used for the payment method.
2. Assign / Change
This button will be visible only if the "Custom" option is selected. When clicked on this button, user will be taken to the image gallery section to assign the required image. The assigned image will be displayed under this button, when returned to this page. Same button can be used to assign a different image. Only one image can be assigned, that is when an image is assigned, the previously assigned image (if any) will be ignored.
Click on "Save Settings" button to save the changes made in this page.
