How Order Confirmation Email Ids Works?
When customer places an order in the website, it is possible to send an order confirmation / notification email to administrators. If this is to be done, the email ids to which the notification mail is to be send should be set from this section. The template that will be used to send this notification mail can be managed from the email template section in the website console. The mail will be send only if email template is not disabled.
It is possible to set a maximum of 5 email ids to where the notifications are to be send.
Enter the email ids in the textboxes and click on "Submit" button to save the details
