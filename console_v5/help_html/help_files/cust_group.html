How Customer Newsletter Groups Works?
Customer newsletter groups can be managed from this section. Customer newsletter groups will be listed in the newsletter subscription section in the website. Newsletter subscribers will be mapped under the newsletter groups. When customers get registered or when they edit their profile, the list of customer groups will be displayed as checkboxes to allow registered customers to subscribe for newsletters.

List Customer Newsletter Groups
Customer Newsletter groups can be searched based on group name. Number of groups to be shown in each of the pages and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the customer newsletter group add page. There are 2 ways to go to the edit page of an existing customer newsletter groups. One method is by clicking the group name and another by tick marking the required group and clicking the "Edit" icon.
Delete operation can be done to more than one customer newsletter groups at a time. Tick mark the groups to be deleted and press the "Delete" icon.
Hidden Status of groups can be changed directly from the listing page. Tick mark the groups which need status changing. Select the required status from the drop down box provided at the section "Change Status" and press the "Change" button.

Key Fields
1. Group Name
Customer newsletter group name can be set in this textfield.
2. Active
This option allows to decide whether the current customer newsletter groups is active in the website. Only active groups will be listed in the website.

Tab: Customers Assigned to this group
Customer who are linked with the current newsletter group will be listed in this section. Click on "+" icon to view the list of customers.
Use "Assign More" button to go to the page where the list of customers will be displayed. Select the required customers and click on "Assign Selected" button to make the assignment of those customers to the current group..
Customers assigned to current newsletter group can be unassigned by selecting the customers in the list and clicking on "Un assign" button.
