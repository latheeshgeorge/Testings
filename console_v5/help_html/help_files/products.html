List Products
Products created by console users will be listed in this section.
Basic criteria that can be used to search for products are the following.
1. Product Name
2. Category
3. Product Id
4. Vendor
5. Bulk Discount
Additional options that can be used to search for products can be obtained by clicking on the "Options" link.
The available options are
1. Retail Price Between
2. Cost Price Between
3. Discount Atleast
4. Total Stock Atleast
5. In Preorder
6. Product Notification Required?
7. Allow ordering even if out of stock?
8. Hidden
Any combination of above said fields can be used in the search criteria. Number of products to be displayed in each of the pages and also the sort order for the product can be selected. Click on "Go" button to begin the search operation.
A facility to set various options for multiple products also can be managed from this section. This can be done by clicking on the "Settings for Multiple Products" link.
Use "Add" link to go the product add page. There are 2 ways to go to the edit page of an existing product. One method is by clicking the product name and another by tick marking the required product and clicking the "Edit" icon.
Delete operation can be done to more than one products at a time. Tick mark the products to be deleted and click the "Delete" icon.
Hidden status of products can also be changed directly from the listing page. Tick mark the products which need status changing. Select the required status from the drop down box provided at the section "Change Hidden Status" and press the "Change" button.
Options to perform more actions on products can be obtained when clicked on the "More Actions" link. When clicked on this link the following options will be displayed.
Back to Top
1. Assign to Category
This option allows to assign multiple products to a selected category. Tick mark the products, select the category to which the selected products are to be assigned in the drop down box displayed at the section "Assign to Category" and then click on the "Assign" button. When this is done, the selected products, which are not already assigned to the selected category, will be mapped with the selected category.
2. Un assign From Category.
This option allows to unassign multiple products from a selected category. Tick mark the products, select the category from which the selected products are to be un assigned in the drop down box displayed at the section "Un assign From Category" and then click on the "UnAssign" button. When this is done, the selected products, which are already assigned to the selected category, will be unassigned from it.
If the import / export feature is active for your site, then option to export products will be displayed under the list of product. There are 2 types of product export
1. Export Selected Products
If this option is selected, the tick marked products will be exported.
2. Export All Products
If this option is selected, all products will be exported.
When clicked on the "Export Now" button, you will be taken to the main import / export section and there you will be able to select the following things related to products being exported
1. Fields to be included in the exported file.
2. How the products should be sorted in the exported file
3. Output Format of export file. The available format as CSV, HTML and SQL Dump.

Add Product
New products can be created through this section.
Key Fields
1. Product Type
This option allows to define whether the current product is a normal product or a downloadable product. If the type is downloadable, then additional options will be displayed in the product edit page to upload the downloadable files. When downloadable products are ordered by a customer, the files set in downloadable section for the products will be displayed in the My Downloads section for the customer. Another thing to be noted here is that, for sites which manage downloadable products, the option to force the customers to login inorder checkout should be activated from the general settings area in consoleshould be forced to login before checkout (Settings » General Settings » Main Shop Settings » Administration Area tab).
One more thing to be noted is that, the variables option will not be available for downloadable products.
2. Product Name
The name for the product can be set in this textbox.
3. Barcode
This is the unique code which will be used to identify the product. Specify the barcode here only if bar codes are not planning to give for variable combinations. The option to set barcodes for various variable combinations for the product will be available in the stock management tab.
4. Product ID
The manufacture id for the product can be set in this textbox. This is an optional field.
5. Fixed Stock
If variable stock is not managed for the product, then the web stock value can be set in this textbox. Option to manage stock will be available in the stock tab in the product edit page.
6. Model
Product model details can be set in this textbox. This is an options field. There is option to set to show the product model as a search criteria in advanced search page in the website. So if product model is set for a product, that value will be displayed in this dropdown box in the advanced search page in the website.
7. Hide
If the product is to be made hidden in the website, this option should be set to Yes.
8. Short Description
Short description for the product can be set in this textbox. It is a good practise to set the key information about the product in short description. There are options to show short description in various product listing sections in the website. Depending on the template used by the site this may change.
9. Long Description
A detailed description for the current product can be set in this html editor. This will be displayed in the product details page in the website. If the current product is set as the featured product, then by default this description will be displayed as the featured product description.
Note: Additional description tabs to be displayed in the product details page of website can be set from the Description tab section in the product edit page.
10. Product Keywords
This is a special field which can be used to specify the various keywords related to current product. When customer search for products from the website, the value set to this field for each of the products will also be searched. This increase the possiblility to display the product in the search result. This is an optional field.
11. Additional Details Section
This section will be visible only if product labels are set for the product. Product labels can be set from Settings » Product Settings » Product Labels section in the console. All labels set for the current site will be listed in this section. Depending on the type of label, it may a dropdown box or a textbox. Set of select value for the labels if they are relevant for the current product. otherwise leave it blank. If value is set for any of the additional details for this product, then it will displayed in the product details page in the website.
12. Categories
This multiselect box allows to select the categories to which the current product will be mapped to. Use ctrl button to select multiple categories.
13. Default Category
If multiple categories are selected for product, then one of then should be selected as the default category. The category selected as default category will be used to generate the tree menu at the top of product details page in the website.
Note: If current product is mapped with only one category, then category should be selected in both "Categories" and "Default Category" section
14. Vendors
The list of vendors added to the site will be listed in this multi select list box. Use ctrl button to select multiple vendors. Vendors for the site can be managed from the Settings » Product Settings » Product Vendors section in the console.
15. Discount Type
The discount to be allowed to the product can be applied from this section. Discount can be specified in 3 ways such as %, Discount Value or Exact Discounted Price. The values available in the dropdown box are the following
a. %
Use this option in case if the discount is given in %. The discounted price will be calculated by decrementing the specified % amount from the retail price. If this option is selected in the dropdown box, then a textbox will be displayed to specify the discount %.
b. Discount Value
If the exact value to be decremented from the retail price of the product is to be specified, then this option should be used. In this case, the discounted price is calculated by decrementing the value given in the discount field from the retail price of the product. If this option is selected in the dropdown box, then a textbox will be displayed to specify the discount value.
c. Exact Discount Price
If the exact discount price is to be specified, then use this option. In this case the value given in the discount field will be taken as the price for the product. If this option is selected in the dropdown box, then a textbox will be displayed to specify the exact discounted price.
16. Preorder
If customer is to be allowed to place order for current product which is not in stock at this moment, but expected to be in stock on a future date, then this can be done using the pre order section. Tick mark the "Allow Preorder" checkbox. When this is tick marked, 2 new textboxes will be displayed. They are as follows
a. Total Preorder Allowed
Use this textbox to specify the number of times this product can be placed in order when not in stock. This count will get reduced each time this product is placed in order.
b. Instock Date
The expected date on which the product will be in stock. This should be a future date.
Note: Preorder settings will reset automatically when the stock arrives for the product. Also if the option "Allow ordering even if out of stock" is set for a product then also the preorder settings for the product will be reset.
17. Product Deposit
If a product is to be placed in order by making only a partial payment for the product, then it can be handled from this section. The remaining amount will be charged at a later stage.
a. Deposit %
Specify what % of product price should be paid by customer while placing the order in this field. If specified, it will be displayed in the product details page.
b. Message
The message to be shown in the details page of current product in website if the product deposit exists for current product can be set from this section.
18. Other Settings
Additional settings that can be set to the current product can be done from this section.
a. Extra Shipping Cost
If extra shipping cost is to be considered when this product is added to the cart then the exact extra shipping value should be specify here. This will be added with the actual shipping calculated during checkout process.
b. Bonus Points
If bonus point is set in this text box, then customers who purchase this product will get those bonus points, when the order is successfull. Customer should be logged in to receive this bonus points.
c. Weight
The weight of current product can be set in this section. If delivery method depends on weight, then value should be set in this field properly inorder to calculate the delivery cost. The unit of weight will be displayed next to this textbox. This can be changed from the general settings section in the console. (Settings » General Settings » Main Shop Settings » Administration Tab).
d. Apply Tax
If tax is to be applied to this product, then this checkbox should be ticked. If this is unticked, even if tax is set in the site it will not be applied for current product.
e. Show Buy Link
Decides whether the Add to Cart /Buy link is to be displayed for the current product in various product listing pages in the website.
f. Show Enquiry Link
Decides whether the Add to enquiry link is to be displayed in listing or details pages of the current product.
g. In-Stock Notification allowed
If the current product is out of stock, then it is possible to allow customers to placed a request for in-stock notification mail. If any customer requests for such notification, a mail will be send to them when stock arrives for respective products. If customer is to be allowed to place in-stock notification for the current product will be decided based on the value set in this field. If this checkbox is ticked, then in the product details page the in-stock request option will be displayed. If the product is completly out of stock the request option will be displayed directly in the product details page. If stock is out for certain variable combination, the message will be displayed only if those combination are selected to add to the cart.
Note: If the option "Allow ordering even if out of stock" is set for the product "In- Stock Notification allowed" setting will be ignored and product will be allowed to be placed in stock.
h. Allow ordering even if out of stock
If is possible to allow customers to place orders for current product, even if stock does not exists for current product. To do this this option should be ticked for current product
Note: If the option "Allow ordering even if out of stock" is set for the product "In- Stock Notification allowed" setting will be ignored and product will be allowed to be placed in stock.
i.  Hide product when out of stock
If product is to be marked as hidden automatically if out of stock, then tick mark this. If variable stock exists for product, then it will be marked as hidden only if the stock for all the variable combinations are 0.
19. Price Settings
Price settings for current product can be managed from this section. Prices for all branches can be set from this section (if any branches exists for the site). The default branch will be "Web", if more branches are added to the console from Settings > Branches section in the console, those branches will be listed in this section to set the price for current product in those stores also. Thus it is possible to set different price for same product in different stores/branches.
Next to the textbox for web price, an icon is displayed. When clicked on this icon, the price set for branch "Web" will be copied to all other branches (if any exists).
Option to set the cost price for the product will also be displayed in this section. Cost price is the price at which the product has been purchased from the vendors.
20. Bulk discount
If discount is to be given if the current product is purchased in bulk, then it can be done from this section. Tick mark the "Enable Bulk Discount" checkbox. If ticked, 10 new rows will be displayed by default to set the rules for bulk discount. The columns available are "Quantiy" and "Price Per Item"
If the normal price of product is £1000, and if 5 qty of current product is added to cart, then if the product can be sold for a price of £950 each, then this setting can be done as follows
# Quantity Price Per Item
1. 5 950
     
In the product edit page more textboxes will be displayed to enter more bulk discount details (if required).
21. Image Display Format in Details Page
Display of product images in product details page can be controlled from this section. Depending on the option chosen in dropdown box, additional options (if any) will be displayed to the user. If any additional details related to currently selected value in the dropdown exists, then it will be displayed on the right hand side of the dropdown box. The values shown in this dropdown box are the following
a. Images Only
If the images are to be displayed in the product details using normal method, then this option should be selected.
b. Images with video
If the images are to be displayed using advanced method with video support, then this option should be selected. A file upload section which allows to upload the video in flv format will be displayed.
c. Images in flash
If the images are to be displayed using flash method, then this option should be selected.
c. Rotating Images using flash
If images for the products are to be viewed with options to rotate them, then this option should be selected. When this option is selected, 18 additional file upload boxes will be displayed. Use this file upload sections to upload the images to be shown in the rotating section in the order in which is it to be rotated. Images uploaded from this section will be displayed one after the other when customer try to rotate the images in product details page in the website. From the product add page only 18 images can be uploaded. Options to upload more images will be available in the product edit page in Images tab.
Click on "Save" button to save the product details.

Long Description, additional descriptions and free downloadable items for the current product can be managed from this section.
1. Long Description
A detailed description for the current product can be set in this html editor. This will be displayed in the product details page in the website. If the current product is set as the featured product, then by default this description will be displayed as the featured product description.
2. Product Keywords
This is a special field which can be used to specify the various keywords related to current product. When customer search for products from the website, the value set to this field for each of the products will also be searched. This increase the possiblility to display the product in the search result.
3. More Description Tabs
If more description tabs are to be added for current product then it can be done from this section. This section list the tabs already added for the current product. Click on "+" icon to view the list of existing tabs. Click on the tab title to go to the page to edit the details.
Click on "Add More" button in this section to go the page where the details of new tabs can be specified.
Hidden status of description tabs can be changed from this section. Tick mark the tabs for which the status is to be changed, select the required status from the section "Change Hidden Status to" and click the "Change" button.
The sort order for each of the tabs can be set by specifying the tab order in the textbox displayed with each of the tabs. Tick mark the tabs for which the order is to be saved and set the orders for the ticked tabs. Click on "Save Order" button to save the orders.
One or more tabs can be deleted at a time. Tick mark the tabs to be deleted and click on "Delete" button.
4. Product Attachments

Click on "Add More" button in this section to go the page where the details of new downloadable item can be specified.
Attachement can be of any of the following types
a. Audio
b. Video
c. PDF
d. Others
The downlodable items set from this section will be displayed in the product details page in the website.

The sort order for each of the downloads can be set by specifying the order in the textbox displayed with each of the rows. Tick mark the rows for which the order is to be saved and set the orders for the ticked rows. Click on "Save Order" button to save the orders.
One or more downloads can be deleted at a time. Tick mark the downloads to be deleted and click on "Delete" button.

Tab: Variables
Variables and variable message settings for current product can be done from this section.
The list of variables already added for the product will be listed in this section. Click on variable name to go to the edit page of an existing variable.
New variable for current product can be added by clicking on the "Add More" link. When clicked on this link, a new page will be loaded where the details of variables can be specified.
Hidden status of variables can be changed from this section. Tick mark the rows for which the status is to be changed, select the required status from the section "Change Hidden Status to" and click the "Change" button.
The sort order for each of the variables can be set by specifying the order in the textbox displayed with each of the rows. Tick mark the rows for which the order is to be saved and set the orders for the ticked rows. Click on "Save Order" button to save the orders.
One or more variables can be deleted at a time. Tick mark the variables to be deleted and click on "Delete" button.
Variables can be either displayed as dropdown box or as checkboxes in the product details page. This depends on the whether they have set to have values or not from its add / edit page.
Click for More Product Variable Options
Settings for variables can be done from this section. Click on "+" icon to view the settings.
a. Variable Price Display Type:
This option will be visible only if the current product have only one variable. This option allows to decide whether the full price or additional price itself should be displayed along with the variables in the product details page. If more than one variable exists for current product, then this option will be hidden and the "Add/Less" option will be selected by default.
i. Add / Less Price
If this option is selected, then the price set for variables will be displayed with "Add" or "Less" text. If the price is >0, then "Add" will be displayed. If the price is <0, then "Less" option will be displayed. The "Add" and "Less" captions can changed from the general settings section in console (Settings » General Settings » Price Display Settings » Price Display Captions tab).
ii. Full Price
If this option is selected, then the full price of the product, (base price + additional price for variable) will be listed in the variables section in product details page.
b. Show variables in a new row?
If this checkbox is ticked, then the variables will be listed in a new row in product details page. This depends on the template used by the site and also may not have any effect in case of some type of image display setting.

Product Messages
Variable messages for current product can be set from this section.
The list of variable messages already added for the product will be listed in this section. Click on message title to go to the edit page of an existing message.

Tab: Stock
Stock related details for the product can be managed from this section. The options displayed in this tab depends on whether or not variables exists for the current product.
1. List From
This is a dropdown box. This will be displayed in this section if branches exists for current site. If no additional branches added for the site, then all the settings will be done to the default branch "Web". If more branches exists for current site, then the list of those branches along with the default branch "Web" will be listed in this dropdown box. The default branch selected will "Web". When a different branch is selected, the stock details for the selected branch will be displayed and can be managed from this section.
Note: Branches for the current site can be managed from "Settings »Branches" section.
Stock settings for various situations are described below
Case 1.
If the current product do not have any variables and no additional branches exists in current site.
The fields that will be displayed in this case are
1. Barcode
The barcode for the current product can be set or edited from this textbox.
2. Stock
The fixed stock for current product can be set or edited from this textbox.
Back to Top
Case 2.
If the current product do not have any variables but additional branches exists in current site. The dropdown box "List From" will be displayed with the branches added to the site.
The fields that will be displayed in this case when each of the branches are selected from the dropdown box are
1. Barcode
The barcode for the current product in the branch currently selected in the dropdown box can be set or edited from this textbox. It is possible to have different barcodes for the same product in different branches.
2. Stock
The fixed stock for current product in the branch currently selected in the dropdown box can be set or edited from this textbox.
3. Move to Branch
This column allows to move stock from the branch currently selected in the "List From" dropdown box to some other store. This section have 2 textbox with caption "Move". The product quantity to be moved from current branch to some other branch should be specified in this textbox. By default the total quantiy available in current branch for the given variable combination will be displayed in this textbox. A drop down box displayed next to this textbox shows the list of branches existing in the site. Select the branch to which the quantity entered in the textbox to be moved to. When clicked on "Save Changes" the details will be saved and also the specified quantiy will be moved to the branch selected in this drop down box (if any selected). If the quantity specified in the move textbox is greater than the available stock, then the maximum available stock value will be moved. If quantity to be moved is set but the branch to which it is to be moved is not selected for a combination, then that quantiy will not be moved.

4. Stock
The textbox will be disabled by default. If the "Individual Stock" checkbox is ticked, then this textbox will be enabled and user can specify the individual stock for each of the combination in this textbox.

List Products Sold
This section shows the list of products which have been sold during the last 90 days. A search feature has been included in this page to search for products. Total quantity ordered and also the total amount of order will displayed at the bottom of each of the pages.
Basic criteria that can be used to search for products are the following.
1. Product Name
2. Category
3. Vendor
4. From and To Dates
Any combination of above said fields can be used in the search criteria. Number of products to be displayed in each of the pages can be selected. Click on "Go" button to begin the search operation.
Details page of any of the products in the list can be viewed by clicking on the name of the products in the list.
