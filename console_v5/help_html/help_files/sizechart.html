How Product Specification Headings Works?
Product specification headings sections allows to define the headings that will be used to define more specification details for various products of the site. This is a common area where various specification headings to be used among the various products in the site will be defined. In the product edit page there are options to assign the headings to be linked with the respective product and also to enter the values required under each of the headings.
 
1. List Product Specification Headings
2. Add / Edit Product Specification Headings
 
List Product Specification Headings
Product specification headings can be searched by giving the name of the heading. Number of headings to be shown in each of the page and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the product specification heading add page. There are 2 ways to go to the edit page of an existing specification heading. One method is by clicking the heading name and another by tick marking the required heading and pressing the "Edit" icon.
Delete operation can be done to more than one heading at a time. Tick mark the headings to be deleted and press the "Delete" icon.
Status of product specification headings can be changed directly from the listing page. Tick mark the headings which need status changing. Select the required status from the drop down box provided at the section "Change Status" and press the "Change" button. If a heading is marked as hidden, then it will not be displayed in any of the products.
Order in which the headings should be displayed in the heading assigning page of products can be set by giving sort order values in textbox given for each of the headings in the list. Click the "Save Order" button to save the sort order.
