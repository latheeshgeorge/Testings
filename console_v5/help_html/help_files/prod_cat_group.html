How Product Category Menu Works?
Product category menu can be called as a category container. More than one categories can be assigned to a category menu and this menu can be displayed at various positions in the website in different styles. The various positions can be Top menu, Left menu, Right menu etc. Same category can be assigned to more than one product category menus.

List Product Category Menu
Product category menu can be searched by giving the name of the category menu. Number of categories in each of the page and also the sort order to be used can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go to the product category menu add page. There are 2 ways to go to the edit page of an existing product category menu. One method is by clicking the name of rhe product category menu and another by tick marking the required category menu and pressing the "Edit" icon.
Delete operation can be done to more than one product category menu at a time. Tick mark the product category menu to be deleted and press the "Delete" icon.
Status of product category menu can be changed directly from the listing page. Tick mark the product category menu which need status changing. Select the required status from the drop down box provided at the section "Change Hide Status" and press the "Change" button.
The field "Layout(position)(order)" in the menu lising is shown as a drop down box. This is because same menu can be set to show in more than one layout/position. This drop down box is just for an informative purpose. No operation is associated with this drop down box.

Add Product Category Menu
Product category menu can be added through this section.
Key Fields
1. Hide Menu name
This field decides whether the name of the product category menu is to be displayed to the customer. If ticked, then the name will be hidden in the site. When category menus are shown in the top or bottom, the menu name will be hidden always irrespective of this setting.
2. Display Location
This field allows to select the areas in which the menu is to be displayed. The list box shows the list of layouts and its positions to which the product category menu can be assigned. The values in this list box have the format "Layout Name (position)". There can be more than one layout for a site. "Layout name" refers to the layouts existing for the site. "Position" refers to the positions to which category menus can be set for the site (eg. right, left, etc). It is possible to show same product category menu in more than one layout/position. Use "ctrl" button to make multiple selection in this list box.
Note: Depending on the display location set for the product category menu, the product category menu will be displayed in various positions on the site such as top, left, right etc. The name of the product category menu will be displayed as the title for the static page menu by default in the website. In order to change this, you have to use the edit title icon of respective positions from the Edit Website Layout Section in the console (Settings » Design Settings » Edit Website Layout).
3. Menu Format
Product category menus can be displayed in 2 formats. First is as "Menu" and the second is as "Drop Down". If the "Menu" Option is selected, then all the categories assigned to the category menu will be displayed one after the other in the website. If the "Drop Down" options is selected, then the assigned categories will be shown in a selectable drop down box in the website.
4. Category Image Type
The type of image to be displayed for categories when the current category menu is set in the home page. Choose the "None" option if image is not to be displayed for home page categories.
Related Topic: The type of image to be displayed for each of the categories while viewing its details page can be set from the "Image Settings" option in "Settings" menu. The type of images to be displayed for each of the subcategories in a given category can be set from the parent category add/edit page.
5. Show in all Pages
Product category menus can be set to show in all pages or only for selected product or categories or static pages. From the "Add" page it is possible only to specify whether it is to be shown for all pages or not. Mapping to certain product or categories or static pages can be done from the product category edit page.

Edit Product Category Menu
Product category menu details can be edited through this section.
Key Fields
Tab: Main Info
Main category menu details can be set from this tab. Use "Save" button to save the changes. Details should be saved before switching the tabs.
1. Hide Menu name
This field decides whether the name of the product category menu is to be displayed to the customer. If ticked, then the name will be hidden in the site. When category menus are shown in the top or bottom, the menu name will be hidden always irrespective of this setting.
2. Display Location
This field allows to select the areas in which the menu is to be displayed. The list box show the list of layouts and its positions to which the product category menu can be assigned. The values in this list box have the format "Layout Name (position)(order)". There can be more than one layout for a site. "Layout name" refers to the layouts existing for the site. "Position" refers to the positions to which category menus can be set for the site (eg. right, left, etc). "Order" is the sort order in which the category menu will be listed in the site. The order is important if more than one component is placed in same position. Components set to each of the positions are displayed in the site in the sort order shown in "Order". It is possible to set the same product category menu to more than one area. Use "ctrl" button to make multiple selection in this list box. By default the value of "Order" will be set to 0. It will be changed automatically when the position of each of the components are set from the Edit Website Layout Section in the console (Settings » Design Settings » Edit Website Layout).
Note: Depending on the display location set for the product category menu, the product category will be displayed in various positions on the site such as top, left, right etc. The title for the product category menu will not get modified in the website even when the menu name is changed from the edit page. In order to change the title you have to use the edit title icon of respective positions from the Edit Website Layout Section in the console (Settings » Design Settings » Edit Website Layout).
3. Menu Format
Product category menu can be displayed in 2 formats. First is as "Menu" and the second is as "Drop Down". If the "Menu" Option is selected, then all the categories assigned to the category menu will be displayed one after the other. If the "Drop Down" options is selected, then the assigned categories will be shown in a selectable drop down box.
4. Category Image Type
The type of image to be displayed for categories when the current category menu is set in the home page. Choose the "None" option if image is not to be displayed for home page categories.
Related Topic: The type of image to be displayed for each of the categories while viewing its details page can be set from the "Image Settings" option in "Settings" menu (Settings » Image Settings). The type of images to be displayed for each of the subcategories in a given category can be set from the parent category add/edit page.
5. Show in all Pages
Product category menus can be set to show in all pages or only for selected product or categories or static pages. The option to set the specific products, categories or static pages for which the current menu is to be displayed will be allowed only if the "Show in all pages" is not ticked for the category menu.

Tab: Categories in this Menu
Categories which are mapped to the product category menu being editing will be show in the area.
To assign more categories to the menu, click on "Assign more" button under this section. This will open up a page which shows the list of categories which are not yet mapped to the current menu. Tick the categories to be mapped and click the "Done" button to make the assignment.
To unassign categories from the menu, tick the categories to be unassigned and click the "Unassign" button in this section.
While listing the categories mapped with the menu, it is possible to set the following things
a. Order
The order in which the categories are to be displayed in the menu.
b. Display Type
While displaying categories it is possible to show it as a normal category or as a heading. This option allows to decide this. By default "Normal" will be selected.
c. Link Req
This section allows to set whether the categories listed in the category menu should be a link or not. By default "Yes" will be selected.
d. Hidden
The hidden status of the mapped category will be shown in this section.
In order to save the changes done to the above things, click the "Save Details" button.

Tab: Display menu for following categories
If the menu is to be shown while viewing the details page of certain caregories only, then those categories should be set from this section. This setting will be ignored if the "Show in all pages" option is set for the menu.
Use "Assign More" button to assign more categories to this section
Categories assigned to this section can be unassigned by tick marking the required categories and clicked the "Unassign" button in this section.
Tab: Display menu for following products
If the category menu is to be shown while viewing the details page of certain products only, then those products should be set from this section. This setting will be ignored if the "Show in all pages" option is set for the menu.
Use "Assign More" button to assign more products to this section
Products assigned to this section can be unassigned by tick marking the required products and clicked the "Unassign" button in this section.
Tab: Display menu for following static pages
If the category menu is to be shown while viewing certain static pages only, then those static pages should be set from this section. This setting will be ignored if the "Show in all pages" option is set for the menu.
Use "Assign More" button to assign more static pages to this section
Static Pages assigned to this section can be unassigned by tick marking the required static pages and clicked the "Unassign" button in this section.
