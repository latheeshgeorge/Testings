How Site Help Works?
This section allows to define the Heading and definitions to be displayed in the site help page in the website. In static page menu section, option has been given to decide whether a particular static page menu need to show a help link or not.
 
1. List Help
2. Add / Edit Help
 
List Help
Help can be searched by giving the heading. Number of help headings to be shown in each of the page and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the help add page. There are 2 ways to go to the edit page of an existing help. One method is by clicking the heading and another by tick marking the required heading and pressing the "Edit" icon.
Delete operation can be done to more than one help at a time. Tick mark the help to be deleted and press the "Delete" icon.
Status of help can be changed directly from the listing page. Tick mark the help which need status changing. Select the required status from the drop down box provided at the section "Change Status" and press the "Change" button. If a help is marked as hidden, then it will not be displayed in the help page in the site.
Order in which the help should be displayed in the site can be set by giving sort order values in textbox given for each of the help in the list. Click the "Save Order" button to save the sort order.

Add/Edit Help
Help can be managed through this section.
Key Fields
1.Heading
The heading to displayed in the help page of the website can be set in this text box
2. Description
Description to be displayed for the above heading can be set in this html editor.
3. Order
Sort Order for the current help heading in the help page can be set in this text box
4. Hide
Help can be made active or hidden using this fields. Only active help will be listed in the website.
