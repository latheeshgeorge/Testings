How Surveys Works?
Survey section allows to add surveys to the site. It is possible to enter more than one survey at a time and place them at different positions in the site. Options have been given to set the date range at which the surveys are to be displayed in the site.

List Survey
The surveys created by console users will be listed in this section. Surveys can be search by giving the survery title and survey status. Number of surveys to be shown in in each of the pages and also the sort order to be used can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the survey add page. There are 2 ways to go to the edit page of an existing survey. One method is by clicking the name of the survey and another by tick marking the required survey and pressing the "Edit" icon.
Delete operation can be done to more than one surveys at a time. Tick mark the surveys to be deleted and press the "Delete" icon.
Survey Status of surveys can be changed directly from the listing page. Tick mark the surveys which need status changing. Select the required status from the drop down box provided at the section "Change Survey Status" and press the "Change" button.
Hidden status of surveys can also be changed directly from the listing page. Tick mark the surveys which need status changing. Select the required status from the drop down box provided at the section "Change Hidden Status" and press the "Change" button.

Add Survey
New surveys can be created through this section.
Key Fields
1. Survey Title
This will be the default title displayed for the survey while displayed it in the site. The title to be displayed in the site for the survey can be set by editing it from the "Website Layout Editing" section.
2. Survey Question
The question to be shown in the survey should be given in this textbox.
3. Display Results
If this is ticked, then the result of the survey (till now) will be displayed to the customer in a graphical format along with the successfull message once they are made their voting. If this is not ticked, only the voting successfull message will be displayed.
4. Display Location
This field allows to select the areas in which the survey is to be displayed. The list box shows the list of layouts and its positions to which the survey can be assigned. The values in this list box have the format "Layout Name (position)". There can be more than one layout for a site. "Layout name" refers to the layouts existing for the site. "Position" refers to the positions to which survey can be set for the site (eg. right, left, etc). It is possible to show same survey in more than one layout/position. Use "ctrl" button to make multiple selection in this list box.
5. Options Text
The multiple options to be shown for the survey can be specified in these text boxes.
6. Order
The sort order for the options for the survey can be set using this option. If this order is set, the multiple options for the survey will be displayed in the specified order.
7. Show in all page
Survey can be set to show in all pages or only for selected product or categories or static pages. From the "Add" page it is possible only to specify whether it is to be shown for all pages or not. Mapping to certain product or categories or static pages can be done from the survey edit page.
8. Survey Status
The status what can be set for survey from the add page are "NEW" and "Active". If the status is set to "NEW", then the survey will not be displayed in the site. If the status is set to "Active", then the survey will be displayed in the site.
9. Hidden
Survey can be made active or inactive using this option.
10. Active Period
If the survey is to be displayed only during a specified range of dates, then tick the option "Change Active Period". Two boxes will be displayed which allows to select the from and to dates within which the survey is to be displayed in the site.

Edit Survey
Surveys can be edited through this section.
Key Fields
Tab: Main Info
Main details realted to survey can be set from this section.
1. Survey Title
This will be default title displayed for the survey while displayed it in the site. The title can be edited from the "Website Layout Editing" section.
2. Survey Question
The question to be shown in the survey should be given in this textbox.
3. Display Results
If this is ticked, then the result of the survey (till now) will be displayed to the customer in a graphical format. If this is not ticked, only the voting successfull message will be displayed.
4. Display Location
This field allows to select the areas in which the survey is to be displayed. The list box show the list of layouts and its positions to which the survey can be assigned. The values in this list box have the format "Layout Name (position)(order)". There can be more than one layout for a site. "Layout name" refers to the layouts existing for the site. "Position" refers to the positions to which survey can be set for the site (eg. right, left, etc). "Order" is the sort order in which the survey will be listed in the site. The order is important if more than one component is placed in same position. Components set to each of the positions are displayed in the site in the sort order shown in "Order". It is possible to set the same survey to more than one area in the site. Use "ctrl" button to make multiple selection in this list box. By default the value of "Order" will be set to 0. It will be changed automatically when the position of each of the components are set from the "Edit Website Layout" section in the site console.
5. Options Text
The multiple options to be shown for the survey can be specified in these textboxes.
6. Order
The sort order for the options for the survey can be set using this option. If this order is set, the multiple options for the survey will be displayed in the specified order.
7. Show in all page
Survey can be set to show in all pages or only for selected product or categories or static pages.
8. Survey Status
The status of survey can be set using this option. The status values available are New, Active, Finish and Publish. When a new survey is created but not displayed in the sites yet, then it should be set to the "New" status. If the status is set to "Active", the survey is ready to be displayed in the site. The "Finish" option will stop displaying the survey in the site. Once the result of the survey is published then the status can be set to "Publish". In short the surveys with status set to "Active" only are ready to be displayed in the site.
9. Hidden
Survey can be made active or inactive using this option.
10. Active Period
If the survey is to be displayed only during a specified range of dates, then tick the option "Change Active Period". Two boxes will be displayed which allows to select the from and to dates within which the survey is to be displayed in the site..

Tab: Show Survey in these Categories
If the survey is to be shown while viewing the details page of certain categories only, then those categories should be set from this section. This setting will be ignored if the "Show in all pages" option is set for the survey.
Use "Assign More" button to assign more categories to this section
Categories assigned to this section can be unassigned by tick marking the required categories and clicked the "Unassign" button in this section.
