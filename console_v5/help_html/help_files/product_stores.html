How Branches Works?
Branches section allows to manage the branches for the sites existing in various places. Once the branches are created, the products which are to be maintained in various branches and also the stock to be kept in various branches can be set from the product edit page in console. While creating a console user the branches created through this section will be shown in a drop down box. This is done to create console users specifically for a particular branch.
 
1. List Branches
2. Add / Edit Branches
 
List Branches
The branches created by console users will be listed in this section. Branches can be searched by giving the branch name. Number of branches to be shown in each of the pages and also the sort order to be used can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the branch add page. There are 2 ways to go to the edit page of an existing branch. One method is by clicking the name of the branch and another by tick marking the required branch and clicking the "Edit" icon.
Delete operation can be done to more than one branches at a time. Tick mark the branches to be deleted and click the "Delete" icon.
Status of branches can be changed directly from the listing page. Tick mark the branches which need status changing. Select the required status from the drop down box provided at the section "Change Hidden Status" and click the "Change" button.
Sort order for branches can be set from the listing page itself by specifying the required sort order in the textboxes provided with each of the branches in the list. Once the order is set for all the required branches, click the "Save Order" button to save the sort orders.

Add/Edit Branches
New branches can be created/modified through this section.
Key Fields
1. Branch name
This is the name by which the branch will be identified through out the console area.
2. Contact Person
This field helps to identify the person who manages the branch. This is just for informative purpose.
3. Active Branch
Branches can be made active or inactive using this option.
4. Order
This option allows to specify the sort order in which the branches should be displayed in various sections in console area.
