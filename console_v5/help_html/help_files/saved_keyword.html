How Saved Keywords Works?
Saved keywords section will work only if the current site is in webclinic. When customer perform a search operation in the website with a keyword and the search have result, then those keywords will be saved in Saved keywords section.
List Entire SEO Saved Search Keywords
The whole list of saved keywords will be listed in this section.
Delete operation can be done to more than one keywords at a time. Tick mark the keywords to be deleted and press the "Delete" icon
Saved keywords will be listed in the "Saved Keywords " section in the Website. Each of the keywords in this page will be a link and when clicked on the link, the search operaion will be performed with the keyword and the result will be displayed in the website.
If any caption is to be displayed to the top of this search result page in the website, then that description can be given in the textbox displayed in the "Description" column in the saved search listing page. So while displaying the result page for a saved search keyword, the description set for that keyword will be displayed in the website.
Click on the "Save" button to save the descriptions set for the saved keywords.
