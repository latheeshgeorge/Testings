How Product Enquiries Works?
Enquiries posted by customers for various products will be listed in this section.
 
1. List Product Enquiries
2. List Product Enquiry Details
3. Customer Details
4. Product Details
5. Notes
 
List Product Enquiries
Product enquiries can be searched by giving the name or email id of person who posted the enquiry, the status of the enquiry and also by specifying the range of dates between which the enquiries are posted. Any combination of the above specified options can be used as search criteria. Number of enquiries to be shown in each of the page and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
By default only the new product enquiries will be listed in this page.
Details of product enquiries can be viewed by clicking on the Date column in the list.
Delete operation can be done to more than one enquiries at a time. Tick mark the enquiries to be deleted and press the "Delete" icon.
Status of enquiries can be changed directly from the listing page. Tick mark the enquiries which need status changing, select the required status from the drop down box provided at the section "Change Status" and press the "Change" button.

List Product Enquiry Details
Details of product enquiries can be viewed in this page. While viewing the details of an enquiry which have the status as "New", the status will be automatically changed from "New" to "Pending".
Key Fields
Tab: Customer Details
This section mainly shows the details of customer who posted the enquiry. If any custom form is set for the product enquiry section, the details of those custom forms also will be displayed in this section.
Status of enquiry can also be changed from this section. Select the required status from the dropdown box and click the "Go" button to change the status.
The "Send Mail" button displayed in this section allows to send emails to the customer who posted the enquiry. When the "Send Mail" button is clicked, options to specify the subject and the content for the mail will be displayed. Specify the details and click on "Send" button to send the mail. Email will be send to the email id given in the enquiry. Use "Cancel" button to hide the send mail details.

Tab: Product Details
The list of products included in the enquiry will be listed in this section. One enquiry can have more than one products. The details of products such as name, variables (if any), retail price, cost price, discount (if any) etc will be displayed. In additional to this if any additional note added for the products in the enquiry, then a text "Details" will be displayed with each of such products in the list. When clicked on this, a new row will be displayed showing the note added for that particular product.

Tab: Notes
This section allows the console users to add notes related to the current enquiry. There are 2 section in notes tab namely "Existing Notes" and "Add Notes". Notes which are already added will be displayed under the "Existing Notes" section. An icon is shown with each of the added notes which allows to delete that note. The name of the console user who added the note and also the date on which the note has been added will be listed in this section.
New notes can be added from the "Add Notes" Section. Type the note in the textarea and click on "Save Note" button to save it. The newly added note will be displayed at the top in the "Existing Notes" section.
