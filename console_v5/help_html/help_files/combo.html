List Combo Deals
Combo details created by console users will be listed in this section. Combo deals can be search by giving the deal name. Number of deals to be shown in in each of the pages and also the sort order to be used can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the combo deal add page. There are 2 ways to go to the edit page of an existing combo deal. One method is by clicking the name of the deal and another by tick marking the required deal and pressing the "Edit" icon.
Delete operation can be done to more than one deals at a time. Tick mark the deals to be deleted and press the "Delete" icon.
Hidden status of combo deal can also be changed directly from the listing page. Tick mark the deals which need status changing. Select the required status from the drop down box provided at the section "Change Hidden Status" and press the "Change" button.

Add Combo
New combo deals can be created through this section.
Key Fields
1. Combo Name
This will be default title displayed for the combo deal while displayed it in the site. The title can be edited from the "Edit Website Layout" section (Settings » Design Settings » Edit Website Layout).
2. Combo Description
If any description is to be displayed while viewing the combo details in the website, that can be specified in this editor.
3. Combo Position
This field allows to select the positions in which the combo deal is to be displayed like left menu, top menu etc. The list box shows the list of layouts and its positions to which the deal can be assigned. The values in this list box have the format "Layout Name (position)". There can be more than one layout for a site. "Layout name" refers to the layouts existing for the site. "Position" refers to the positions to which deal can be set for the site (eg. right, left, etc). It is possible to show same deal in more than one layout/position. Use "ctrl" button to make multiple selection in this list box.
4. Hide Combo
Combo deal can be made active or inactive using this option.
5. Hide Combo Name
If the title of combo name is to be made hidden in the website, then this should be set to "Yes".
6. Show in all
Combo deals can be set to show in all pages or only for selected product or categories or static pages. From the "Add" page it is possible only to specify whether it is to be shown for all pages or not. Mapping to certain product or categories or static pages can be done from the combo deal edit page.
7. Change Active Period
If the combo deal is to be displayed only during a specified range of dates, then tick the option "Change Active Period". Two boxes will be displayed which allows to select the from and to dates within which the combo dealis to be displayed in the site.

Edit Combo
Combo deals can be edited through this section.
Key Fields
Tab: Main Info
Main details for the combo can be set from this tab.
1. Combo Name
When a combo deal is added, by default the combo name will be displayed as the title for the deal in the website. Editing this name from this page will not change the title of combo in website. The tile can be edited from the "Edit Website Layout" section (Settings » Design Settings » Edit Website Layout).
2. Combo Description
If any description is to be displayed while viewing the combo details in the website, then that can be specified in this editor.
3. Combo Position
This field allows to select the areas in which the combo deal is to be displayed such as left menu, top menu etc. The list box show the list of layouts and its positions to which the deal can be assigned. The values in this list box have the format "Layout Name (position)(order)". There can be more than one layout for a site. "Layout name" refers to the layouts existing for the site. "Position" refers to the positions to which deal can be set for the site (eg. right, left, etc). "Order" is the sort order in which the deal will be listed in the site. The order is important if more than one component is placed in same position. Components set to each of the positions are displayed in the site in the sort order shown in "Order". It is possible to set the same deal to more than one area in the site. Use "ctrl" button to make multiple selection in this list box. By default the value of "Order" will be set to 0. It will be changed automatically when the position of each of the components are set from the Edit Website Layout Section in the console (Settings » Design Settings » Edit Website Layout).
4. Hide Combo
Combo deal can be made active or inactive using this option.
5. Hide Combo Name
If the title of combo name is to be made hidden in the website, then this should be set to "Yes".
6. Show in all pages
Combo deals can be set to show in all pages or only for selected product or categories or static pages. The assignment to certain products, categories or static pages will be allowed only if "Show in all pages" is not ticked.
7. Change Active Period
If the combo deal is to be displayed only during a specified range of dates, then tick the option "Change Active Period". Two boxes will be displayed which allows to select the from and to dates within which the combo deal is to be displayed in the website.

Tab: Products in this deal
The products which are bundled in the combo deals are to be set using this section.
Use "Assign More" button to assign more products to the combo deal.
Use "Unassign" button to unassign the products already assigned to the combo deal.
The order in which the bundled products to be displayed while showing the combo deal in site and also the discount to be applied to the product when purchased in bundle can be set from this area. In the products list shown in this section, there are 2 text boxes associated with each of the products. The first textbox can be used to set the sort order for the product and the second texbox can be used to set the discount % to be applied to the respective product when purchased using this deal.

Tab: Display Deal for Following Products
If the combo deail is to be displayed while viewing the details page of certain products, then those products should be set from this section. This setting will be ignored if the "Show in all pages" option is set for the deal.
Use "Assign More" button to assign more products to this deal.
Products assigned to this section can be unassigned by tick marking the required products and clicking the "Unassign" button in this section.

Tab: Display Deal for Following Categories
If the combo deal is to be shown while viewing the details page of certain categories, then those categories should be set from this section. This setting will be ignored if the "Show in all pages" option is set for the deal.
Use "Assign More" button to assign more categories to this section
Categories assigned to this section can be unassigned by tick marking the required categories and clicking the "Unassign" button in this section.

Tab: Display Deal for Following Static Pages
If the combo deal is to be shown while viewing certain static pages, then those static pages should be set from this section. This setting will be ignored if the "Show in all pages" option is set for the deal.
Use "Assign More" button to assign more static pages to this section
Static Pages assigned to this section can be unassigned by tick marking the required static pages and clicking the "Unassign" button in this section.
