List Site Headers
Site headers can be searched by giving the header title. Number of headers in each of the page and also the sort order to be used can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the site header add page. There are 2 ways to go to the edit page of an existing site header. One method is by clicking the name of the site header and another by tick marking the required site header and pressing the "Edit" icon.
Delete operation can be done to more than one site headers at a time. Tick mark the site headers to be deleted and press the "Delete" icon.
Status of site headers can be changed directly from the listing page. Tick mark the site headers which need status changing. Select the required status from the drop down box provided at the section "Change Hide Status" and press the "Change" button.

Add Site Headers
Site headers can be added through this section.
Key Fields
1. Header Title
Title for the site header should be given in this field. This title is just to identify the headers in the console area. The title have no effect on the website.
2. Select Image
Selet the image to be uploaded as the site header. Use browse button to select the image to be uploaded.
3. Resize Image
If the image being uploaded is to be resized automatically, then this checkbox should be ticked. By default this will be ticked. If unticked, the image will not be resized while uploading.
4. Show in all Pages
Site headers can be set to show in all pages or only for selected product or categories or static pages. From the site header add page it is possible only to specify whether it is to be shown for all pages or not. Mapping to certain product or categories or static pages can be done from the site header edit page.
4. Hidden
This option allows to decide whether the site header is active or inactive in the website.
5. Periodic Change Required
If the site header is to be displayed during a given period of dates, then this should be ticked. Two new fields will be displayed when this is tick marked. The range of dates during which the site headers should be displayed can be set using these 2 fields.
Click on "Submit" button to save the changes.

Tab: Display Site Headers for Following Categories
If the site header is to be shown while viewing the details page of certain categories, then those categories should be set from this section. This setting will be ignored if the "Show in all pages" option is set for the site header.
Use "Assign More" button to assign more categories to this section
Categories assigned to this section can be unassigned by tick marking the required categories and clicked the "Un Assign" button in this section.
