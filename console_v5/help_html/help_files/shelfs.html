How Promotional Shelves Works?
Promotional shelves for the site can be managed from this section. Shelf is a container where products can be grouped. Different display styles and listing style may be available for shelves depending on the template used by the site.

List Shelves
Shelves can be searched by giving the shelf name. Number of shelves to be shown in each of the page and also the sort order to be used can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go to the shelf add page. There are 2 ways to go to the edit page of an existing shelf. One method is by clicking the name of the shelf and another by tick marking the required shelf and clicking the "Edit" icon.
Delete operation can be done to more than one shelves at a time. Tick mark the shelves to be deleted and press the "Delete" icon.
Status of shelves can be changed directly from the listing page. Tick mark the shelves which need status changing. Select the required status from the drop down box provided at the section "Change Status" and press the "Change" button.

Add Shelves
Shelves can be added through this section.
Key Fields
1. Shelf Name
Name for the shelf can be set from this section.
2. Show in all page
Shelves can be set to show in all pages or only for selected product or categories or static pages. From the shelf add page it is possible only to specify whether it is to be shown for all pages or not. Mapping to certain product or categories or static pages can be done from the shelf edit page.
3. Shelf Position
This field allows to select the areas in which the shelf is to be displayed. The list box shows the list of layouts and its positions to which the shelves can be assigned. The values in this list box have the format "Layout Name (position)". There can be more than one layout for a site. "Layout name" refers to the layouts existing for the site. "Position" refers to the positions to which shelf can be set for the site (eg. right, left, etc). It is possible to show same shelf in more than one layout/position. Use "ctrl" button to make multiple selection in this list box.
4. Fields to be displayed in shelf
This section allows to decide the fields to be displayed for the products which are mapped to current shelf while showing the shelf in the website.
a. Show Image
If the image is to be displayed for the products, then tick mark this.
b. Show Title
If the product title is to be displayed for the products, then tick mark this.
c. Show Description
If the product short description is to be displayed for the products, then tick mark this.
d. Show Price
If the product price is to be displayed for the products, then tick mark this.
5. Display Type
The display type for the current shelf can be set form this section. The values in this drop down box may vary for different templates. This can have values such as "one in a row", "3 in a row", "Menu" etc. This field decides how the products in current shelf is to be displayed in the website.
6. Listing Style
Depending on the template used by the site, there can be different pre-defined listing styles applicable for shelves such as "Normal", "Christmas", "New Year" etc.
7. Active Period
If the shelf is to be displayed only during a given range of dates, then this can be done from this section. When the "Active Period" is ticked, 2 new textboxes will be displayed which allows to specify the from and the to date ranges between which the shelf is to be displayed in the website.
8. Description
While viewing the shelf in the middle area, if any descriptions is to be displayed, then it can be set in this html editor. The description wiill be displayed before starting the listing of products assigned to the shelf.

Tab: Products in this Shelf
Products assigned to current shelf can be managed from this section. The list of products already asssigned to current shelf will be listed in this section.
Use "Assign More" button to assign more products to current shelf. When this button is clicked, a new page will be displayed which shows the list of products. Tick mark the products to be assigned to current shelf and click the "Assign Selected" button to make the assignment.
Use "Unassign" button to unassign any of the products already assigned to current shelf. Tick mark the products to be unassigned and click on "Unassign" button.
The sort order for products assigned to current shelf can be set using the textbox shown with each of the products in the list. Set the sort order for products in respective textboxes and click on "Save Order" button to save the changes. Make sure that the products for which the sort order to be saved is tick marked before clicking the "Save Order" button.

Tab: Show Shelf in these Products
If the current shelf is to be displayed while viewing the details page of certain products only, then those products should be set in this section. The products set in this section will be ignored if the "Show in all Pages" option is ticked for current shelf. Use "Assign More" button to view the page which allows to select the products to be assigned. Select the required products and click on "Assign Selected" button to make the assignment.
Use "Unassign" button to unassign any of the products mapped in this section. Tick mark the products to be unassigned and click on "Unassign" button.
Tab: Show Shelf in these Categories
If the current shelf is to be displayed while viewing the details page of certain categories only, then those categoreis should be set in this section. The categories set in this section will be ignored if the "Show in all Pages" option is ticked for current shelf. Use "Assign More" button to view the page which allows to select the categories to be assigned. Select the required categories and click on "Assign Selected" button to make the assignment.
Use "Unassign" button to unassign any of the categories mapped in this section. Tick mark the categories to be unassigned and click on "Unassign" button.
Tab: Show Shelf in these Static Pages
If the current shelf is to be displayed while viewing certain static pages only, then those static pages should be set in this section. The static pages set in this section will be ignored if the "Show in all Pages" option is ticked for current shelf. Use "Assign More" button to view the page which allows to select the static pages to be assigned. Select the required static pages and click on "Assign Selected" button to make the assignment.
Use "Unassign" button to unassign any of the static pages mapped in this section. Tick mark the static pages to be unassigned and click on "Unassign" button.
