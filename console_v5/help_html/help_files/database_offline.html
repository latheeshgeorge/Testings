How Manage Database Offline Works?
Manage database offline section allows to download the products / product variables & messages/ products Labels details as CSV file. The details in this CSV file can be edited using application such as MS Excel and then reupload it.
 
1. Download
2. Upload
 
While editing the downloaded file, the header row (first row) and also the values in columns indicated as "Dont Modify" should not be modified as this will compromise the database integrity and cause the upload to fail. For certain columns, the values that can be given in those columns are specified within the header column itself. Values in such columns should be one of the values specified in the header row, if any other value is specified, then the upload will fail.

Download
This section allows to download the products, product variables & messages and product label details in CSV format. The data in the csv file can be modified and the changes can be uploaded using the "Upload" section.
Key fields
1. Select Download Option
What is to be downloaded can be decided using this drop downbox. The values displayed in this dropdown box are the following
a. Products
If the details of products are to be downloaded, then this option should be selected.
b. Product Variables & Messages
If the details of product variables and messages are to be downloaded, then this option should be selected.
c. Product Labels
If details of product labels linked with products are to be downloaded, then this option should be selected.
2. All products
This checkbox decides whether the details of all products or products under selected categories are to be downloaded. If this is not ticked, then options will be displayed to select the categories
3. Select Categories from where the products are to be downloaded
This option will be visible only if the checkbox "All Products" is unchecked. The list of all categories will be displayed in this multi-select box. Use control button to select multiple categories. The products under the selected categories only will be considered in the download.
Click on "Download" button to start the download.

Upload
This section allows to upload the modified CSV file. The CSV file being uploaded should be the one exported using the "Download" section.
Upload will take some time depending on the amount of data.
Key fields
1. Select Upload Option
The type of uploading file should be selected in this dropdown box. The values displayed in this dropdown box are the following
a. Products
If the details of products are to be uploaded, then this option should be selected.
b. Product Variables & Messages
If the details of product variables and messages are to be uploaded, then this option should be selected.
c. Product Labels
If details of product labels linked with products are to be uploaded, then this option should be selected.
2. Select Upload File
The file to be uploaded should be selected using the browse button
Click on "Upload" button to upload the updated file.
Upload operation will be performed only if the file being uploaded is fully free of errors.
