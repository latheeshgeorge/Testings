How Delivery Settings Works?

The values to be used in the various drop down boxes in the delivery options page can be decided from this page.
If delivery option selected for the site is based on weight, then the delivery option managing page will have 2 drop down boxes for each of the settings. For all other delivery types there will be only one drop down for each of the values.
The two drop downs are refered as First drop down box and Second drop down box respectively. If the delivery type selected for the site is not related to weight, then the values set in the First drop down section only will be considered and the values entered in Second drop down section will be ignored.
Key Fields
1. First Drop Down Box Settings
a. Minimum Value
The starting value in the drop down box can be set in this text box. The value should be greater than or equal to 0.
b. Maximum Value
The last value in the drop down box can be set in this text box. The value should be greater than 0.
c. Increment value
The difference between 2 concecutive values in the drop down should be specified in this textbox. For e.g. If the values in the drop down box should be 1,3,5,7 etc, then the increment value should be 2.
2. Second Drop Down Box Settings ( This setting will be considered only if delivery method depends on weight)
a. Minimum Value
The starting value in the drop down box can be set in this text box. The value should be greater than or equal to 0.
b. Maximum Value
The last value in the drop down box can be set in this text box. The value should be greater than 0.
c. Increment value
The difference between 2 concecutive values in the drop down should be specified in this textbox. For e.g. If the values in the drop down box should be 1,3,5,7 etc, then the increment value should be 2.
3. Exclude delivery charge from gift voucher/ promotional code discount calculation
If delivery charge is to be excluded from the discount calculation due to gift voucher or promotional code, then this checkbox should be ticked.
