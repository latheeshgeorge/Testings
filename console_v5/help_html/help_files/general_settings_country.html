How Country Works?
 
1. List Country
2. Add / Edit Country
 
This section allows to define the countries to be used in your website. List of most of the countries will be set in this section by default. Console user can add new countries or edit / delete countries from the list.
List Country
Countries can be searched by giving the name of the country. Number of countries to be shown in each of the page and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the country add page. There are 2 ways to go to the edit page of an existing country. One method is by clicking the country name and another by tick marking the required country and clicking the "Edit" icon.
Delete operation can be done to more than one countries at a time. Tick mark the countries to be deleted and click the "Delete" icon.
Status of countries can be changed directly from the listing page. Tick mark the countries which need status changing. Select the required status from the drop down box provided at the section "Change Status" and click the "Change" button.

Add/Edit Country
Countries can be managed through this section.
Key Fields
1. Country Name
The name for the country should be specified in this textbox.
2. Country Numeric Code
The numeric iso country code for the country should be specified in this textbox.
3. Active
Countries defined for the website can be made active or hidden using this fields. Only active countries will be displayed in various pages in the website.
