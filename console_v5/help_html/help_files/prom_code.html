List Promotional Codes
Promotional codes can be searched by giving the code. Number of promotional codes to be shown in each of the pages and also the sort order to be used can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the promotional code add page. There are 2 ways to go to the edit page of an existing promotional code. One method is by clicking the code and another by tick marking the required promotional code and clicking the "Edit" icon.
Delete operation can be done to more than one promotional code at a time. Tick mark the promotional codes to be deleted and click the "Delete" icon.
Hidden status of promotional codes can be changed directly from the listing page. Tick mark the promomotional codes which need status changing. Select the required status from the drop down box provided at the section "Change Status" and press the "Change" button.
Promotional codes can be used only by registered customers.

Add Promotional Codes
Promotional codes can be added through this section.
Key Fields
1. Promotional code
This is the code which can be send to customers which they can be use to obtain discount while placing orders in the site. This should be a unique value.
2. Start From / End on
The Date range within which the promotional code will remain active. "Start From" is the start date and "End on" is the ending date.
3. Code Types
Discounts will be applied based on the selected code type. The explanation for each of the code types is as follows. Depending on the code type selection, different options will be displayed to the console user to fill in.
a. % off on grand total
If this code type is selected from the drop down box, a new option named "Discount %" will be displayed. The discount percentage specified in this field will be deducted from the grand total of current order.
b. Money off on minimum value of grand total
When this code type is selected from the drop down box, 2 new textboxes will be displayed namely "Discount for minimum" and "Discount Value". The discount criteria is as follows. If the grand total for the order is atleast the value specified in the field "Discount for Minimum", the customer will be granted with the discount equavalent to the amount specified in the "Discount Value" field. For e.g.In case if 100 is to be deducted from the total if purchase is made for a minimum value of 1000, then specify 1000 in "Discount for Minimum" and 100 in "Discount Value" field.
c. % off on minimum value of grand total
When this code type is selected from the drop down box, 2 new textboxes will be displayed namely "Discount for minimum" and "Discount % ". The discount criteria is as follows. If the grand total for the order is atleast the value specified in the field "Discount for Minimum", the customer will be granted with the discount of x%, where x is the value specified in the "Discount % " field. For e.g. In case if 10% is to be deducted from the total if purchase is made for a minimum value of 1000, then specify 1000 in "Discount for Minimum" and 10 in "Discount % " field.
d. Value off on selected products
When this code type is selected from the drop down box, options to assign products to current promotional code will be displayed in the promotional code edit page. Once the products are assigned, options to specify the promotional price for each of the assigned products will be displayed. When customer use this type of promotional code, if the order being placed includes any of the products assigned to the current promotional code, promotional price set for that product will be taken as the sale price for that product.
4. Total usage limit
The number of times the promotional code can be used is decided by this field. This is the total usage of promotional code (not just the usage of one customer). If the "Unlimited" checkbox is ticked, then any number of customers can use this promotional code. If the "Unlimited" checkbox is unchecked, then a new textbox will be displayed where the total usage count for the current promotional code can be set.
5. Total usage for same customer
This field decides whether the current promotional code can be used by same customer any number of times or for a specified number of times. If the "Unlimited" checkbox is ticked, then the same customer can use the current promotional code any number of times. If this checkbox is unchecked, then a new texbox will be displayed which allows to specify the exact number of times the code can be used by the same customer.
Use "Save" button to save the changes.

Tab: Products in this Promotional Code
This tab will be visible only if the promotional code type selected is "Value off on selected products". Products which are assigned to current promotional code will be listed in this section.
"Assign More" button can be used to assign more products to current promotional code. When clicked on this button a new page will be displaye which shows the list of products which are not yet assigned to current promotional code. Tick mark the products to be assigned and click the "Assign Selected" button to make the assignment.
Promotional price for each of the assigned products can be set using the textbox provided for with each of the products in the list. Set the price in the respective textbox and click on "Save Price" button to save the prices.
Status of assigned products can be changed from this section. Tick mark the products for which the status is to be changed, select the required status in the dropdown box from the "Change Status" section and then click on "Change" button to change the status.
Use "Unassign" button to unassign any of the products already assigned to current promotional code. Tick mark the products to be unassigned and click on "Unassign" button.

Tab: Orders which have used this Promotional Code
This section lists the orders for which the current promotional code have been used. The order id field in the list is a link and can take the user to the order details page. Customer name is the list is also a link and can take the user to the edit page of current customer, if that customer still exists in the website.

Tab: Customers used this Promotional Code
This section shows the list of customer who have used this promotional code. The displayed fields are Customer name, Total usage and Details. The "Details" is a link and when click on that link a new row will be displayed which shows the list of orders for which the current customer have used this promotional code.
Note: When a promotioal code is added or its details are updated, checking is made to see whether there already exists any other promotional code with the same code. Another checking will also be done to see whether there exists any gift vouchers which have the same code as that of current promotional code. In short, one website cannot have same code for promotional and gift vouchers.
If Delivery charge is to be excluded from the gift voucher or promotional code discount, the setting can be done from the Delivery Settings option in Settings section (Settings » Delivery » Delivery Settings).
