How Site Reviews Works?
Site reviews submitted by customers from the website can be managed from this section. Reviews submitted by customer need to be approved inorder to display it in the site.
 
1. List Site Reviews
3. Edit Site Reviews
 
When a new review is posted from the website by the customer, its status will be "New" by default. When details of the review is viewed by the console user, the status will be automatically change to "Pending".
List site Reviews
Site reviews can be searched based on author, status and between date range. Any combination of these can be used in search. Number of reviews to be shown in each of the pages and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
There are 2 ways to go to the edit page of an existing reviews. One method is by clicking the review date and another by tick marking the required review and pressing the "Edit" icon.
Delete operation can be done to more than one reviews at a time. Tick mark the reviews to be deleted and press the "Delete" icon.
Review status can be changed directly from the listing page. Tick mark the reviews which need review status changing. Select the required status from the drop down box provided at the section "Change Review Status" and press the "Change" button.
Hidden Status of review can be changed directly from the listing page. Tick mark the reviews which need status changing. Select the required status from the drop down box provided at the section "Change Hide Status to " and press the "Change" button.

Edit site Reviews
Reviews added by customers can be edited by the console user before approving it for displaying in the website. The edit page shows all the details related to the review such as the author details, rating given for the website and also the review text. Console user can edit the details, if required.
When a site review with status as "New" is edited, the status will be changed automatically to "Pending". The status should be changed to "Approved" to make it eligible to display in website.
Click on "Submit" button to save the changes.
