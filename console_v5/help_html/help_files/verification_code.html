How Google Settings Works?
Various settings related to google can be done from this page.
1. Google Meta Verification Code
If google meta verification code is to be displayed in the site header, then this checkbox should be ticked. When this is ticked, the textbox next to this checkbox will be enabled and the meta verification code can be set in this textbox.
2. Google Webtracker Code
If google webtracker script is to be used in the site, then this checkbox should be ticked. When ticked, the textbox to specify the google webtracker code will be enabled. Enter the code in this textbox.
3. Google Urchin Webtracker Code
If google urchin webtracker script is to be used in the site, then this checkbox should be ticked. When ticked, the textbox to specify the google webtracker urchin code will be enabled. Enter the code in this textbox.
4. Google Purchase Conversion
If purchases are to be tracked by google, then this option should be ticked. When this option is ticked, the following textboxes will be enabled. Values should be set for each of the textboxes in order to perform the purchase trackings.
a. Conversion Id
b. Conversion Language
c. Conversion Format
d. Conversion Color
e. Conversion Label
Click on "Save" button to save the changes.
When a section is disabled by unticking the checkboxes, If any values exists in the textboxes linked with the respective checkboxes, those values will not be removed. The values will be retained in the textboxes, but will not be used in the site until the respective checkboxes are ticked.
