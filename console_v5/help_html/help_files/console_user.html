Console Users
This section allows to manage the users who are able to access the console area of the site. Here there is provision for adding a User, editing, & deleting it.
 
1. List Console Users
2. Add/Edit Console Users
 
List Console Users
Users can be searched by entering the username. Sort order and the number of users to be listed per page in the search result can be set. Press the "Go" button to search for the users.
Use "Add" link to go to the page which allows to add new console users. There are 2 ways to go to the edit page of an existing console user. One method is by clicking the name in the list and another method is by ticking the user to be edited and press the "Edit" icon.
In order to delete console user(s), mark the tick box of respective user(s) and press the "Delete" icon. It is possible to delete more than one user at a time.
Status of users can be changed using the section "Change Active Status to". Tick mark the users whos status are to be changed and select the required status from the drop down and then press the "Change" button. Only active users are allowed to login to the console area of the site.

Add/Edit Console Users
New console users can be created / updated from this section.
Key Fields
1. User Type
There are 2 types of console users.
1. System Administrator (Sa) :
This user have full control over the console area
2. System User (Su)
When "System User" is selected from the "User Type" dropdown box, a new section will be displayed which allows to set the permission for the system user. The sections which are accessible to the user should be ticked.
2. Shop
This fields allows to select whether the current user have access to the web or to any subshop of current site. Default value will be "Web".
3. Active?
This field decides whether the current console user is active or not. Only active users will be allowed to login to console area.
4. Email
Email id will be username for the user
Use "Save" button to add/update the details
