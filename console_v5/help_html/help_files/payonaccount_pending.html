How Pay on Account Pending Details Works?
When customer make payment against their pay on account purchases, if the payment status of such transactions are not "Paid", then those transactions will get listed in the pay on account pending details section. Such transactions should be approved by the console administrator to consider it as a valid pay on account transaction.

List Pay on Account Pending Details
Pay on account pending details can be searched based on Customer name or Email id, Pay date range and also based on Payment Type used in the transaction. Any combination of these can be used as search criteria. Number of transactions to be shown in each of the pages and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
To view the details of a transaction, click on the "View Details" link in the list.

Account Pending Details
The details of a selected pay on account pending transaction will be displayed in this page. The details includes the following
1. Credit Limit and Current Account Balance
2. Details of customer who did the transaction
3. Transaction details such as
a. Transaction Date
b. Transaction Amount
c. Payment Type
d. Payment Method (if any)
c. Payment Status
d. Additional Description (if any)
If the payment type is cheque, then a new block will be displayed which shows the cheque details.
If the payment method used in "SELF", then the credit card details will be displayed in a new block.
Below all these details the section which allows to approve the transaction will be displayed. This section have the following
a. Payment Method
This drop down list shows a list of payment methods. The method by which the payment is actually received should be selected from this dropdown box.
b. Additional Details
If any additional details are to be stored along with this transaction at the time of approval, it should be specified in this textarea.
If the payment method of current transaction is related to any payment gateway, then console user have to manually check the payment gateway before approving the transaction.
Click on "Approve Payment" button to approve the current transaction. When a transaction is approved, it will get listed under the "Pay on Account Details" section of the respective customer. The approved transaction will no longer be visible in the Pay on Account Pending Transaction listing section once it is approved..
