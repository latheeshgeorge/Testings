How Tax Works?
This section allows to define the tax to be used in your website. It is possible to define more than one tax for yout websites. Even if tax details are set for a website, tax will be applied to a product only if the "Apply Tax" checkbox is ticked for that product.
 
1. List Tax
2. Add/Edit Tax
3. Tax Settings
 
List Tax
Taxes can be searched by giving the name of the tax. Number of taxes to be shown in each of the pages and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the tax add page. There are 2 ways to go to the edit page of an existing taxes. One method is by clicking the tax name and another by tick marking the required tax and clicking the "Edit" icon.
Delete operation can be done to more than one taxes at a time. Tick mark the taxes to be deleted and click the "Delete" icon.
Status of taxes can be changed directly from the listing page. Tick mark the taxes which need status changing. Select the required status from the drop down box provided at the section "Change Status" and click the "Change" button. If a tax is marked as hidden, then it will not be considered in tax calculations in the website.

Add/Edit Tax
Taxes can be managed through this section.
Key Fields
1. Tax Name
The name for the tax should be specified here. For e.g. VAT.
2. Description
A small description about the tax can be specified here. This is just for informative purpose and will be used only in the console area. This is an optional field.
3. Tax %
The tax percentage to be used in the tax calculation for current tax should be specified in this textbox.
4. Active
Taxes defined for the website can be made active or hidden using this fields. Only active taxes will be used in the tax calculation of the website.

Tax Settings Section
Various settings related to tax can be done from this section.
1. Apply tax before a discount is applied
If tax is to be calculated for a product before applying any discount for the product, then this checkbox should be ticked. For e.g. if the price for a product is 100 and if 10% discount is set for that product. So the sale price for the product will be 90. If this checkbox is ticked the tax will be calculated based on the original price of the product (i.e 100). If this is not ticked then the tax will be calculated based on the discounted price (i.e 90).
2. Apply tax to delivery charge
If delivery charge is to be considered while calculating the tax calculation during the checkout process of your website, then this should be ticked.
3. Apply tax to Giftwrap
If gift wrap charge is to be considered while calculating the tax calculation during the checkour process of your website, then this should be ticked.
Click on "Save Settings" button to save the changes.
