How Site FAQ Works?
This section allows to define the frequently asked questions and their answers to be displayed in the site. In static page menu section, option has been given to decide whether a particular static page menu need to show a FAQ link or not.
 
1. List FAQ
2. Add / Edit FAQ
 
List FAQ
FAQ can be searched by giving the question. Number of questions to be shown in each of the page and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the FAQ add page. There are 2 ways to go to the edit page of an existing FAQ. One method is by clicking the question and another by tick marking the required question and pressing the "Edit" icon.
Delete operation can be done to more than one FAQ at a time. Tick mark the FAQ to be deleted and press the "Delete" icon.
Status of FAQ can be changed directly from the listing page. Tick mark the FAQ which need status changing. Select the required status from the drop down box provided at the section "Change Status" and press the "Change" button. If an FAQ is marked as hidden, then it will not be displayed in the FAQ page in the site.
Order in which the FAQ should be displayed in the site can be set by giving sort order values in textbox given for each of the FAQ in the list. Click the "Save Order" button to save the sort order.

Add/Edit FAQ
FAQ can be managed through this section.
Key Fields
1.Question
The question to displayed in the FAQ pageof website can be set in this text box
2. Answer
Answer to be displayed for the above question can be set in this html editor.
3. Order
Sort Order for the current question in the FAQ page can be set in this text box
4. Hide
FAQ can be made active or hidden using this fields. Only active FAQ will be listed in the site.
