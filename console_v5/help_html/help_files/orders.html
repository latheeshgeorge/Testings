List Orders
This section lists the Product Orders placed on your website by various customers. Normally orders can be search by specifying the order id and also the status of the order. The number of orders to be displayed in each of the pages and also the sort order can be specified in the search criteria.
When clicked on the link "More Options", additional criteria that can be used for searching the orders will be displayed. The additional options available are "Customer / Company Name", "Email Id", and "Between Dates". Any combination of these can be used to build the search criteria. Click on "Go" button to begin the search operation.
To generate the packing slip for orders, tick mark the orders and click on the "Print Packing Slips?" button. This will open a popup window where the packing slip details of all the selected orders will be listed one after the other.
To generate the printer friendly details for orders, tick mark the orders and click on the "Printer Friendly?" button. This will open a popup window where the printer friendly details of all the selected orders will be listed one after the other.
Orders with different status will be represented using different colors. The color codes which are used for various order status can be viewed when clicked on the "Show Colour Codes" link. When clicked on this link a new row will be displayed where the the background color and text color that will be used for each of the order status will be displayed.
Back to Top
Details of an order can be viewed either by clicking directlly on the respective row or by tick marking the required order and then clicking on the "Details" link in the page.
Key Fields
1. Name
This column shows the name of the customer who placed the order. If the current customer is a registered customer, then it will be indicated by showing an icon next to the name in this column. When mouse is placed on top of this icon, the text "Registered Customer" will be displayed as tool tip text.
2. Pre?
This field indicate whether any or all of the products in the order are in preorder or not.
3. Order Status
The current status of order will be displayed in this section. When a new order is placed, its status will be set as "Unviewed". While viewing the details of an order, if the current status of the order is "Unviewed", then it will be automatically changed to "Pending".
4. Refund
The total amount refunded for each of the orders in the list will be displayed in this column.
5. Pay Status
The payment status of orders will be displayed in this column. If order payment status is changed manually for an order from the console of the website, then an icon will be displayed next to the payment status of such orders in this column. When mouse is placed on this icon, the name of the person who changed the payment status and also the date and time on which it has been changed will be displayed as tool tip text.
If the import / export feature is active for your site, then option to export orders will be displayed under the list of orders. There are 2 types of order export
1. Export Selected Orders
If this option is selected, the tick marked orders will be exported.
2. Export All Orders
If this option is selected, all orders will be exported.
When clicked on the "Export Now" button, you will be taken to the main import / export section and there you will be able to select the following things related to products being exported
1. Fields to be included in the exported file.
2. How the orders should be sorted in the exported file
3. Output Format of export file. The available format are CSV, HTML and SQL Dump.

Tab: Order Summary
The main details related to a selected order will be displayed in this section. The order id and date, payment type, payment method (if any used), payment status, order status and button to view the "Printer Friendly?" will be displayed at the top of the section.
"Change Order Status " allows to change the status of current order. The different status which are applicable to current order will be displayed in this dropdown box. The various status displayed are as follows
1. Unviewed
When the order status is to be set as "Unviewed", then this option should be selected in the dropdown box. When this option is selected, a button "Change Order Status?" will be displayed. Click on this button to change the order status to "Unviewed".
2. Pending
When the order status is to be set as "Pending", then this option should be selected in the dropdown box. When this option is selected, a button "Change Order Status?" will be displayed. Click on this button to change the order status to "Pending".
 
Back to Top
3. On Hold
When the order status is to be set as "On Hold ", then this option should be selected in the dropdown box. When this option is selected, a new block will be displayed with the heading "Order Status Change Reason (optional)". The reason for changing the order status can be set in the textarea displayed in this block if required. If reason is specified, it will be saved automatically in the "Notes" section of current order. Click on "Click here to Change the Status" button to change the status of current order to "On Hold". On Hold orders are orders for which the despatch has been hold for some reason.
Back to Top
4. Back Orders
When the order status is to be set as "Back Order ", then this option should be selected in the dropdown box. When this option is selected, a new block will be displayed with the heading "Order Status Change Reason (optional)". The reason for changing the order status can be set in the textarea displayed in this block if required. If reason is specified, then it will be saved automatically in the "Notes" section of current order. Click on "Click here to Change the Status" button to change the status of current order to "Back Order ". This is also a status which indicates that the order is prevented from getting despatched due to some reason.
Back to Top
5. Cancel
When current order is to be cancelled, then this options should be selected in the dropdownbox. When this option is selected, a new block will be displayed. There will be 2 headings in this new block. They are as follows.
1. Reason For Order Cancellation
The reason for order cancellation should be specified in this textarea. This is a compulsory field. The reason will be added to the "Notes" section of current order.
2. Options available while cancellation
This section display various options available while cancelling the order. They are as follows.
i. Place the quantity of products in order back to stock
This option will be displayed only if stock/inventory in maintained in the site. This option allows to decide whether the quantity of products placed in this order should be placed back in stock, when the order is cancelled. If this checkbox is ticked while cancelling the order, the quantity of product placed in current order will be added back to the stock of respective product in current website, provided product still exists in the website.
ii. Return the bonus points used by customer back to their account.
This option will be visible only if current order is placed by a registered customer and also bonus points have been used while placing the current order. If this is ticked, while cancelling the order, the bonus points used in the order will be added back to the bonus points of registered customer who placed this order.
iii. Take back the bonus points earned by customer from this order.
This option will be visible only if current order is placed by a registered customer and also the customer had earned some bonus points by purchasing the products. Bonus points set for a product from the console of the website will be added to the bonus points of registered customer when they purchase those products. If this is ticked while cancelling the order, the bonus points earned due to this order will be deducted from the bonus points of registered customer who placed this order.
iv. Decrement the maximum usage count for the gift voucher / Promotional code (if any).
This option will be visible only if gift voucher or promotional code is used in the current order. If this is ticked while cancelling the order, the total usage count of gift voucher / promotional code used in current order will be decremented by 1.
v. Cancel the order even if any of the products have already despatched or refunded.
Normally orders will not be allowed to be cancelled if any of the products in the order are despatched or any refund operation have been performed on the order. If forcefully such orders are to be cancelled, then this checkbox should be ticked. This will force the order to be cancelled irrespective of whether any despatch or refund occured for the order.
vi. Send Alternate Products
While cancelling an order an email will be send to the customer who placed the order with the details of order cancellation. If the site administrator would like to send a list of alternate products to the customer along with this cancellation email, then this checkbox should be ticked. When this is ticked, few other options will be displayed. They are as follows.
a. Choose Alternate Products
This section shows the category dropdown and a textbox to search for products to be set as alternate product. Select the category and/or enter the name of product and click on "Go" button. If products matching the given criteria exists in current site, then all those products will be listed in a multi-select box just below this search section. Select the products from the multi-select box and click on "Add Selected" button. The selected products will be moved to the multi-select box displayed under the section "Selected Alternate Products". Use ctrl button to make multiple selection.
b. Selected Alternate Products
The products selected to be send as alternate products along with the order cancellation mail will be listed in this multi-select box. If any products are to be removed from this multi-select box, then select those products, and click on "Remove" button. Use ctrl button to select multiple products.
Click on "Click here to Cancel this Order" button to cancel the current order". The notes added by the user will get saved in the "Notes" section of current order.
Billing Address
The billing address for current order will be displayed in this section.

Notes
Notes related to current order will be listed in this section. Click on "+" icon to view the notes. There are 2 blocks in this area namely "Existing Notes" and "Add Notes"
1. Existing Notes
Notes which are already existing for the current order will be listed in this section. The type of note will be displayed with each of the note listed in this section. To delete an existing note, click on the icon displayed with it.
2. Add Notes
New notes can be added through this section. Enter the note to be added in the textarea and click on "Save Note" button. The saved note will get displayed under the "Existing Notes" section.
Emails
Various emails related to current order which have been generated at different actions will be listed in this section.
Each of the rows in this section have the following columns.
1. Type
The type of email will be displayed in this column.
2. Subject
Subject of the email will be displayed in this column.
3. Mail Disabled?
This field indicated whether the respective email template was disabled at the time when tried to send the email for the first time. Email templates can be disabled by tick marking the disable option from the "Email Templates" section under "Settings" menu in console.
4. Send Atleast Once
This field indicates whether the current mail has been send out atleast once.
5. Details
When clicked on the "Details" caption, displayed on each of the rows in this list, the details of email will be displayed. The exact content of the mail can be viewed when clicked on the "Message" caption.
In order to send the mail again, click on "Send this Mail" button. When this button is clicked, the mail will be send again and also an entry will be displayed in the "Email Resend History" section to indicated the date and time and also the console user who resend the email..

Tab: Customer Queries
Customer queries related to current order and also the number of new posts will be displayed in this tab. To the top of current section the details such as order id and date, payment type, payment method (if any used), payment status, order status and button to view the "Printer Friendly?" will be displayed.
"Change Order Status " allows to change the status of current order. The different status to which the order can be set will be displayed in the dropdown box.
If a new enquiry or post exists for current order, then the total count (new queries + new posts) will be notified in the tab in the format "Customer Queries (x)", where x is the total count. The enquiries posted for current order will be listed in this section. The "Subject" column is a link which can take the console user to the section to view the details of respective query and posts.

Details such as tax, gift vouchers, promotional code and downloadable products related to current order will be displayed in this section.

This section will be visible only if tax is there in current order. When click on the "+" icon, the tax details will be displayed.
2. Gift Voucher Details
This section will be visible only if gift voucher is there in current order. When click on the "+" icon, the details will be displayed.
3. Promotional Code Details
This section will be visible only if promotional code is there in current order. When click on the "+" icon, the details will be displayed.
4. Downloadable Items
This section will be visible only if downloadable items exists for current order. When click on the "+" icon, the details will be displayed.
