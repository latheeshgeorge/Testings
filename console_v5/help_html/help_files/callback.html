How Call Back Requests Works?
In the website there is a feature named callback request. This feature allows customers who visit the website to post call back requests by entering their contact details. Such call back requests will be listed in the call back section in the site console.
 
1. List Callback Request
2. View Details of call back request
 
List Callback Request
Call back requests can be searched by giving the customer name, email id and the status of the call back request. Number of requests to be shown in each of the page and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
By default only the new call back requests will be listed.
Details of call back request can be viewed by clicking on the Name from the listing page.
Delete operation can be done to more than one call back requests at a time. Tick mark the requests to be deleted and press the "Delete" icon.
Status of requests can be changed directly from the listing page. Tick mark the requests which need status changing, select the required status from the drop down box provided at the section "Change Status" and press the "Change" button.

View Details of call back request
Details of call back requests can be viewed in this page. While viewing the details of a call back requests which have the status as "New", the status will be automatically changed from "New" to "Read".
Key Fields
1. Status
The various status to which the call back request can be set to will be listed in this drop down box. To change the status of the request currently being edited, select the required status from the dropdown and click the "Save" button.
