Tab: Design & Layout
1. Display Clear Cart button
If the button to clear the cart is to be displayed in the viewcart page of your website, then this should be ticked.
2. Display Promotional code / Voucher text box in View Cart
If the textbox to specify the promotional code or gift voucher number is to be displayed in the viewcart page in your website, then this should be ticked.
3. Display Clear Wish List button
If the button to clear the wishlist is to be displayed in your website, then this should be ticked.
4. Hide New User link in login
If "New User" link is to be made hidden from the login section of your website, then this should be ticked. When this is ticked, users will not be allowed to get registered with the site.
5. Hide Forgot Password link in login
If "Forgot Password" link is to be made hidden from the login section of your website, then this should be ticked.
6. Enable Product Compare feature
If product compare option is to be enabled in your website, then this should be ticked. If product compare feature is not included in the package you have purchased for your website, then this option will be ignored. When this checkbox is ticked, the following options will be displayed.
a. Maximum number of products to Compare
The maximum number of products that will be allowed to be compared at a time should be selected in this drop down box.
b. Show Product Compare in Product Listing Pages
If the option to add product to compare list is to be displayed in various product lisitng sections in your website, then this option should be ticked.
c. Show Product Compare in Product Details Page
If the option to add product to compare list is to be displayed in the product details page in your website, then this option should be ticked.
7. Display Quantity box
If textbox to enter the quantity is to be displayed in various product listing sections in your website, then this option should be ticked.
8. The product listing styles to be used in the following sections of your website can be set by selecting the required value in the dropdown box displayed with each of the options.
a. Search Product Listing
b. Linked Product Listing
c. Best Sellers Listing
d. Favorite Product Listing
e. Products in favorite Categories(ShowAll)
f. Recently purchased products(ShowAll)
g. Preorder Product Listing
The options that will be displayed in these dropdown boxes will depend on the theme used by the website. The products will be displayed in the website based in these settings.
9. Continue Shopping button
The action to be taken when customer clicks on the "Contine Shopping" button in the view cart page of your website will be decided by the value set in this dropdown box. The values available in this dropdown box are
a. Go Back to Home Page
If this option is selected in the dropdown box, then user will be taken to the home page of the website, when clicked on the "Continue Shopping" button in viewcart page.
b. Go Back to Previous Page
If this option is selected in the dropdown box, then user will be taken to the page from where they placed the latest product to cart.

14. Default Settings for Category Add page
The default settings that will be used to set the values by default in the category add page can be set from this section.
a. Subcategory List
The default position for displaying the subcategories of a category can be set using this dropdown box. The value set in this drop down box will be displayed as the default value for this option in the category add page.
b. Product Display Method
The default style for displaying the products in category details page can be set using this dropdown box. The value set in this drop down box will be displayed as the default value for this option in the category add page.
c. Product List

d. Fields to be displayed for Products
The default fields to be displayed for products under a category can be set using these checkboxes. The options which are marked as ticked here will be set as ticked in the category add page.

Tab: Security
Some of the features related to security can be set from this tab.
a. Ban these IP Address from placing orders:(press Enter after each IP address)
If customer from a certain IP address is to be banned from placing order, enquiries etc in your website, then those IP addresses can be set in this textarea. Each row should contain exactly one IP address. That is press enter key after entering each IP address.
b. Image Verification Settings in Various Sections
The image verification security allows to avoid injuction from hackers to various section of your website. This section allows to decide whether the image verification option is to be displayed in each of following sections.
i. Newsletter Image Verification
If image verification is to be displayed in the newsletter subscription section, then this should be ticked.
ii. Gift Voucher Image Verification
If image verification is to be displayed in the gift voucher buying section, then this should be ticked.
iii. Site Reviews Image Verification
If image verification is to be displayed in the site review adding section, then this should be ticked.
iv. Customer Registration Image Verification
If image verification is to be displayed in the customer registration section, then this should be ticked.
v. Product review Image Verification
If image verification is to be displayed in the product review adding section, then this should be ticked.
vi. Pay On Account Details
If image verification is to be displayed in the payonaccount payment section, then this should be ticked.
Click on "Save Settings" button to save the details

Tab: Administration Area
Some of the administrative settings required for the website can be done from this tab.
a. Terms and Conditions required during checkout
If "Accept Termsn & Conditions" checkbox is to be displayed in the checkout page, then this should be ticked.
b. Hide Add to Cart until Login
If "Buy / Add to cart" option is to be made hidden from customer until they login, then this should be ticked.
c. Enable caching for the site
If caching of various sections of your website is to be enabled, then this should be ticked. Caching improves the loading time of the website by reducing the database references by caching certains sections of the site as html files.
d. Hide Product price until Login
If price of products are to be made hidden in the website until customer logs in, then this should be ticked.
e. Force Customer to Login before Checkout
If customer is to be forced to login before they can proceed to checkout, then this should be ticked.
f. Force the Customer's billing & shipping to be the same address
If option to specify the address for delivery different from the billing address, then it can be decided using this option. If this is ticked, then billing address and shipping address will be the same one. If not ticked, customer will be given option in the checkout page to decide whether they want a different delivery address.
g. Hide Errors and Warnings from the console home page
Settings which are important for the smooth working of the website are not set properly, messages will be displayed in the home page of website console. If these messages are not to be displayed, even if errors or warnings exists, then this should be ticked. In short, the errors are warnings in the console home page will be displayed (if any exists) only if this option is not ticked.
h. Voucher Prefix
While generating the voucher number automatically by the system, if a common prefix is to be used, then that prefix can be set in this textbox. For e.g. if the prefix set in the textbox is "BSH" and the gift voucher number generated by the system is "1234567890", the voucher number will be "BSH1234567890".
i. Unit of Weight
The unit of weight to be displayed in various sections in site console should be set in the textbox.
i. Default Product Specification Heading
If additional product specifications exists for products and if heading to be displayed for those additional specification in the website is not defined, then the value set in this textbox will be displayed as heading. Product specifications details are displayed in the product details page in console.
Click on "Save Settings" button to save the details.details.

Tab: Inventory Management
Settings related to inventory can be done from this tab.
a. Enable Stock management
If stock is maintained in the website, then this should be ticked. If this is ticked then the following additional options will be displayed
i. Decrement Product Stock on Purchase
If the stock of products being purchased is to be reduced each time an order is placed successfully, then this should be ticked. If this is not ticked, the quantity of ordered products will not be decremented when an order is placed.
ii. Display stock details to customers in product detail page
If the details of stock is to be displayed in the product details page in the website, then this option should be ticked.
Click on "Save Settings" button to save the details.
