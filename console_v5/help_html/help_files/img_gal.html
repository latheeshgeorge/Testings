How Image Gallery Works?
Image gallery helps to manage the images to be assigned to categories, products, shops, gift wraps etc. In order to assign images, first the images should be uploaded to the image gallery. A search feature is included in this section which allows to search the uploaded images based on their title. Search can be done in current directory or in all directories.

Create Directory
A new image directory or subdirectory can be created using this section. Type the name of the directory to be created and press the Save button. The new directory will be created under current directory. By default Root Directory will be the current directory.
Image Directories
Image directories existing in the site will be listed in this area in a tree menu. “Expand All” and “Collapse All” links can be used to expand or collapse the tree menu. When clicked on the directory name, images under that directory will get displayed in the middle area and the currently viewing directory will be shown in “Currently in :” section in middle area. The name of the directory is shown in a text box (if not Root Directory). If the name is to be changed, change the name in text box and press the “Update Name” button. An image can be selected by clicking it. A border is shown for selected images to identify them easily. Clicking again on a selected image will deselect it and the border will not be shown for those images. When double clicked on any of the images, the page to edit the details of images will be displayed.

Upload Images
Images can be uploaded to the current directory by clicking on the “Upload More Images?” button. When this button is clicked a new page will be shown which allows to select the image files to be uploaded. Name of the directory to which the files will be uploaded will be shown at the top of this page. Ten images can be uploaded at a time.
The following are shown in each files upload section.
1. Caption
If any captions is to be associated with the file being uploaded, it can be specified over here. It is an optional field. It is recommended to give caption for images to make it possible to list them in image search result section.
2. Big Image
The full sized image file to be uploaded should be selected in this section. Site console will resize the image to the required dimension.
3. Small Image
The small size image file should be selected in this section. Site console will resize the image to the required dimension. It is not required to upload small image. By default this field will be disabled. To enable this field uncheck the checkbox “ Disable Small Image Upload (recommended to keep this ticked)” in the upload image page.
If only the big image file is selected and small image field is blank , then the small and other size images will be created automatically. This is why the small image field in disabled by default. If both the big and small images are selected to upload, then the resize will be applied to respective selected file. Once the files to be uploaded are selected, click on “Upload” button to upload the files. The result page will show the messages regarding whether the files have been uploaded or not. The files which are successfully uploaded will be listed in the images listing sections of current folder.

Operations on Image
Various operations that can e performed on images are listed under this section. The options listed in this section vary based on whether or not various features or objects are available at the site console. The
main operations that are allowed are
1. Move images to directory
This option allows to move selected images to the ima ge directory selected in the drop down box. To do this, highlight the images to be moved by clicking on then. Once the required images are selected, press the “Go” button in “Move image to directory” section. If images to be move are in multiple pages, select then by navigating to respective pages. In such a case the selection in a page will be retained. For example consider there are 3 pages. User selected 2 images in page1 and moved to page 2 to select more images. When user get back to page1 the 2 images which are previously selected will remain as selected.
2. Assign Images to Product Category
This section allows to assign selected images to the product category selected in the drop down box. To do this, select the images and select the product category to which the images are to be assigned and click the “Go” button in this section.
3. Assign Images to giftwrap paper
This section allows to assign selected images to giftwrap papers selected in the drop down box. To do this, select the images and select he giftwrap paper to which the images are to be assigned and click the “Go” button in this section. This option will be visible only if giftwrap paper exists in console.
4. Assign Images to giftwrap card
This section allows to assign selected images to giftwrap cards selected in the drop down box. To do this, select the images and select the giftwrap card to which the images are to be assigned and click the “Go” button in this section. This option will be visible only if giftwrap card exists in console.
Assign Images to giftwrap Bows
This section allows to assign selected images to giftwrap bows selected in the drop down box. To do this, select the images and select the giftwrap bow to which the images are to be assigned and click the “Go” button in this section. This option will be visible only if giftwrap bow exists in console.
Assign to shopbybrand
This section allows to assign selected images to Product shopbybrandselected in the drop down box. To do this, select the images and select he shopbybrand to which the images are to be assigned and click the “Go” button in this section. This option will be visible only if product shopbybrand exists in console.
Assign to Products in category
This section allows to assign selected images to selected products in category selected in the drop down box. To do this, first select the images, then select the category under which the product to which the images to be assigned exists and press the “Go” button in this section. A pop up message will be displayed to confirm the operation. When the “Ok” button in the confirmation message is clicked, then a new page will be shown with the list of products under the selected category. Select the products to which the selected images to be assigned and press the “Done” button to make the assignment. User will be redirected to the image listing page and appropriate message is shown

Image Edit Page
In the image edit page the details are shown in 3 tabs namely
Tab: General Info
This section lists the general information related to the images. The mage caption is shown in a text box. Various dimensions of small and big images are shown under 2 sections
1. Small Image and its other instances
1. Small
This is the default image size which will be shown in the product listing page by default
2. Gallery Thumb
This is the image which will be displayed in the image listing section in image gallery.
2. Big Image and its other instances
1. Big
This is the default image which will be shown in the product detail page in customer area.
2. Medium
This is the default images which will be shown for subcategories in customer area.
3. Category Main
This is the default image which will be shown in the category details page in customer area.
4. Icon

This is the image which will be used in case if icon type image is to be displayed in some sections in customer area. This may not be required in all the site. If the small image is to be changed, then use the “Browse” button next to the section“Change Small Image” to select the new image. When the small image is changed, the gallery thumb instance also will be changed automatically. If the small image file being uploaded is to be resized then tick mark the “Resize to fit” option next to the browse button. By default this will be ticked. It is recommended to keep his ticked. Resizing of
gallery thumb image will be done irrespective of this tick box.
If the big image is to be changed, then use the “Browse” button next to the section “Change Big Image” to select the new image. When the big image is changed, the medium, Category main and icon instances also will be changed automatically. If the big image file being uploaded is to be resized then tick mark the “Resize to fit” option next to the browse button. By default this will be ticked. It is recommended to keep this ticked. Resizing of instances of the image other than the big image will be done irrespective of this tick box.
3. Full Size Image
Full size image is the non resized version of the uploaded image. It can be viewed by clicking on the "Click here" link. If a new image is to be uploaded as full size image, then use the "Change Full Size image" upload section. Uploading a new full size image will have no impact on other instances of the images.
Use "Save" button to save the image caption.
"Delete" button delete the image. When an image is deleted, all mapping for that image to various products, categories etc also will be removed.

Tab: Assigned To
This section shows to which all sections the current image is assigned to. By default the sections are shown in collapsed manner. When clicked on the “+” icon in each of the section the list will be displayed. To unassign current image from any section, tick the required row and press the “Unassign” button in respective section.
