How Price Display Settings Works?
Settings required for price display in the website can be done from this section.
 
1. Main Settings Tab
2. Price Display Captions
3. Other Settings
 
Tab: Main Settings
The price display methods can be defined from this section.
a. Product Price Display Type
The format to be used for price display in website can be set using this. The options available are
i. Show Price only
If this option is selected, the product price will be displayed in various product listing pages with out tax. The tax will be applied to the product when it is added to the cart (provided Apply tax option is set for the product).
ii. Show Price + Tax
If this option is selected, the product price will be displayed in the various product listing pages in the format 'original price + tax' (provided Apply Tax option is set for the product). The caption Tax can be changed from the Headings and Captions section in settings menu.
iii. Show Price inc Tax
If this option is selected, the product price will be displayed in various product listing pages with the tax portion included in it (provided Apply tax option is set for the product).
iv. Show Both
If the product price in various listing pages should show both the price with out tax and price with tax, then use this option.
The decision of whether each of the items in the price display such as price, discount etc are to be displayed in one rows itself or in different rows in various section of the site can be decided from this section. Settings can be done to following sections
1. Shelves
2. Search Results
3. Category Detail
4. Combo Deals
5. Best Sellers
6. Linked Products
7. Shops
8. Other Product Listing
Click on "Save Changes" save the settings.

Tab: Price Display Captions
The captions to be used in price display of the website can be set from this tab.
1. Prefix for Normal Price
While displaying the price of products which do not have variables or which have variables but no additional price set for variables, the caption to be displayed as prefix is to be set in this textbox.
2. Suffix for Normal Price
While displaying the price of products which do not have variables or which have variables but no additional price set for variables, the caption to be displayed as suffix is to be set in this textbox.
3. Prefix for 'From' Price
While displaying the price of products which have variables and if additional price set for variables, the caption (if any) to be displayed as prefix is to be set in this textbox.
4. Suffix for 'From' Price
While displaying the price of products which have variables and if additional price set for variables, the caption (if any) to be displayed as suffix is to be set in this textbox.
5. Prefix for 'Special Offer' Price
While displaying the offer price of products, the caption (if any) to be displayed as prefix is to be set in this textbox.
6. Suffix for 'Special Offer' Price
While displaying the offer price of products, the caption (if any) to be displayed as suffix is to be set in this textbox.
7. Prefix for 'You Save' Price
If offer exists for a product and if "You Save" option is set to display for products, then the caption (if any) to be displayed as prefix is to be set in this textbox.
8. Suffix for 'You Save' Price
If offer exists for a product and if "You Save" option is set to display for products, then the caption (if any) to be displayed as suffix is to be set in this textbox.
9. Caption for no Price
If no price is set for a product, the caption to be displayed in the price display should be set in this textbox.
10. Prefix for 'Discount'
If offer exists for a product and if discount is set to display for products, then the caption (if any) to be displayed as prefix is to be set in this textbox.
11. Suffix for 'Discount'
If offer exists for a product and if discount is set to display for products, then the caption (if any) to be displayed as suffix is to be set in this textbox.
12. Prefix for 'Available Date'
If product is in preorder, the prefix to be used while displaying the expected stock date should be set in this textbox.
13. Suffix for 'Available Date'
If product is in preorder, the suffix to be used while displaying the expected stock date should be set in this textbox.
14. Prefix for 'Add Variable Price'
If additional price is set for variables, the prefix caption to be used while displaying the additional price in the website should be set in this textbox.
15. Suffix for 'Add Variable Price'
If additional price is set for variables, the suffix caption to be used while displaying the additional price in the website should be set in this textbox.
16. Prefix for 'Less Variable Price'
If negative value is set at additional pricet for variables, the prefix caption to be used while displaying the additional price in the website should be set in this textbox.
17. Suffix for 'Less Variable Price'
If negative value is set at additional pricet for variables, the suffix caption to be used while displaying the additional price in the website should be set in this textbox.
18. Prefix for 'Full Variable Price'
If additional price is set for variables and if setting is made to show full price of product while displaying the variable values , the prefix caption to be used should be set in this textbox.
19. Suffix for 'Full Variable Price'
If additional price is set for variables and if setting is made to show full price of product while displaying the variable values , the suffix caption to be used should be set in this textbox.
Click on "Save Changes" button to save the details.

Tab: Other Settings
More settings related to price display can be done from this tab.
1. Variable Price Display in Product details page
This option allows to decide whether additional price set for variables is to be displayed in the product details page or not.
a. Show additional price of variables
If this is selected, then the additional price of variables will be displayed in the product details page.
b. Hide additional price of variables
If this is selected, then the additional price of variables will not be displayed in the product details page.
2. Additional Settings
a. Show 'You Save' Price
If a product is in offer and the "You Save" option is to be displayed in the price display, then this should be ticked.
b. Strike out base price for special offers
If a product is in offer and if base price should be striked out in the price display, then this should be ticked.
c. Apply Discount to Variable
If discount is available for a product and if additional price exists for variables of product, then while applying discount the decision of whether it is to be applied to the total of base price and additional price or apply discount to the base price of the product first and then add the variable price can be done from this section. If this option is ticked, then the discount will be applied to the total of base price and additional price.
d. Show Discount with price display
If a product is in offer and if the discount applicable for that product is to be displayed in the price display, then this should be ticked. If this is not ticked, the discount part (if any) will not be displayed in the price display.
Click on "Save Changes" button to save the details.
