How Currency Works?
The currencies to be used in the website can be managed from this section. As the site support multiple currencies, more than one currency can be set for a site.
One of the currencies in this section will be marked as the default currency. The various price setting done from the site console should be in this currency.
It is not possible to change the default currency from the site console. It can be done only through the set up wizard or by super administrator.

2. Add / Edit Currencies
 
List Settings Currencies
Currencies added to the web site can be searched based on currency name. Number of currencies to be shown in each of the pages and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the currency add page. There are 2 ways to go to the edit page of an existing currency. One method is by clicking the currency name and another by tick marking the required currency and clicking the "Edit" icon.
Delete operation can be done to more than one currencies at a time. Tick mark the currencies to be deleted and click the "Delete" icon. The currency which is set as default currency will not be allowed to delete.
The rates for currencies with respect to the default currency can be set manually or can be picked automatically online. The checkbox "Fetch the currency rates automatically" allows to decide whether the currency rates should be picked automatically or to be set manually. If this checkbox is ticked , a new bottom namely "Get Live Currency Rate" will be displayed. When clicked on this button, the current rate for currencies other than the default currency (with respect to the default currency) will be picked automatically.
It is also possible to set a margin for currencies. This is allowed only for non default currencies. This allows to set a margin value for currencies. This margin value (if set) will be added to the rate of respective currencies to get the rate to be used for the currency. For e.g. if the rate of currency is set as 1.5 and the margin is set as 0.5, then the rate for that currency will be taken as 2.0 (1.5 + 0.5) in the website.

Add / Edit Currencies
Currencies to be used in the site can be managed from this page..
Key Fields
1. Currency Name
The name of the currency should be displayed in this field.
2. Currency Symbol
The currency symbol to be used for the currency should be set in this textbox.
3. Currency Code
The 3 letter standard code for the currency should be set in this textbox. This will be used while communicating with various payment gateways for making payments from the website.
4. Currency Rate
This will be visible only if "Fetch the currency rates automatically" is not ticked in the currency listing page. The rate of currency, with respect to the default currency, should be specified in this textbox.
5. Margin for Rate
If any margin is required for the currency rate, then it can be set in this textbox. This margin value (if set) will be added to the rate of current currency to get the actual rate to be used for current currency. For e.g. if the rate of currency is set as 1.5 and the margin is set as 0.5, then the rate for that currency will be taken as 2.0 (1.5 + 0.5).
6. Currency Numeric Code
The standard numeric code for the current currency should be specified in this field. This will be used while communication with various payment gateways for making payments from the website.
