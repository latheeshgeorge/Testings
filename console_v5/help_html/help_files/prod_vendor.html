How Product Vendors Works?
Vendors who supply various products can be managed from this page.
Various contact details of vendors and a statistics of quantiy of products sold and also the total income incured due to that sale also can be viewed.

List Vendors
Product vendors can be searched by giving the vendor name. Number of vendors to be shown in each of the page and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the vendor add page. There are 2 ways to go to the edit page of an existing vendor. One method is by clicking the vendor name and another by tick marking the required vendor and clicking the "Edit" icon.
Delete operation can be done to more than one vendors at a time. Tick mark the vendors to be deleted and click the "Delete" icon.
Status of vendors can be changed directly from the listing page. Tick mark the vendors which need status changing. Select the required status from the drop down box provided at the section "Change Status" and press the "Change" button. If a help is marked as hidden, then it will not be displayed in the help page in the site.
It is possible to assign products to vendors from the listing page. Tick mark the vendors to which the products are to be assigned and then click on "Assign more Products" button. When this button is clicked, a new page will be displayed where the list of products will be displayed. Tick mark the required products and click on "Assign Products" to make the assignment. The selected products, which are not yet assigned to the selected vendors only will be assigned.

Add Vendors
Vendors can be added through this page.
Key Fields
1. Vendor Name
The name of the vendor should be specified in this textbox. This is a compulsory field.
2. Email
The email id of vendor is to be specified in this textbox. This is also a compulsory field.
3. Order
Sort Order for the current help heading in the help page can be set in this text box
4. Hidden
This field decides whether the current vendors is still supplying products to us or not.
Click on "Submit" button to save the details.

Tab: Products mapped with this vendor
List of products which are mapped with current vendor will be listed in this section.
More products can be assigned by clicking on the "Assign More" button. When clicked on this button a new page will be displayed which shows the list of products. Tick mark the products to be assigned and click on "Assign Products" button.
Products can be unassigned from current vendor by tick marking the products and then clicking on the "Un Assign" button.
