How StateWorks?
This section allows to define the states for countries in your website. Console user can add new state or edit / delete states from the list. States can be added only if countries exists in your website.
 
1. List State
2. Add / Edit State
 
List State
State can be searched by giving the name of the state. Number of states to be shown in each of the pages and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the state add page. There are 2 ways to go to the edit page of an existing state. One method is by clicking the state name and another by tick marking the required state and clicking the "Edit" icon.
Delete operation can be done to more than one states at a time. Tick mark the states to be deleted and click the "Delete" icon.
Status of states can be changed directly from the listing page. Tick mark the states which need status changing. Select the required status from the drop down box provided at the section "Change Status" and click the "Change" button.

Add/Edit State
States for countries can be managed through this section.
Key Fields
1. Country Name
The list of countries added to your website will be listed in this dropdown box. The country under which the current state is to maintained should be selected in this dropdown box.
2. State Name
Name of the state should be specified in this textbox.
3. Active
State can be made active or hidden using this fields. Only active states will be displayed in various pages in your website.
