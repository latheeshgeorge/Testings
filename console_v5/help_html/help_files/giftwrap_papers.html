How Giftwrap Papers Works?
Gift wrap papers section allows to define the details of papers that will be used for wrapping the gifts. Giftwrap details will be displayed in the cart page.
 
1.List Giftwrap Papers
2. Add Giftwrap Papers
3. Edit Giftwrap Papers
 
List Giftwrap Papers
Papers can be searched by giving the name of the paper. Number of papers in each of the page and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the paper add page. There are 2 ways to go to the edit page of an existing paper. One method is by clicking the name of the paper and another by tick marking the required paper and pressing the "Edit" icon.
Delete operation can be done to more than one papers at a time. Tick mark the papers to be deleted and press the "Delete" icon.
Status of papers can be changed directly from the listing page. Tick mark the papers which need status changing. Select the required status from the drop down box provided at the section "Change Status" and press the "Change" button.
Order in which the papers should be displayed in the cart page can be set by giving sort order values in textbox given for each of the papers in the list. Click the "Save Order" button to save the sort order.

Add Giftwrap Papers
Giftwrap papers can be added through this section.
Key Fields
1. Paper name
The name that will be shown for the paper while displayed in the cart page
2. Extra Price
If extra payment have to be made for a paper, it should be specified in this section.

Edit Giftwrap Papers
Giftwrap papers details can be edited through this section.
Key Fields
Tab: Main Info
Main details of giftwrap papers can be set from this tab.
1. Paper name
The name that will be shown for the paper while displayed in the cart page
2. Extra Price
If extra payment have to be made for a paper, it should be specified in this section.
Tab: Paper Images
This section allows to map the images for the paper.
Use "Assign More" button in this section to go to the image gallery section to assign the image(s) for the paper. Once the images are assigned, option to set the order in which the images will be displayed to the customer and also the title for the images will be displayed. Specify these details and press the "Save Details" button to save the changes. One thing to be noted is that, the image details will be saved only for those images for which the checkbox is ticked.
Use "Un assign" button to unassign any assigned image(s) set for the paper.
