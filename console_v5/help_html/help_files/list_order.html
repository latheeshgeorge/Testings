Tab: Product Listing
Settings required for various product listing pages in website can be done from this tab.
1. On Search result page
The following settings can be done for the product listing in search result page
a. Sort by
The field by which the search result should be sorted is to be selected from this dropdown box. The options available are the following
i. Product Name
If the search result is to be displayed in the order of product name, then this should be selected.
ii. Price
If the search result is to be displayed in the order of price, then this should be selected.
iii. Date Added
If the search result is to be displayed in the order in which the product is added to the website console, then this should be selected.
b. Sort Order
The sort order to be used in the product search result page can be selected from this dropdown box. The options available are the following
i. Asc
If sorting is to be done in ascending order, then this option should be selected.
ii. Desc
If sorting is to be done in descending order, then this option should be selected.
c. Products per page
The number of products to be listed per page for search results should be set in this dropdown box.

a. Sort by
The field by which the best seller products should be sorted is to be selected from this dropdown box. The options available are the following
i. Custom Ordering

ii. Product Name
If the bestseller products are to be displayed in the order of product name, then this should be selected.
iii. Price
If the bestseller products are to be displayed in the order of price, then this should be selected.
iv. Date Added

b. Sort Order
The sort order to be used for bestseller products can be selected from this dropdown box. The options available are the following
i. Asc
If sorting is to be done in ascending order, then this option should be selected.
ii. Desc
If sorting is to be done in descending order, then this option should be selected.
c. Products per page
The number of products to be listed per page for show all of bestsellers should be set in this dropdown box.
d. Limit Bestseller In Left/Right Panel To
If bestsellers is set to display in left/right panel, the maximum number of products that can be displayed can be set using this option.
