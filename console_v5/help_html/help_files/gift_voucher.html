List Gift Vouchers
Gift vouchers can be searched by giving the voucher number, payment status and voucher added by options. Any combination of these can be used in the search criteria. Number of gift vouchers to be shown in each of the page and also the sort order to be used can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the gift voucher add page. There are 2 ways to go to the edit page of an existing gift voucher. One method is by clicking the voucher number and another by tick marking the required gift vouchers and clicking the "Edit" icon.
Delete operation can be done to more than one gift vouchers at a time. Tick mark the gift vouchers to be deleted and press the "Delete" icon.
Hidden status of gift vouchers can be changed directly from the listing page. Tick mark the gift vouchers which need status changing. Select the required status from the drop down box provided at the section "Change Status" and press the "Change" button.
The Payment Status of gift vouchers created from the console area will be always "Paid". When customer purchase a gift voucher, the value of this field will depend on the payment type and payment method used by customer while purchasing.
The field "Added By" indicates whether the gift voucher is added from the console area or have been purchase by the customer from the website.
The "Spent" field indicates the number of times the gift voucher had been used by customers in orders and also the total number of times it can be used. For e.g. the value 1/2 in this field will indicate that the gift voucher can be used maximum of 2 times and now it had been spend once.

Add Gift Voucher
Gift vouchers can be added through this section.
Key Fields
1. Activate On / Expires on
The start and end date between which the gift voucher will be active and can be used by customers while placing the orders will be based on the values in these fields..
2. Voucher Type
Gift vouchers can be of type "Value" or "%". If "Value" is selected in the drop down box, then an additional textbox namely "Discount Value" will be displayed to enter the discount value. If "%" is selected in the dropdown box, then an additional textbox namely "Discount %"will be displayed to specify the discount %.
3. Maximum Usage
Total number of times the gift voucher can be used can be set from this field.
4. User must be logged in to use this voucher.
If this gift voucher is to be set to used only by logged in customers, then this checkbox should be ticked.
Note: Voucher number will be generated automatically by the system when the voucher details are saved. In the general settings section, there is an option to specify the prefix for the gift voucher. If any value is given as prefix, then that will be set as prefix for the automatically generated gift voucher. The option to set the prefix for gift voucher can be accessed from Settings » General Settings » Main Shop Settings » Administration Area in console area.
Use "Save" button to save the changes.

Edit Gift Vouchers
Gift voucher details can be edited from this section. The details displayed in the edit page depends on whether the gift voucher has been created from the console area or whether it was purchased by the customer from the website.
Key Fields
Tab: Main Info
Gift voucher main details can be set from this section.
1. Voucher Number
System generated gift voucher number will be displayed here. This is not editable.
2. Created on
The date on which the gift voucher is created will be displayed here. This is not editable.
3. Payment Status
Payment status of current gift voucher will be displayed here.
4. Activated on
The start date of the range of dates for which the current voucher will be active. If the current gift voucher is created by customer from the website, this field will not be editable, otherwise it will be editable.
Note: If payment status of current gift voucher is not "Paid", then a "-- Not Activated yet --" will be displayed in this field. This field will get the actual value when the payment is marked as "Paid".
5. Expires On
The end date of the range of dates for which the current voucher will be active. If the current gift voucher is created by customer from the website, this field will not be editable, otherwise it will be editable.
Note: If payment status of current gift voucher is not "Paid", then a "-- Not Activated yet --" will be displayed in this field. This field will get the actual value when the payment is marked as "Paid".
6. Voucher Type
Gift vouchers can be of type "Value" or "%". If "Value" is selected in the drop down box, then an additional textbox namely "Discount Value" will be displayed to enter the discount value. If "%" is selected in the dropdown box, then an additional textbox namely "Discount %"will be displayed to specify the discount %.
7. Maximum Usage
Total number of times the gift voucher can be used can be set from this field.
8. User must be logged in to use this voucher.
If this gift voucher is to be set to used only by logged in customers, then this checkbox should be ticked.
Note: Use "Printer Friendly?" button to view a previews of the voucher details in a pop up and to print it.

Tab: Payment Details
This tab will be visible in the gift voucher edit page only if the gift voucher is purchased by customer from the website. The payment status and the details of payment will be displayed in this tab
1. Payment Status
Payment status of current gift voucher will be displayed in this field.
2. Change Payment Status
This option will be displayed only if the payment status of current gift voucher is not "Paid". A drop down will be displayed which allows to change the payment status
The values available in the dropdown box are the following
a. Payment Received
If payment status is to be marked as "Paid", then select this option in the dropdown box. When this value is selected, the following option will be displayed
i. Payment Method
The actual method by which the payment is received. This is just for future reference.
ii. Reason
This textarea allows to specify any reason / note when the payment is marked as "Paid". This is an option field.
Click on "Click here to Change the Payment Status" to save the changes. A mail will be send to the customer when payment is markes as "Paid". The template for this mail can be set form the "Email Templates" sub menu under "Settings Menu".
b. Payment Failed
If payment status is to be marked as "Payment Failed", then select this option in the dropdown box. When this value is selected, a textarea will be displayed which allows to specify the reason for payment failure. This is an optional field.
Click on "Click here to Change the Payment Status" to save the changes. A mail will be send to the customer when payment is marked as "Failed". The template for this mail can be set form the "Email Templates" sub menu under "Settings Menu" in console.
Note: When payment status is changed manually, then an additional field "Payment Status Changed By" will be displayed in this tab. This fields shows the name of console user who changed the payment status and also the date and time on which it has been changed.

Tab: Refund
This tab will be visible only if the gift voucher is purchased by customer from the website. If any refund related to current gift voucher is to be performed, this it can be done through this tab.
1. Amount to be refunded
The amount that is to be refunded should be specified in this textbox. Total Refundable Amount will be displayed next to this textbox. This is the maximum amount that will be allowed to be refunded.
2. Reason
Reason for refunding should be specified in this textarea. The note will be automatically saved in the notes section of gift voucher.
Click on "Click here to Refund" button to perform the refund operation.
When a refund is performed, its details will also be listed in this tab. The columns that will be displayed are the date of refund, the name of the console user who performed the refund and the amount that has been refunded.

Tab: Emails
This tab will be visible only if the gift voucher is purchased by customer from the website. Various emails related to current gift voucher which have been generated at different actions will be listed in this section.
Each of the rows in this section have the following columns.
1. Type
The type of email will be displayed in this column.
2. Subject
Subject of the email will be displayed in this column.
3. Mail Disabled?
This field indicated whether the respective email template was disabled at the time when tried to send the email for the first time. Email templates can be disabled by tick marking the disable option from the "Email Templates" section under "Settings" menu in console.
4. Send Atleast Once
This field indicates whether the current mail has been send out atleast once.
5. Details
When clicked on the "Details" caption, displayed on each of the rows in this list, the details of email will be displayed. The exact content of the mail can be viewed when clicked on the "Message" caption.
In order to send the mail again, click on "Send this Mail" button. When this button is clicked, the mail will be send again and also an entry will be displayed in the "Email Resend History" section to indicated the date and time and also the console user who resend the email..

Tab: Notes
Notes related to current gift voucher can be managed in this section. Already added notes will be listed under "Existing Notes" section. Use the "x" icon associated with each of the existing notes to to delete the note.
To add a new note, enter the note in the textarea displayed under the "Add Notes" section and click on "Save Note" button to save it. Saved notes will get listed in the "Existing Notes" section.
