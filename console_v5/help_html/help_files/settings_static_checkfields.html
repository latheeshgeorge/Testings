How Static Fields Works?
This section allows you to set the order and error message to be used while displaying various static fields in various sections on the site such as Checkout, voucher purchase, Cheque payment method etc. It is possible to decide the order in which the fields to be displayed in the site, mark a field as compulsory and also the error message to be shown if compulsory field is left blank.
1. Select type
This drop down box show the list of sections for which the settings can be done for various static pages. The values that will be displayed in this dropdown box are the following
a. Personal
When this option is selected, the fields that will be displayed to specify the billing address in the checkout page will be listed in this section.
b. Delivery
When this option is selected, the fields that will be displayed to specify the delivery address in the checkout page will be listed in this section.
c. Card
When this option is selected, the fields that will be displayed to specify the credit card details in the checkout page will be displayed.
d. Cheque
When this option is selected, the fields that will be displayed to specify the cheque details in the checkout page will be displayed.
e. Voucher
When this option is selected, the fields that will be displayed to specify the details of the customer who pays for the gift voucher will be displayed.
2. Field Order
The order in which the fields are to be displayed in the website can be set using the textbox displayed with each of the row in the list.
3. Compulsory?
If a field is compulsory, then this check box should be ticked for it. If this is ticked for any of the fields, then customers will not be allowed to proceed until they specify value for it.
4. Error Message
The error message to be displayed to the customer in case if field is compulsory should be set in these textboxes.
Click on "Save Changes" button to save the changes.
