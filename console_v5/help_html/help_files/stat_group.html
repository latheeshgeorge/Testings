How Static Page Menu Works?
Static Page menu can be called as a static page container. More than one static pages can be assigned to a static page menu and this menu can be displayed at various positions in the website in different styles. The various positions can be Top menu, Left menu, Right menu , Bottom menu etc. Same static page can be assigned to more than one static page menus.

List Static Page Menu
Static page menu can be searched by giving the name of the menu. Number of menus in each of the page and also the sort order to be used can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go to the static page menu add page. There are 2 ways to go to the edit page of an existing static page menu. One method is by clicking the name of the menu and another by tick marking the required static page menu and pressing the "Edit" icon.
Delete operation can be done to more than one menus at a time. Tick mark the menus to be deleted and press the "Delete" icon.
Status of static page menus can be changed directly from the listing page. Tick mark the menu which need status changing. Select the required status from the drop down box provided at the section "Change Hide Status" and press the "Change" button.
The field "Layout(position)(order)" in the menu lising is shown as a drop down box. This is because same menu can be set to show in more than one layout/position. This drop down box is just for an informative purpose. No operation is associated with this drop down box.

Add Static Page Menus
Static page menus can be added through this section.
Key Fields
1. Hide name
This field decides whether the name of the menu is to be displayed to the customer. If ticked, then the name will be hidden in the site. When menus are shown in the top or bottom, the menu name will be always hidden irrespective of this setting.
2. Display Location
This field allows to select the areas in which the menu is to be displayed. The list box shows the list of layouts and its positions to which the static page menus can be assigned. The values in this list box have the format "Layout Name (position)". There can be more than one layout for a site. "Layout name" refers to the layouts existing for the site. "Position" refers to the positions to which static page menus can be set for the site (eg. right, left, etc). It is possible to show same menu in more than one layout/position. Use "ctrl" button to make multiple selection in this list box.
Note: Depending on the display location set for the static page menu, the static page menu will be displayed in various positions on the site such as top, left, right etc. The name of the static page menu will be displayed as the title for the static page menu by default in the website. In order to change this, you have to use the edit title icon of respective positions from the Edit Website Layout Section in the console (Settings » Design Settings » Edit Website Layout).
3. Show in all Pages
Menus can be set to show in all pages or only for selected product or categories or static pages. From the Static page menu add page it is possible only to specify whether it is to be shown for all pages or not. Mapping to certain product or categories or static pages can be done from the static page menu edit page.
4. Show home page link
If the home link is to be included in the static page menu, this option should be selected.
5. Show sitemap link
If the sitemap link is to be included in the static page menu, this option should be selected.
6. Show help link
If the help link is to be included in the static page menu, this option should be selected.
7. Show FAQ link
If the faq link is to be included in the static page menu, this option should be selected.
8. Show saved search link
If the saved search link is to be included in the static page menu, this option should be selected.
9. Show XML sitemap link
If the XML sitemap link is to be included in the static page menu, this option should be selected.
Click on "Save" button to save the changes.

Edit Static Page Menus
Static page menu details can be edited through this section.
Key Fields
Tab: Main Info
Static page menu main details can be set from this section.
1. Hide name
This field decides whether the name of the menu is to be displayed to the customer. If ticked, then the name will be hidden in the site. When menus are shown in the top or bottom, the menu name will be hidden always irrespective of this setting.
2. Display Location
This field allows to select the areas in which the menu is to be displayed. The list box show the list of layouts and its positions to which the static page menus can be assigned. The values in this list box have the format "Layout Name (position)(order)". There can be more than one layout for a site. "Layout name" refers to the layouts existing for the site. "Position" refers to the positions to which menus can be set for the site (eg. right, left, etc). "Order" is the sort order in which the menu will be listed in the site. The order is important if more than one component is placed in same position. Components set to each of the positions are displayed in the site in the sort order shown in "Order". It is possible to set the same menu to more than one area. Use "ctrl" button to make multiple selection in this list box. By default the value of "Order" will be set to 0. It will be changes automatically when the position of each of the components are set from the "Edit website layout " section in the site console.
Note: Depending on the display location set for the static page menu, the static page menu will be displayed in various positions on the site such as top, left, right etc. The title for the static page menu will not get modified in the website even when the menu name is changed from the edit page. In order to change the title you have to use the edit title icon of respective positions from the Edit Website Layout Section in the console (Settings » Design Settings » Edit Website Layout).
3. Show in all Pages
Menus can be set to show in all pages or only for selected product or categories or static pages. If "Show in all pages" is ticked then the menu will be displayed in all the pages irrespective of the categories or products or static pages for which it is assigned to.
4. Show home page link
If the home link is to be included in the static page menu, this option should be selected.
5. Show sitemap link
If the sitemap link is to be included in the static page menu, this option should be selected.
6. Show help link
If the help link is to be included in the static page menu, this option should be selected.
6. Show FAQ link
If the FAQ link is to be included in the static page menu, this option should be selected.
7. Show saved search link
If the saved search link is to be included in the static menu, this option should be selected.
8. Show XML sitemap link
If the XML sitemap link is to be included in the static menu, this option should be selected.
Click the "Save" button to save the changes before switching to other tabs.

Tab: Static Pages in this Menu
Static pages which are mapped to this static page menu will be show in this area.
To assign more static page to the menu, click on "Assign more" button under this section. This will open up a page which shows the list of static pages which are not yet mapped to the current menu. Tick the static pages to be mapped and click the "Done" button to make the assignment.
To unassign static pages from the menu, tick the static pages to be unassigned and click the "Un Assign" button in this section.
While listing the static pages mapped with the current menu, it is possible to set the order in which the static pages will be displayed in the menu. Click the "Save Order " button to save the sort order.

Tab: Display Menu for Following Categories
If the menu is to be shown while viewing the details page of certain categories in the website , then those categories should be set from this section. This setting will be ignored if the "Show in all pages" option is set for the menu.
Use "Assign More" button to assign more categories to this section
Categories assigned to this section can be unassigned by tick marking the required categories and clicked the "Un Assign" button in this section.

Tab: Display Menu for Following Products
If the static page menu is to be shown while viewing the details page of certain products in the website, then those products should be set from this section. This setting will be ignored if the "Show in all pages" option is set for the menu.
Use "Assign More" button to assign more products to this section

Tab: Display Menu for Following Static Pages
If the static page menu is to be shown while viewing certain static pages in the website, then those static pages should be set from this section. This setting will be ignored if the "Show in all pages" option is set for the menu.
Use "Assign More" button to assign more static pages to this section
Static Pages assigned to this section can be unassigned by tick marking the required static pages and clicked the "Un Assign" button in this section.
