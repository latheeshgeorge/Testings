How Email Templates Works?

Templates for various emails that is send from the website and console can be managed from this section.

 
1. List Email Templates
2. Edit Email Template
 
List Email Template
Email templates can be searched based on email title. Then number of templates to be displayed in each of the pages and also the sort order in which the templates to be displayed can be specified. Click on "Go" button to start the search operation.
It is possible to disable the sending of various emails from this section. To block an email from being send, the respective template should be marked as disabled. Tick mark the templates to be disabled and click on "Save" button to save the selection.
Details of a particular template can be viewed either by clicking on the title of the template from the list or by tick marking the respective template and clicking on the "Edit" icon.

Edit Email Template
Details of selected email template can be managed from this section.
Key Fields
1. Template From email
The from address to be used while sending the current email should be specified in this textbox.
2. Template Subject
The subject to be used while sending the current email should be specified in this textbox.
3. Template Letter Content
The content for the email should be specified in this html editor. Various codes can be placed in the templates. The codes will be replaced with respective values at the time the mail is send. The codes that can be placed are displayed on the right hand side of the content html editor.
Click on "Update" button to save the changes.
