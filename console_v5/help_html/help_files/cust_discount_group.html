How Customer Discount Groups Works?
This section allows to define the discount groups for customers. Discount group should be created and customers can be assigned to the group. It is also possible to assign products to the discount group so that discount will be applicable when the selected customers purchase the assigned products. So basically customer discount groups may or may not have products assigned to them. If no products are assigned to a group, then the discount given for the group will be applicable for all product purchased by the customers mapped with that discount group. On the other hand if products are assigned to a discount group, then group discount will be applicable only for those products.

List Customer Discount Groups
Customer discount groups can be searched by giving the group name. Number of groups to be shown in each of the page and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the customer discount group page. There are 2 ways to go to the edit page of an existing customer discount group. One method is by clicking the group name and another by tick marking the required customer discount group and clicking the "Edit" icon.
Delete operation can be done to more than one customer discount groups at a time. Tick mark the groups to be deleted and click the "Delete" icon.
Hidden Status of customer discount groups can be changed directly from the listing page. Tick mark the discount groups which need status changing. Select the required status from the drop down box provided at the section "Change Status" and click the "Change" button.

Add Customer Discount Group
Customer discount groups can be added through this section.
Key Fields
1. Customer Discount Group Name
The name for the customer discount group can be set in this text box.
2. Discount Allowed (%)
The % of discount to be set for the current discount group can be given in this textbox.
Click on "Submit" button to save the details.

Tab: Customers in this group
Customers assigned to the current customer discount group will be listed in this section.
Click on "Assign More" button to view the list of customers to be assigned to the current group. When this button is clicked, a new page will be displayed which show the list of customer to be assigned. Tick mark the required customers and click the 'Assign' button to make the assignment.
Customers assigned to the current discount group can be unassigned by selecting the already assigned customers and the clicking on the "Un assign" button.

Tab: Products in this Group
The list of products which are linked with the current customer discount group will be listed in this section.
Click on "Assign More" button to assign more products to the current customer discount group. When this button is clicked, a new page will be displayed which shows the list of products to be assigned. Tick mark the required products and then click on "Assign" buttom to make the assignment.
Products assigned to current discount group can be unassigned by selecting the already assigned products and clicking on the "Un assign" button.
There are some restrictions for assigning customers and products to a customer discount group.
Criteria 1: Same customer cannot be there in more than one group which no products mapped to it.
Criteria 2: Same customer cannot be there in more than one group which have the same set of products assigned to it.
