List Banners
Banners can be searched by giving the banner title. Number of banners to be shown in each of the pages and also the sort order to be used can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the banner add page. There are 2 ways to go to the edit page of an existing banner. One method is by clicking the title of the banner and another by tick marking the required banner and clicking the "Edit" icon.
Delete operation can be done to more than one banner at a time. Tick mark the banners to be deleted and press the "Delete" icon.
Status of banners can be changed directly from the listing page. Tick mark the banners which need status changing. Select the required status from the drop down box provided at the section "Change Status" and press the "Change" button.

Add Banners
Banners can be added through this section.
Key Fields
1. Banner Title
Title for the banner can be set in this textbox.
2. Show in all page
Banners can be set to show in all pages or only for selected product or categories or static pages. From the banner add page it is possible only to specify whether it is to be shown for all pages or not. Mapping to certain product or categories or static pages can be done from the banner edit page.
3. Display Location
This field allows to select the areas in which the banner is to be displayed. The list box shows the list of layouts and its positions to which the banner can be assigned. The values in this list box have the format "Layout Name (position)". There can be more than one layout for a site. "Layout name" refers to the layouts existing for the site. "Position" refers to the positions to which banners can be set for the site (eg. right, left, etc). It is possible to show same banner in more than one layout/position. Use "ctrl" button to make multiple selection in this list box.
4. Banner Type
Depending on the value selected for banner type, various options will be displayed to the user.
a. Image Upload
This option allows to upload an image to be used as the banner. When this option is selected the following 2 additional fields will be displayed
1. Select File
The image to be uploaded as banner should be selected in this field. There is a checkbox next to this upload box namely "Resize Image". If this is ticked, the image being uploaded will be resized automatically, otherwise it will be uploaded in its original size.
2. Link for Banner
If any link is to be given for the current banner, then it should be specified in this textbox.
b. Image URL
If the image to be displayed as banner is in a different domain, the full url of that image can be set in this textbox. (e.g. http://www.domainname.com/imagename.gif)
1. Specify image location
The full url (including http/https) of the banner image should be set in this textbox.
2. Link for Banner
If any link is to be given for the current banner, then it should be specified in this textbox.
c. Text/HTML
If the advert include just plain text or HTML scripts, then this option should be selected. When this options is selected, a text area will be displayed which allows to enter the text/html to be displayed as banner.
d. Flash
If flash file is to be displayed as advert, then this option should be selected. When this option is selected, a file upload box will be displayed from where the flash file can be uploaded.
5. Banner Link Open in
This option will be visible only if the banner type supports links for the banner. There are 2 options for this dropdown box.
a. New Window
If this is selected in the dropdown box, then the banner link will open in a new window.
b. Same Window
If this option is selected in the dropdown box, then the banner link will open in same window.
7. Active Period
If the banner is to be displayed only during a given range of dates, then this can be done from this section. When the "Active Period" is ticked, 2 new textboxes will be displayed which allows to specify the from and the to date ranges between which the banner is to be displayed in the website.

Tab: Show Banner in these Categories
If the current banner is to be displayed while viewing the details page of certain categories only, then those categoreis should be set in this section. The categories set in this section will be ignored if the "Show in all Pages" option is ticked for current banner.
Use "Assign More" button to view the page which allows to select the categories to be assigned. Select the required categories and click on "Assign Categories " button to make the assignment.
Use "Un Assign" button to unassign any of the categories mapped in this section. Tick mark the categories to be unassigned and click on "Unassign" button.

Tab: Show Banner in these Products
