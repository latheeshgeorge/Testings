List Shop by brands
Shop by brand can be searched using the following options
1. Shop Name
3. Shop by Brand Menu
Combination of the above can be used to search for shops existing in the site.
Use "Add" link to go the Shop by brand add page. There are 2 ways to go to the edit page of an existing Shop by brand. One method is by clicking the name of rhe Shop by brand and another by tick marking the required shop and clicking the "Edit" icon.
Delete operation can be done to more than one Shop by brands at a time. Tick mark the Shop by brands to be deleted and click the "Delete" icon.
Settings can be done to multiple shops at a time. Click on "Settings for Multiple Shops" link to go to the page where settings can be done to multiple shops.

Add Shop by brands
Shop by brands can be added through this section.
Key Fields
1. Parent Shop
The parent shop for the current shop can be selected from this drop down. By default "-- Root Level -- " will be shown in this drop down box, which mean there will not be any shops to the top of current shop in hierarchy.
2. Shop Menu
The shop menus to which the current shop is to be mapped with can be done by selecting the required shop menu from this list box. Use "Ctrl" button to make multiple selection.
3. Shop Description
The long description (if specified) will be displayed in the shop details page. Html tags are supported by this section.
4. Subshop List Type
Allows to set where the subshops of current shop (if any) should be displayed. The options available are
i. Show in Middle Area
Shows the subshops only in the middle area.
ii. Show below Selected shop
Shows the first level subshops of current shop below the current shop itself. Subshops in levels other than the first level will be shown in the middle area only.
5. Product Display Method
This option helps to set how the products should be displayed in the shop details page in the website. The options shown in the dropdown box depends on the type of layout used by the site.
6. Fields to be displayed for products
The fields to be included while displaying the products of current shop in details page can be set from here. The fields that are to displayed should be ticked. It is compulsory that atleast one of the fields should be ticked. The options available are
1. Product Image
2. Product Title
3. Product Description
4. Product Price
Click on "Save" button to save the details.

Tab: Sub shops
Subshops of current shop will be displayed in this section.
Use "Save Order" to save the sort order for subshops. To save new sort order, tick mark the selected shops, enter the required sort order value in textboxes and click the "Save Order" button.
Back to Top
Tab: Images
Images assigned to the shop will be listed in this section. Images can be assigned to shops from this section. It is possible to assign more than one image to a shop.
Use "Assign More" buttom to assign more images. When clicked on this button a page will be shown with the list of images existing in the site. Select the required images and click the "Assign" button.
Use "Un assign" button to unassign any image assignments for current shop. Tick mark the images to be unassigned and then click on the "Un assign" button.
Sort order and the title can be set for the images assigned to the shop. To save these details, tick the images for which the details to be saved and then click the "Save Details" button.
If the option "Show any product image" option is ticked, then the images assigned to the shop will be ignored and the system will automatically pick an image from any one of the products assigned to current shop.
Back to Top
Tab: Products in shop
Product which are mapped to current shop will be listed in this section.
Use "Assign More" button to assign more products to current shop. When thick button is clicked a new page will be displayed. Select the required products to be mapped to current shop and then click on "Assign Selected" button to make the assignment.
Use "Unassign" button to unassign any products assigned to current shop. Tick mark the products to be unassigned and then click on the "Unassign" button.
Sort order can be set for the products assigned to the shop. To save these details, tick the products for which the details to be saved and then click the "Save Order " button.
