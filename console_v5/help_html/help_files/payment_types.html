How Payment Types Works?
Lists of payment types that are enabled for your website will be displayed in this page. Here there is provision for disabling or enabling the payment types for your website by unchecking or checking the check boxes. If the payment type "credit card" is selected, after saving the changes an additional icon will be listed beside the credit card option which allows to select the credit card types to be allowed to be used in the website.
Each of the payment types to be used in your website should be selected by tick marking the checkboxes displayed with each of the payment types.
If the payment type "Credit Card" is there in the list, then an icon will be displayed just before the checkbox which allows to select the credit card types to be displayed in the website. When clicked on this icon a new page will be displayed with the list of credit cards supported by the system. Tick mark the types of credit cards to be used in the website. The sort order in which the credit cards to be displayed in the website can also be set from this page. The sort order can be specified in the textbox provided with each of the payment types. Click on "Save Changes" button to save the details.
If you wish to show a custom icon for the selected payment types when it is displayed in your website, it can be set from this section. Images to be assigned to the payment types should be uploaded in the image gallery.
Once the images are uploaded to the image gallery, click on the icon in the "Icons" column of payment types. When clicked on this icon, user will be taken to the image gallery to make the assignment.
When a custom image is assigned to a payment type, it will be displayed at the right end of the payment type in the list. A new icon will be displayed in "Icons" column of payment types for which the custom images are assigned. Click on this icon to remove the assigned custom image.
Click on "Save Changes" button to save the changes.
