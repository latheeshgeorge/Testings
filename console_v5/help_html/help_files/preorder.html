How Preorder Settings Works?
The list of all products which are set to allow pre-ordering will be listed in this page.
Products in preorder can be searched based on the following
1. Product Name
2. Category
3. Product Id
4. Vendor
5. Bulk discount
6. In stock date range
Click on "Go" button start the search operation.
The details such as Product name, Total Preorder Allowed, instock date and also the sort order is displayed in this page. The sort order in which the products are to be displayed in the preorder section can be set by specifying the sort order in the textboxes displayed in the list. Specify the sort order in the textboxes and click on "Save Order" button to save the order.
