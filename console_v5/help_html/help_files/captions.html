How Headings & Captions Works?

Various headings and textual data used in different section of your website can be managed from this section.

List Captions
Searching can be done by giving the caption. Available sections are shown in a drop down box. It is possible to narrow the search by selecting a section from the drop down box. By default "All" will be selected in the sections dropdown box. The number of captions to be shown in each of the pages and the order in which the result is to be displayed can also be specified. Click on "Go" button to start the search operation.
Captions can be edited either by clicking on the caption in the list or by tick marking a caption and then clicking on the "Edit" icon.

Edit Captions
Details of selected email template can be managed from this section.
Key Fields
1. Sections
The list of sections will be displayed in this dropdown box. The section of the caption currently being edited will be selected by default in this dropdown box.
2. General Settings Caption
The actual text that is being displayed in the website will be displayed in this textbox.
Click on "Update" button to save the changes.
