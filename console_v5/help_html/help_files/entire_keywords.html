How Entire Keywords Works?
This section shows the list of entire keywords used in the site. All Keywords set to various sections of the site by SEO engineers will be listed in this section.
List Entire Keywords
Entire keyword list can be search by giving the keyword. Number of keywords to be shown in in each of the pages and also the sort order to be used can be specified. Use "Go" button to begin the search operation.
Delete operation can be done to more than one keywords at a time. Tick mark the keywords to be deleted and click the "Delete" icon. If the keywords being deleted is used in some part of the site, then those assignments will also be removed.
The field "Used in Site?" tell us whether or not the keyword is used anywhere in the site. If the keyword is used in some section of the site, then this field will display "Yes" otherwise it will display "No"
To edit a keyword, make the changes to the keyword in the respective texbox and then click on "Save Keywords" button to save the changes. When keywords are changed, the change will be applied to all areas where those keyword is mapped.
