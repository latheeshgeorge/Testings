How Gift Wrap Settings Works?
Settings required for gift wrap can be done from this section.
Key fields
1. Gift Wrap Per
Gift wrap option can be set to be applied for each of the products in the order or to the order as a whole. This drop down box allows to choose this settings. The values available in this dropdown box are
a. Item
If gift wrap is to be applied to each of the products in the order, then select this option.
b. Order
If gift wrap is to be applied to the order as a whole, then this option should be selected.
2. Gift Wrap Minimum Price
This is the base price for gift wrap. That is, if the gift wrap option is selected from the viewcart page in the website, the the amount specified in this textbox will be added to the total gift wrap amount in the cart .
3. Gift Wrap Message Price
This is the price for additional gift wrap message. Option to selet the gift wrap message will be displayed in the viewcart page only if the gift wrap option is selected. If the gift wrap message option is selected, then the amount set in this textbox will be added to the total gift wrap amount in the cart and a section will be displayed to enter the additional message.
4. Gift Wrap Active
This option allows to set whether the giftwrap option is active in the website. The gift wrap option will be displayed in the view cart page of the website only if gift wrap option is active.
