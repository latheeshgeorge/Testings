How Meta Description Templates Works?
This page allows to define the template to be used for meta descriptions in various sections of the site. The template set for a page will be ignored if the meta description is set directly for that page from the "Manage Site Titles & Descriptions" submenu of SEO menu in the console ( Settings » SEO » Manage Site Titles & Descriptions).
Templates can be defined for the following sections of the site.
1. Home
If meta description is not directly set for the home page, this template will be used. If codes are placed in the template, it will be replaced with the respective values when displayed in website.
2. Static Pages
If meta description is not directly set for a static page, this template will be used. If codes are placed in the template, it will be replaced with the respective values when displayed in website.
3. Product Details Page
If meta description is not directly set for a product page, this template will be used. If codes are placed in the template, it will be replaced with the respective values when displayed in website.
4. Category Details Page
If meta description is not directly set for a category page, this template will be used. If codes are placed in the template, it will be replaced with the respective values when displayed in website.
5. Search Result Page
If meta description is not directly set for a search keyword , this template will be used. If codes are placed in the template, it will be replaced with the respective values when displayed in website.
6. Other Pages
For all other pages for which the meta description cannot be set directly from "Manage Site Titles & Descriptions" section, this template will be used. If codes are placed in the template, it will be replaced with the respective values when displayed in website.
7. Search Page Top Content
While displaying the search result, if description is not directly given for the keyword being searched, then this template will be used. This description will be placed above the search result product listing in the website. . If codes are placed in the template, it will be replaced with the respective values.
The codes which can be placed in above templates are as follows
Code Description
[title] Site Title
[keywords] Keywords for the page
[first_keyword] Show First Keyword
Click on the "Save Templates " button to save the templates for various sections of the site.
