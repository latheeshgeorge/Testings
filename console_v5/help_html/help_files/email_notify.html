How Email Notifications Works?
While registering or profile editing, user have the option to decide whether they wish to receive email notification regarding new products and discounted products. Console users creates email notifications to be send periodically to all customers who have requested to receive notifications. Email notification sending will be an automated process.

List Email Notifications
Email notifications can be searched by giving the notification title. Number of notification to be shown in each of the page and also the sort order to be used can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the email notification add page. There are 2 ways to go to the edit page of an existing email notifications. One method is by clicking the title of the notification and another by tick marking the required notification and clicking the "Edit" icon.
Delete operation can be done to more than one email notification at a time. Tick mark the notifications to be deleted and press the "Delete" icon.

Add Email Notifications
Email notifications can be added through this section.
Key Fields
1. Notification Template
List of email notification templates will be listed in this section. If required, any of the templates can be selected. When a template is selected, the page will get reloaded and the notification title and also the notification contents will be automatically loaded to respective textboxes based on the selected template.
2. Notification Title
Title for the email notification can be set from this section. Title will be send as the subject for the email notification.
3. Notification Contents
The content for email notification mail is to be set up using this html editor. If any template is selected, by default the content set in the selected template will be displayed in this section. User can make any necessary modifications if required.
The codes which can be used in the notification contents are displayed below the notification html editor. When notification are send to customers, the notification codes will be replaced with its respective values.
Click on "Continue" button to save the changes. Once the details are saved, user will be redirected to the email notification edit page to set the notification settings.

Edit Email Notifications
Email notifications can be edited through this section.
Key Fields
Tab: Main Info
Main details related to email notification can be set from this section.
1. Current Template
If any template has been used in building this email notification, then the name of that template will be listed in this section. This is a non editable field.
2. Do you want to change template
In order to change the current template or to choose a new template, tick mark this option. The page will get reloaded and a new dropdown box namely "Notification template" will be displayed which shows the list of all available templates.
3. Notification Title
Title for the email notification can be set from this section. Title will be send as the subject for the email notification.
4. Notification Contents
The content for email notification mail is to be set up using this html editor. If any template is selected, by default the content set in the selected template will be displayed in this section. User can make any necessary modifications if required.
The codes which can be used in the notification contents are displayed below the notification html editor. When notification are send to customers, the notification codes will be replaced with its respective values.
Click on "Continue" button to save the changes. Once the details are saved, user will be redirected to the email notification settings tab.

Tab: Notification Settings
Settings for the notification mails can be set in this section
1. Send Notification Mails
This option allows to decide whether the current email notification mail is to be send weekly of monthly.
a. Weekly
If this option is ticked, another dropdown box will be displayed which allows to pick the weekday on which the mail is to be send.
b. Monthly
If this option is ticked, one additional dropdown box will be displayed which allows to pick the day of the month on which the mail is to be send.
2. Settings for New Products
If new products are to be included in the current email notification, then the following things should be set.
a. No. of new products to be included in email.
The maximum number of new products to be included in current email notification. If new products are not to be included in current email notification, then set this value to 0.
b. Select Categories.
If new products are to be picked only from certain categories, then those categories should be selected from this multi-select dropdown box. If all categories are to be considered, then select the " -- All -- " option.
3. Settings for Discounted Products
If discounted products are to be included in the current email notification template, then the following things should be set.
a. No. of discounted products to be included in email.
The maximum number of discounted products to be included in current email notification. If discounted products are not to be included in current email notification, then set this value to 0.
b. Select Categories.
If discounted products are to be picked only from certain categories, then those categories should be selected from this multi-select dropdown box. If all categories are to be considered, then select the " -- All -- " option.
c. How to pick discounted products?
This option allows to set additional criteria to pick the discounted products. The 2 options available are
1. Pick Randomly
If this option is selected, the discounted products will be picked randomly.
2. Discount Range
If this options is selected, two additional text boxes will be displayed to specify the From and To discount values. These From and To values will be used while picking the discounted products. Products whose discount value fall with the From and top values only will be considered.
Note: Either new products or discount products settings is compulsory. User will not be allowed to continue to next step with both number of new products and discounted products set to 0.
Click on "Continue" button to save the changes and continue to the preview tab
Tab: Preview Notification
This section shows a preview of the mail. The new or discounted products shown in this preview will be based on the current availability of new or discounted products. This is just to give an idean to the customer of how the actual email will look like. The new and discounted products shown in preview may not be the one's that will be send to the customer when the notification is actually send.
