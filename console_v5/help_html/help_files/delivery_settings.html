List Delivery Methods
The list of delivery methods that can be used in your website will be listed in this section. At a time only of them can be used.
Select the required delivery method and click on "Save" button. This will save the selection and when retunred to the page a new button will be displayed next to the selected delivery method. When clicked on this button, various options to be set for the selected delivery method will be displayed in a new page.
Split delivery is a facility which allows to send items in one order at different periods. For example if an order has 2 products. If one of the products is in pre order and another is in stock. The product in preorder will be available only at a future date. In such situation customers can decide whether they want the available product to be delivered at the earliest and the product in pre order product at the time it comes in stock or both the products should be sent together when both are in stock. Customers will be given the option to make this decision only if the "Allow Split Delivery" is set to 'Y' in this page.
It is possible to specify groups for delivery methods. Groups allows to define different price groups for the selected delivery method. For e.g. With in 7 days, next day itself etc. The price and delivery calculation criteria can be different for each of such groups.

The list of delivery methods are the following
1. None
If no delivery method is to be set in your website, then select this option.
2. Charge Delivery by Weight
If the delivery charge is to be calculated based on the total weight of products in the order, then this option should be selected. If this is the delivery method you wish to use in your website, then the weight for each of the products should be set correctly from the product add/edit page in your site console. When clicked on the "Save" button, the selection will be saved and when returned to this page, a new button will be displayed next to this option with the caption "Set Values". When clicked on this button, a new page will be displayed which allows to set the conditions to be used to calculate the delivery charge.
The options that will be displayed are
Option Charge (£)
More than zero but less than   kg
More than previous but less than   kg
More than previous but less than   kg
More than previous but less than   kg
More than previous but less than   kg
This allows to specify the criteria and the charges to be used to calculate the delivery charges in your website.
By default 5 rows will be displayed, but when details are saved, 5 new rows will be automatically created to specify more values.
The above settings will work as
a. If total weight of products in order is less than 100kg, then £10 will be the delivery charge.
b. If total weight is less than 400kg, but greater than or equal to 100kg, then £20 will be the delivery charge.
c. If total weight is greater than or equal to 400kg, then also £20 will be taken as the delivery charge. That is if the total weight of order does not fall in any of the criteria specified above, then the price set for the last option will be considered.
If delivery method groups exists in your website, then option to specify the calculation criteria and the charges seperately for each of the delivery method groups will be displayed in this page. That is, the above given tabular section will be displayed under each of the delivery method groups.
Click on "Submit Rates" to save the details and return back to delivery settings page.
Click on "Submit and enter more rates" to save the details and return back to the current page to enter more values.

List Delivery Method Groups
Delivery method groups allows to define different price groups for the selected delivery method. For e.g. With in 7 days, next day itself etc. The price and delivery calculation criteria can be different for each of such groups.
Delivery method groups can be searched by entering the group name . Sort order and the number of groups to be listed per page in the search result can be set. Click the "Go" button to search for the users.
Use "Add" link to go to the page which allows to add new delivery method groups. There are 2 ways to go to the edit page of an existing delivery method group. One method is by clicking the name in the list and another method is by ticking the delivery method group to be edited and clicking the "Edit" icon.
In order to delete delivery method group(s), mark the tick box of respective delivery method group(s) and click the "Delete" icon. It is possible to delete more than one group at a time.
Status of groups can be changed using the section "Change Hide Status to". Tick mark the group whos status are to be changed and select the required status from the drop down and then press the "Change" button.
The sort order in which the delivery method is to be displayed in the website can be set with the help of the textbox shown with each of the groups in the list. Specify the sort order in the textboxes and click on "Save Order" button to save the sort orders.
