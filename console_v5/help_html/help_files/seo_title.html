How Manage Page Titles and Meta Descriptions Works?
This section allows the search engine optimisers to set the page title and page meta descriptions to be displayed for various pages in the website.
Depending on the type of value selected in the "Select Type" drop down box, various section will be displayed with 2 textboxes each, one to specify the page title and another to specify the meta description.
 
1. Home
2. Categories
3. Products
4. Shops
5. Combo Deals
6. Shelves
7. Best Sellers
8. Static Pages
9. Saved Search
10. Saved Search Main
11. Help
12. Registration
13. Sitemap
14. Forgot Password
15. Site Reviews
 
1. Home
When this value is selected in the dropdown box, the page will get reloaded and a set of 2 textboxes will be displayed. The page title and meta description set for the home page from this section will be displayed in the website. Click on "Save Titles & Descriptions" to save the details.
Back to Top
2. Categories
When this value is selected in the dropdown box, the page will get reloaded and the list of all categories existing in the site will be displayed in the page. For each of those categories a set of 2 textboxes will be displayed. The title and meta description set in the textboxes will be used in the website while viewing the details page of categories. Click on "Save Titles & Descriptions" to save the details.
If the category being displayed is a subcategory, an icon will be displayed in the last column of such rows. When mouse is placed on this icon, the hierarchy of that category will be displayed as a tool tip text.
Categories can be searched by giving the category name and also by selecting the parent category. Combination of these 2 can be used to search for categories. Click on "Go" button to begin the search.
