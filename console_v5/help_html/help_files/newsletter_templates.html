How Newsletter Templates Works?
Templates that can be used in the newsletter section can be set from this section. Templates created from this section will be listed in drop down in newsletter section.
 
1. List Newsletter Templates
2. Add / Edit Newsletter Templates
 
List Newsletter Templates
Newsletter templates can be searched based on template name. Number of templates to be shown in each of the pages and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the newsletter templates add page. There are 2 ways to go to the edit page of an existing template. One method is by clicking the template name and another by tick marking the required template and clicking the "Edit" icon.
Delete operation can be done to more than one templates at a time. Tick mark the templates to be deleted and click the "Delete" icon.
Hidden Status of templates can be changed directly from the listing page. Tick mark the templates which need status changing. Select the required status from the drop down box provided at the section "Change Hidden Status" and press the "Change" button.

Add / Edit Newsletter Templates
Newsletter templates can be added / edited through this section.
Key Fields
1. Template Name
Name for the template can be set in this textfield. The name will be used to identify the template in various sections of console.
2. Template Content
Template content can be set in this section. This is the body of the email to be send as newsletter. This field can contain html tags.
3. Product Layout Design
The layout and color schemes in which the products to be placed in the newsletter can be set from this section. This field can contain html tags. The content in this field can have the following tags. Each of the following tags will be replaced with the original content while sending the newsletter.
a. [IMG]
This tag, if specified, will be replaced with the image of products.
b. [TITLE]
This tag, if specified, will be replaced wiht the title of products.
c.[DESCRIPTION]
This tag, if specified, will be replaced with the short description of products.
d. [PRICE]
This tag, if specified, will be replaced with the price of products.
4. Hide Template
This option allows to decide whether the current newsletter template is active. Only active templates will be listed in the newsletter template selection sections in console.
