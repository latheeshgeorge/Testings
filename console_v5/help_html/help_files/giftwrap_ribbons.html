How Giftwrap Ribbons Works?
Gift wrap ribbons section allows to define the details of ribbons that will be used for wrapping the gifts. Giftwrap details will be displayed in the cart page.
 
1. List Giftwrap Ribbons
2. Add Giftwrap Ribbons
3. Edit Giftwrap Ribbons
 
List Giftwrap Ribbons
Ribbons can be searched by giving the name of the ribbon. Number of ribbons in each of the page and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the ribbon add page. There are 2 ways to go to the edit page of an existing ribbon. One method is by clicking the name of the ribbon and another by tick marking the required ribbon and pressing the "Edit" icon.
Delete operation can be done to more than one ribbons at a time. Tick mark the ribbons to be deleted and press the "Delete" icon.
Status of ribbons can be changed directly from the listing page. Tick mark the ribbons which need status changing. Select the required status from the drop down box provided at the section "Change Status" and press the "Change" button.
Order in which the ribbons should be displayed in the cart page can be set by giving sort order values in textbox given for each of the ribbons in the list. Click the "Save Order" button to save the sort order.

Add Giftwrap Ribbons
Giftwrap ribbons can be added through this section.
Key Fields
1. Ribbon name
The name that will be shown for the ribbon while displayed in the cart page
2. Extra Price
If extra payment have to be made for a ribbon, it should be specified in this section.

Edit Giftwrap Ribbons
Giftwrap ribbons details can be edited through this section.
Key Fields
Tab: Main Info
Main detail of giftwrap ribbons can be set from this tab.
1. Ribbon name
The name that will be shown for the ribbon while displayed in the cart page
2. Extra Price
If extra payment have to be made for a ribbon, it should be specified in this section.
Tab: Ribbon Images
This section allows to map the images for the ribbon.
Use "Assign More" button in this section to go to the image gallery section to assign the image(s) for the ribbon. Once the images are assigned, option to set the order in which the images will be displayed to the customer and also the title for the images will be displayed. Specify these details and press the "Save Details" button to save the changes. One thing to be noted is that, the image details will be saved only for those images for which the checkbox is ticked.
Use "Un assign" button to unassign any assigned image(s) set for the ribbon.
