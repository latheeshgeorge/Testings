How Hits to Sale Report Works?
Hits to sale report is capable of showing the total clicks occured from each of the search engines against each of the keywords and also the total clicks which have been converted to sales. The total sale amount for each of the keywords from various search engines will also be displayed. The report will be helpful for Search Engine Optimisers to analyse the keyword which have maximum hits.
 
1. List Hits to Sale Report
2. Monthly view
 
List Hits to Sale Report
This section lists the keywords that have brought customers to your website from various search engines. The keywords are listed with their search engines, total clicks it has generated, total no:of sales it has produced etc. Also you can see the sale amount a particular keyword has generated for your website. Click on the Monthly View link to view the total sales produced each month.
Report can be searched based on the keyword and also by selecting the search engines. Combination of keywords and search engines can be used to perform the search. Number of rows to be shown in each of the pages and also the sort order for the report can also be specified. Click on "Go" button to begin the search.
To view the monthly report for a particular entry in the list, click on the "Monthly View" link displayed with each of the rows.

Monthly view
This page lists the Month wise view of total clicks, total sale clicks and sale amount produced each month. The sort order in which the list of months to be displayed can be set in this page. Click on "Go" button to reflect the new sort order.
