List Newsletters
Console users can create any number of newsletters. This page shows the lists of such newsletters. Newsletters can be searched by giving the newsletter title. Number of newsletters to be shown in in each of the pages and also the sort order to be used in the search result can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the newsletter add page. There are 2 ways to go to the edit page of an existing newsletter. One method is by clicking the title of the newsletter and another by tick marking the required newsletter and clicking the "Edit" icon.
Delete operation can be done to more than one newsletters at a time. Tick mark the newsletters to be deleted and click the "Delete" icon.
It is possible to move directly to the Newsletter preview page or Customer type selection page of current newsletter from the listing page. Click on "Preview" link to move to the newsletter preview page. Click on "Send Mail" link to move to the Customer Type selection page.

Add Newsletters
New newsletters can be created through this section.
Key Fields
1. Newsletter Template
This dropdown box lists the newsletter templates existing in the site. If any template is to be used in current newsletter, then select the template from the dropdown box. When a template is selected from the dropdown box, the page will get reloaded to fill respective boxes in the page with the values set in the template.
2. Newsletter Title
The title for the newsletter can be set from this section. Title set for newsletter will be used as the subject of the newsletter mail. If any template is selected, the default title set for the template will be displayed in this textbox.
3. Newsletter Contents
The content to be send as newsletter can be set from this section. if any templates is selected, the default content set for the template will be listed in this textarea. The code that can be placed in thie section are listed below this textarea. The codes should be placed in required positions and it will be replaced with respective values while sending the newsletter.
Click on "Continue" button to save the changes and go to the "Assigned Products" section in the newsletter edit page.

Edit Newsletters
Newsletters can be edited through this section.
Key Fields
Tab: Main Info
Main details for the newsletter can be set from this tab.
1. Current Template
If any template is selected for this newsletter, then the name of that template will be displayed in this section.
2. Do you want to change the template
If a different template is to be selected for current newsletter, then tick mark this. This will cause the page to reload and a drop down box will be displayed in the Current Template section.
3. Newsletter Template
This dropdown box lists the newsletter templates existing in the site. If any template is to be used in current newsletter, then select the template from the dropdown box. When a template is selected from the dropdown box, the page will get reloaded to fill respective boxes in the page with the values set in the template.
4. Newsletter Title
The title for the newsletter can be set from this section. Title set for newsletter will be used as the subject of the newsletter mail. If any template is selected, the default title set for the template will be listed in this textbox.
5. Newsletter Contents
The content to be send as newsletter can be set from this section. if any templates is selected, the default content set for the template will be listed in this textarea. The code that can be placed in thie section are listed below this textarea. The codes should be placed in required positions and it will be replaced with respective values while sending the newsletter.
Click on "Continue" button save the changes and go to the "Assigned Products" section.

Tab: Assigned Products
If any products are linked with current newsletter, then those products will be listed in this section.
Use "Assign" button to assign more products to the current newsletter.
Use "Un Assign" button to unassign the products already assigned to current newsletter.
Click on "Continue" button to proceed to the next step in the newsletter wizard.
Note: If no product are assigned to current newsletter, then a "Skip" button will be displayed. If products are not to be linked with current newsletter, the skip button can be used to skip the product assignment section.

Tab: Preview Newsletter
This section allows to view exactly how the newsletter content and subject will look like when it is send as mail. It is possible to make changes to the title and contents if required. While displaying the content in this section, the codes will be replaced with the respective values. The title and content displayed in the section will be used as the subject and content respectively for the newsletter mail while sending it to customers.
Click on "Save & Continue" to save the details and continue to next step in the newsletter wizard. Click on "Save" button to save the details and go back to newsletter listing page.

Tab: List Customer Types
This section allows to select the type of customers to which the current newsletter is to be send.
There will be 3 types namely
1. Business Customers
This section represents the business customers. The name of the business will be listed in the drop down box. If the newsletter is to be send to all businesses, then select the "All" option from the dropdown box. When a business is selected in the dropdown box, then the list of departments will be listed in another multiselect box. Select the required departments from the list. Use ctrl button to make multiple selection.
Once the selections are made, click on "Proceed" button in the business customer section to proceed to next step in the newsletter wizard.
2. Registered Customers
If the newsletter is to be send to registered customers only, the click on the "Proceed" button in the registered customer section to go to the next step.
3. Newsletter Groups
This section lists the customer newsletter groups available in the site. Tick mark the required newsletter groups to which the newsletter is to be send. If the newsletter is to be send to all newsletter customers, then tick mark the "Send to All Newsletter Customers". Once the selection is made, click on "Proceed" button in this section to continue to the next step in the newsletter wizard.
Tab: List Customers to send the Newsletter
This tab will be displayed only when clicked on "Proceed" button in the Customer Type tab. This tab shows the list of customers based on the selection made in the "Customer Type" step. Customers will be listed in pages. Tick mark the customers to which the newsletter is to be send. It is possible to make selection by navigation to different pages.
If newsletter is to be send to all customers in the list, then there is no need to go and select each of the customers in the list, instead tick mark the "Send to all Customers" checkbox.
Click on "Send Newsletter" button to send the newsletter to customers.
