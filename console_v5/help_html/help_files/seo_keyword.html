How Manage Site Keywords Works?
This section allows the search engine optimisers to set the meta keywords to be displayed for various pages in the website. Maximum of 5 keywords can be set for each of the pages.
Depending on the type of value selected in the "Select Type" drop down box, various section will be displayed with 5 textboxes each. The drop down box have the following values.
 
1. Home
2. Categories
3. Products
4. Shops
5. Combo Deals
6. Shelves
7. Best Sellers
8. Static Pages
9. Saved Search
10. Saved Search Main
11. Help
12. Registration
13. Sitemap
14. Forgot Password
15. Site Reviews
 
Back to Top
1. Home
When this value is selected in the dropdown box, the page will get reloaded and a set of 5 textboxes will be displayed. The keywords set in the given 5 textboxes will be displayed as the meta keywords for the home page in the website. Enter the required keywords in the textboxes and click on "Save Keywords" to save and assign the keywords to the home page. It is not necessary that all the 5 keywords should be specified.
Back to Top
2. Categories
When this value is selected in the dropdown box, the page will get reloaded and the list of all categories existing in the site will be displayed in the page. For each of those categories a set of 5 textboxes will be displayed. The keywords set in the given 5 textboxes for each of the categories will be displayed as the meta keywords of respective category details page in the website. Click on "Save Keywords" to save and assign the keywords to the respective categories. It is not necessary that all the 5 keywords should be specified.
If the category being displayed is a subcategory, an icon will be displayed in the last column of such rows. When mouse is placed on this icon, the hierarchy of that category will be displayed as a tool tip text.
Categories can be searched by giving the category name and also by selecting the parent category. Combination of these 2 can be used to search for categories. Click on "Go" button to begin the search.
