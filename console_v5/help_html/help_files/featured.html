How List Featured Product Works?
Featured products can be set to view in different positions of the site depending on the type of layout used by the site. There can be only one featured product at a time.
Use "Assign" icon to assign a new featured product. When clicked on this icon, a new page will be displayed which shows the list of products existing in the site. The product to be marked as the featured can be selected from the list. If a featured product already exists in the site and again try to assign another product as featured product, then the previous featured product mapping will be removed and the new product will be set as the featured product.
The page will display the details of product which is set as the featured product.
Use "Delete" icon if featured product is no longer need to be displayed in the website.
Key Fields
1. Featured Product Description
It is possible to add a seperate description for featured product. This is in addition to the normal short description for the product. If this featured product description is specified, then the normal product short description will be ignored and the featured description will be displayed for the product in the featured product section.
2. Fields to be shown for featured product
a. Image
Tick mark this if image is to be displayed for the featured product.
b. Title
Tick mark this if product title is to be displayed for the featured product.
c. Short Desc
Tick mark this if short description / featured description is to be displayed for the featured product.
d. Price
Tick mark this if price is to be displayed for the featured product.
3. Image Type
The type of image to be displayed for the featured product can be selected from the drop down box.
4. Display Location
This field allows to select the areas in which the featured product is to be displayed. The list box show the list of layouts and its positions to which the featured product can be assigned. The values in this list box have the format "Layout Name (position)(order)". There can be more than one layout for a site. "Layout name" refers to the layouts existing for the site. "Position" refers to the positions to which featured product can be set for the site (eg. right, left, etc). "Order" is the sort order in which the featured product will be listed in the site. The order is important if more than one component is placed in same position. Components set to each of the positions are displayed in the site in the sort order shown in "Order". Use "ctrl" button to make multiple selection in this list box. By default the value of "Order" will be set to 0. It will be changes automatically when the position of each of the components are set from the Edit Website Layout Section in the console (Settings » Design Settings » Edit Website Layout).
