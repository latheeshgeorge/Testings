How Best Sellers Works?
Products which are to be displayed as best sellers in your website can be set either to pick automatically or manually. If the option automatic is selected, then the best sellers will be displayed based on the actual sales occured in the site. If the manual option is selected, the user can set few products to be shown as the best sellers.

2. Products to be displayed as Bestsellers
 
Tab: Main Info
This tab allows to decide whether the products to be shown as bestsellers in the website need to be picked automatically or manually. The options available are the following.
1. Pick Automatically
If the bestseller products are to be picked automatically, then this option should be selected. The products to be displayed will be picked automatically based on the orders placed in the website.
2. Pick Manually
If the bestseller products are to be picked manually, then this option should be selected. If this option is selected the products to be displayed under bestseller in website should be selected by the console user

Tab: Products to be displayed as Bestsellers
If the option "Pick Manually" is selected, then the products to be displayed under bestsellers in the website can be selected using this section.
The list of products which are already assigned will be listed in this tab.
Click on "Assign More" button to assing more products as bestsellers. When clicked on this button a new page will be displayed, which shows the list of products available in the website. Tick mark the required products and click on "Assign Selected" button to assign those products.
Use "Unassign" button to unassign products which are already assigned as best sellers.
The order in which each of the best seller products to be displayed in the website can be set using the textbox displayed with each of the products in the list. To save the sort order first tick mark the products for which the sort order is to be saved, and then click on "Save Order" button.
