How Shop By Brand Menu Works?
Shop by brand menu can be called as containers for shops. More than one shops can be assigned to a shop by brand menu and this menu can be displayed at various positions in the website in different styles. The various positions can be Top menu, Left menu, Right menu etc. Same shop by brand can be assigned to more than one .

List Shop by Brand Menu
Shop by Brand Menu can be searched by giving the name of the shop by brand menu. Number of shops menus to be displayed in each of the pages and also the sort order to be used can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go to the shop by brand menu add page. There are 2 ways to go to the edit page of an existing shop by brand menu. One method is by clicking the name of rhe shop by brand menu and another by tick marking the required shop by brand menuand clicking the "Edit" icon.
Delete operation can be done to more than one shop by brand menu at a time. Tick mark the shop by brand menus to be deleted and click the "Delete" icon.
Status of shop by brand menu can be changed directly from the listing page. Tick mark the shop by brand menu which need status changing. Select the required status from the drop down box provided at the section "Change Hide Status" and click the "Change" button.
The field "Layout(position)(order)" in the menu lising is shown as a drop down box. This is because same menu can be set to show in more than one layout/position. This drop down box is just for an informative purpose. No operation is associated with this drop down box.

Add Shop by Brand Menu
Shop by brand menu can be added through this section.
Key Fields
1. Hide Product Shop Group Name
This field decides whether the name of the shop by brand menu is to be displayed to the customer. If ticked, then the name will be hidden in the site.
2. Display Location
This field allows to select the positions at which the menu is to be displayed. The list box shows the list of layouts and its positions to which the shop by brand menu can be assigned. The values in this list box have the format "Layout Name (position)". There can be more than one layout for a site. "Layout name" refers to the layouts existing for the site. "Position" refers to the positions to which shop by brand menus can be set for the site (eg. right, left, etc). It is possible to show same shop by brand menu in more than one layout/position. Use "ctrl" button to make multiple selection in this list box.
Note: Depending on the display location set for the shop by brand menu, the shop by brand menu will be displayed in various positions on the site such as top, left, right etc. The name of the shop by brand menu will be displayed as the title for the static page menu by default in the website. In order to change this, you have to use the edit title icon of respective positions from the Edit Website Layout Section in the console (Settings » Design Settings » Edit Website Layout).
3. Shop List Type
Shop by Brand menu can be displayed in 2 formats. First is as "Menu" and the second is as "Drop Down". If the "Menu" option is selected, then all the shops assigned to the shop by brand menu will be displayed one after the other in the website. If the "Drop Down" options is selected, then the assigned shops will be shown in a selectable drop down box in the website.
4. Show in all Pages
Shop by brand menus can be set to show in all pages or only for selected product or shops or static pages. From the "Add" page it is possible only to specify whether it is to be shown for all pages or not. Mapping to certain product or shops or static pages can be done from the shop by brand edit page.
5. Change Active Period
If shop by brand menus are to be displayed only during a specified range of dates in the website, then it can be done by tick marking this check box. When this checkbox is ticked, two new textboxes will be displayed which allows to specify the start and end dates within which the shop by brand menu is to be displayed in the website.

Edit Shop by Brand Menu
Shop by brand menu details can be edited through this section.
Key Fields
Tab: Main Info
Main shop by brand menu details can be set from this tab. Use "Save" button to save the changes. Details should be saved before switching the tabs.
1. Hide Product Shop Menu Name
This field decides whether the name of the shop by brand menu is to be displayed to the customer. If ticked, then the name will be hidden in the site.
2. Display Location
This field allows to select the positions at which the menu is to be displayed. The list box show the list of layouts and its positions to which the shop by brand menu can be assigned. The values in this list box have the format "Layout Name (position)(order)". There can be more than one layout for a site. "Layout name" refers to the layouts existing for the site. "Position" refers to the positions to which shop by brand menus can be set for the site (eg. right, left, etc). "Order" is the sort order in which the shop by brand menuwill be listed in the site. The order is important if more than one component is placed in same position. Components set to each of the positions are displayed in the site in the sort order shown in "Order". It is possible to set the same shop by brand menu to more than one area. Use "ctrl" button to make multiple selection in this list box. By default the value of "Order" will be set to 0. It will be changed automatically when the position of each of the components are set from the Edit Website Layout Section in the console (Settings » Design Settings » Edit Website Layout).
Note: Depending on the display location set for the shop by brand menu, the shop by brand will be displayed in various positions on the site such as top, left, right etc. The title for the shop by brand menu will not get modified in the website even when the menu name is changed from the edit page. In order to change the title you have to use the edit title icon of respective positions from the Edit Website Layout Section in the console (Settings » Design Settings » Edit Website Layout).
3. Shop List Type
Shop by Brand menu can be displayed in 2 formats. First is as "Menu" and the second is as "Drop Down". If the "Menu" option is selected, then all the shops assigned to the shop by brand menu will be displayed one after the other in the website. If the "Drop Down" options is selected, then the assigned shops will be shown in a selectable drop down box in the website.
4. Show in all Pages
Shop by brand menus can be set to show in all pages or only for selected product or shops or static pages. From the "Add" page it is possible only to specify whether it is to be shown for all pages or not. Mapping to certain product or shops or static pages can be done from the shop by brand edit page.
5. Change Active Period
If shop by brand menus are to be displayed only during a specified range of dates in the website, then it can be done by tick marking this check box. When this checkbox is ticked, two new textboxes will be displayed which allows to specify the start and end dates within which the shop by brand menu is to be displayed in the website.

Tab: Shops in this Menu
Shops which are mapped to the shop by brand menu being editing will be show in the area.
To assign more shops to the menu, click on "Assign more" button under this section. This will open up a page which shows the list of shops which are not yet mapped to the current menu. Tick the shops to be mapped and click the "Assign Selected " button to make the assignment.
To unassign shops from the menu, tick the shops to be unassigned and click the "Unassign" button in this section.
While listing the shops mapped with the menu, it is possible to set the order in which the shops are to be displayed in the menu. Specify the order in the textbox provided with each of the shops assigned to the menu. Click on "Save Order " button to save the changes.

Tab: Display menu for following categories
If the menu is to be shown while viewing the details page of certain caregories only, then those categories should be set from this section. This setting will be ignored if the "Show in all pages" option is set for the menu.
Use "Assign More" button to assign more shops to this section
Shops assigned to this section can be unassigned by tick marking the required shops and clicked the "Unassign" button in this section.
