How Newsletter Customers Works?
This section shows the list of newsletter customers who are registered for newsletter subscription. This include both registered users and non registered users. Newsletters can be subscribed from the newsletter subscription section in the site or can subscribe from the customer registration or edit profile page. It is not compulsory that newsletter customers should be the registered customers of the site.
While registering or profile edit, the list of newsletter groups will be displayed to the customer. If they tick mark any of the newsletter groups, then those customer will be automatically displayed in the newsletter customer listing section.
 
1. List Newsletter Customers
2. Add / Edit Newsletter Customers
 
List Newsletter Customers
Newsletter customers can be searched based on customer name and customer email. Number of customers to be shown in each of the pages and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the newsletter customer add page. There are 2 ways to go to the edit page of an existing newsletter customer. One method is by clicking the customer name and another by tick marking the required customer and clicking on the "Edit" icon.
Delete operation can be done to more than one newsletter customers at a time. Tick mark the customers to be deleted and click the "Delete" icon.
Hidden Status of newsletter customers can be changed directly from the listing page. Tick mark the customers which need status changing. Select the required status from the drop down box provided at the section "Change Status" and press the "Change" button.
The column "registered" in the newsletter customer listing section denotes whether the customer is registered customer of site or not.

Add / Edit Newsletter Customers
Newsletter Customers can be managed through this section.
Key Fields
Personal Details
Customer title, name and email are compulsory fields.
Newsletter customers can be deactivated by ticking the "Hide" checkbox.
Select the Newsletter Group
The list of newsletter group will be displayed in this section. Subscription to a particular newsletter group can be done by ticking it. This section will be visible only if newsletter groups added to the site from console area.
Newsletter groups can be manged for the site from the location Settings » Manage Newsletters » Newsletter Group in the console.
