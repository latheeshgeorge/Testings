How Customer Registration Company Types Works?
The company types which are to be displayed in the customer registration / edit profile page can be set from this section.
 
1. List Company Types
2. Add/Edit Company Types
 
List Company Types
Company types can be searched by giving the name of the type. Number of company types to be shown in each of the page and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the company types add page. There are 2 ways to go to the edit page of an existing company type. One method is by clicking the type name and another by tick marking the required company type and clicking the "Edit" icon.
Delete operation can be done to more than one company types at a time. Tick mark the company types to be deleted and click the "Delete" icon.
Status of company types can be changed directly from the listing page. Tick mark the company types which need status changing. Select the required status from the drop down box provided at the section "Change Status" and press the "Change" button. If a company type is marked as hidden, then it will not be displayed in the website.
Order in which the company types are to be displayed in the registration or edit profile page can be set by giving sort order values in textbox given for each of the company types in the list. Click the "Save Order" button to save the sort order.

Add/Edit Company Types
Company types can be managed through this section.
Key Fields
1. Type Name
The name of the company type should be set in this textbox.
2. Company Type Order
Order in which the company types are to be displayed in the registration or edit profile page can be set in the textbox.
Click on "Save" or "Update" button to save the details.
