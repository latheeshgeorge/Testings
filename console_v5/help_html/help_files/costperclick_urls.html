How Cost Per Clicks Works?
Cost per click section allows to keep track of the paid links placed in third party sites(Eg. Sponsored Links in GOOGLE). The site administrator is liable to make a payment for the clicks coming from such third party sites.

In order to track the clicks that are coming from such paid links placed in third party websites, first an entry should be added in the cost per click section in the console of your site. An entry can be created with details like Keyword to be advertised, where you are advertising, your website url where visitors to be landed when clicked from the sponsored links, cost per click which you have to pay for each click you are getting and the fraud click rule. Once the entry is created, the url to be placed as link in the third party site will be displayed in the edit page of that cost per click entry. Copy this URL and publish it in the thrid party website.

When a visitor clicks on your keyphrase from the thrid party website, she/he will be initially taken to this URL first and which in turn will take the visitor to the actual intended page(f.e product or category) in your website after tracking the IP address and checking the fraud click rule you have set. If the system recognize a click as a fraud click that is repeatedly coming from the same IP address then an appropriate fraud detection message will be displayed to the visitor.
List Cost Per Clicks
Cost per click entries can be searched by giving any textual data and / or by selecting the keyword or advertised location. Number of rows to be shown in each of the page and also the sort order to be used can be specified. Use "Go" button to begin the search operation.
Use "Add" link to goto the cost per click add page. There are 2 ways to go to the edit page of an existing cost per click details. One method is by clicking the keyword and another by tick marking the required cost per click and clicking the "Edit" icon.
Delete operation can be done to more than one cost per click entry at a time. Tick mark the entries to be deleted and press the "Delete" icon.
Status of cost per click entries can be changed directly from the listing page. Tick mark the entries which need status changing. Select the required status from the drop down box provided at the section "Change Status" and press the "Change" button.
Key fields
1. Keyword
This is the keyword which is advertised in the third party site.
2. Third party site
The detail of third party site on which the advertisement or link is placed.
3. Cost Per Click
The price to be paid to the third party site for each click.
4. Genuine Clicks
The total number of genuine clicks happend till now from the third party site for the current cost per click entry.
5. Fraud Clicks
The total number of fraud clicks happend till now from the third party site for the current cost per click entry.
6. Sale Amount
The total sale amount due to click from the third party site for the current cost per click entry till now.
7. Total Cost Per Click
This value is obtained by multiplying Cost Per Click with the value in Genuine Clicks field.
Click on "Monthly view" link to view the monthly view details of a particular cost per click entry.

Cost Per Click Monthly Report
This page shows the monthly report of a selected cost per click entry. The page shows the details of selected cost per click entry and also the monthly details of selected cost per click entry.
1. Date
The month and year of the cost per click details.
2. Total Genuine Clicks
The total number of genuine clicks happend for each month from the third party site for the current cost per click entry.
3. Total Fraud Clicks
The total number of fraud clicks happend for each month from the third party site for the current cost per click entry.
4. Total Sale Amount
The sale happend due to click from the third party site for the current cost per click for each month.
5. Total Cost Per Click
This value is obtained by multiplying Cost Per Click with the value in Total Genuine Clicks for respective month.
6. Genuine Clicks
When clicked on this link, a new page will be displayed which show the list of genuine clicks occured during the selected month.
6. Fraud Clicks
When clicked on this link, a new page will be displayed which show the list of fraud clicks occured during the selected month.
Search can be performed on the page based on particular month and year.
If the cost per click report for the selected month is to be exported, then click on "Export" button.

Add Cost Per Click
Cost per click details can be added through this section.
Key Fields
1. Keyword
The keyword for which the advertisement is being placed at the third party location should be selected from the drop down box. If the required keyword is not there in the list, then select the "- Other -" option. This will show up a new textbox where the new keyword can be specified. When the cost per click details are saved, then the new keyword will also be saved so that it will get displayed in the dropdown list from next time onwards.
2. Third Part Site
This is the third party location at which the advertisement / link is being placed. If the required loaction is not there in the list, then select the "- Other -" option. This will show up a new textbox where the new location can be specified. When the cost per click details are saved, then the new location will also be saved so that it will get displayed in the dropdown list from next time onwards.
3. My Page URL
The exact url of the page in your website to which the customers is to be redirected, when coming from the advertised location. This should be the full path to the destination page.
4. Cost Per Click
Cost that is being charged by the advertised location for each click coming from their site should be specified in this textbox.
5. Fraud Check Rules
Since payment is to be made for each click, tracking should be made to identify fraud clicks. This section allows to define rules to trace out genuine and fraud clicks.
This section allows to specify the maximum number of clicks that can occur from same advertised location against current cost per click entry.
The rule is Allow <x> Clicks in <y> Days. This means maximum of "x" number of clicks happening in "y" number of days will be treated as genuine clicks and all click which violates this condition will be treated as fraud clicks.
When the cost per click details are saved, the system will generate a url. This is the url which is to be placed as the link in the advertised location.
