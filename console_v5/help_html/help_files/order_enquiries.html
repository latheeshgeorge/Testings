How Order Enquiries Works?
Enquiries posted by customers, releated to their orders, can be managed from this section.
 
1. List Order Queries
2. List Order Query Details
2.1. Query Details Tab
2.2. Post Details Tab
 
Note: When a new enquiry is posted by the customer, a notification will be displayed in the console home page. When clicked on this notification, console user will be taken to the listing page of order enquiries.
When a new enquiry is posted from the website by the customer, its status will be "New" by default. When details of an enquiry is viewed by the console user, if it's current status is "New", then the status will be automatically changed to "Read".
Console user can write replies for the customer enquiries by writing posts under enquiries. It works like a ticketing system. All issues related to a particular enquiry should be communicated via posts. If customer wish to address any new new issue or topic, then they should post it as a new enquiry.
Enquiries can be added only by website customers. Console users can respond to queries using posts.
List Order Queries
Order queries can be searched based on order id, status and date ranges. Any combination of these can be used as search criteria. Number of enqueries to be shown in each of the pages and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
By default enquiries with status as "New" only will be listed.
Enquiry details can be viewed by clicking the enquiry date in the list.
Delete operation can be done to more than one enquiries at a time. Tick mark the enquiries to be deleted and click the "Delete" icon.
Status of enquiries can be changed directly from the listing page. Tick mark the enquiries which need status changing. Select the required status from the drop down box provided at the section "Change Status" and click the "Change" button.

List Order Query Details
The details of a selected order query and the lists of posts added against the selected order query can be managed from this section.
Tab: Query Details
The details of order query can be viewed from this section. The Query title, the actual query, date on which it has been added and also the customer who posted the query will be displayed. The status of query will be displayed in the drop dow box.
It is possible to change the status of order query from this page. Select the required status in the dropdown box and click on "Go" button.
Back to Top
Tab: Post Details
The list of posts added against the current order query will be listed in this section. This include both the posts added by the customer as well as those added by console user.
Posts can be searched by giving the range of dates. The number of posts to be displayed in each of the pages and also the sort order for the post can be specified in the search criteria. Click on "Go" button to begin the search.
In the column "Added by" in the list, the details such as name of the person who added the post and also whether it is a customer or a console user will be specified.
In order to view the details of a post, click on the "Details" text displayed with each of the rows in the list. This will display a new row below the clicked row with the details of the selected post.
