Size Chart Headings
Size chart headings sections allows to define the headings that will be used in the size chart section in various products of the site. This is a common area where the size chart headings will be managed. In the product edit page there are options to assign the headings to be linked with the respective product and also to enter the values required under each of the headings.
List Product Size Chart Headings
Product size chart headings can be searched by giving the name of the heading. Number of headings to be shown in each of the page and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the size chart heading add page. There are 2 ways to go to the edit page of an existing size chart heading. One method is by clicking the heading name and another by tick marking the required heading and pressing the "Edit" icon.
Delete operation can be done to more than one size chart heading at a time. Tick mark the headings to be deleted and press the "Delete" icon.
Status of size chart headings can be changed directly from the listing page. Tick mark the headings which need status changing. Select the required status from the drop down box provided at the section "Change Status" and press the "Change" button. If a heading is marked as hidden, then it will not be displayed in any of the products.
Order in which the headings should be displayed in the heading assigning page of products can be set by giving sort order values in textbox given for each of the headings in the list. Click the "Save Order" button to save the sort order.
Add/Edit Size Chart Headings
Size chart headings can be added/edited through this section.
Key Fields
1. Size Chart Heading
The headings for the size chart
2. Sort Order
Order in which the size chart headings will be displayed in the size chart heading assigning section for products.
