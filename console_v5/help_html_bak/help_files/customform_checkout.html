Custom Checkout Form

Custom checkout form allows to define additional fields that need to be displayed in the checkout page. To add custom fields, first a section should be created. Section is the container for the additional fields. Once the section is created, the fields to be added to the section should be defined.
List Checkout Form Sections
Custom checkout sections can be searched by giving the name of the section. Number of sections to be shown in each of the page and also the sort order to be used in search result can be specified. Use "Go" button to begin the search operation.
Use "Add" link to go the section add page. There are 2 ways to go to the edit page of an existing section. One method is by clicking the section name and another by tick marking the required sections and pressing the "Edit" icon.
Delete operation can be done to more than one sections at a time. Tick mark the sections to be deleted and press the "Delete" icon.
Status of sections can be changed directly from the listing page. Tick mark the sections which need status changing. Select the required status from the drop down box provided at the section "Change Status" and press the "Change" button. If a section is marked as hidden, then it will not be displayed in the checkout page.
Page to define the fields to be included in a secion can be obtained by clicking the icon to the right hand side of each of the sections in the listing page.
Add Checkout Form Sections
Checkout form sections can be added through this section.
Key Fields
1. Section Name
This is the name that will be used to identify the section. This name will be displayed as heading for the section while displaying the section in the checkout area.
2. Position
Position decides the exact area where the section and its associated fields will be displayed. The values available for this field are
1. Top
Section which have position set to "Top" will be shown above the common billing address fields in checkout section as a seperate block. The heading for the block will be the section name itself.
2. Top with in the Static Section
Section which have position set to "Top with in the Static Section " will be shown above the common billing address fields in checkout section. The fields will not be displayed as a seperate block; instead they will be placed just above the normal billing address fields. The section name will not be displayed in this case.
3. Bottom within the Static Section
Section which have position set to "Bottom with in the Static Section " will be shown below the common delivery address fields in checkout section. The fields will not be displayed as a seperate block; instead they will be placed just below the normal delivery address fields. The section name will not be displayed in this case.
4. Bottom
Section which have position set to "Bottom" will be shown below the common delivery address fields in checkout section as a seperate block. The heading for the block will be the section name itself.
3. Only for specific products
This option allows to decide whether a custom checkout section should be displayed every time a customer checks out or only if certain products are included in the order. If the checkout section is to be displayed only if certain products are there in the order, then select "Yes" for this option. If "No" is selected, then the section will be displayed everytime the checkout page is viewed. If "Yes" is selected, the option to select the products will be shown in the edit page of the section.
4. Help Instructions
While displaying a section in checkout page, if any description or help message is to be included, then it should be specified in this field. The help instruction will be displayed only for those sections for which the positions are set as "Top" or "Bottom". This description or help will be displayed as the first thing in the section block; above the fields in the section.

List Forms
The fields that are required in each of the sections are defined from this area. The fields that are already set will be shown in a list. The sort order for the fields can be set using the textbox shown with each of the existing fileds. In order to make the field as compulsory, tick mark the mandatory checkbox. The error message to be shown if the mandatory field if left blank can be specified when clicked on the "ErrorMsg" button. If mandatory checkbox is ticked and if error message is not specified, then the field will be considered as mandatory in the checkout area.
Field details can be edited by tick marking the field and clicking on the edit icon.
One or more fields can be deleted by marking the fields to be deleted and clicking the delete icon.
New fields can be added by click respective button in this listing page. The available buttons are
1. Add Textbox
This button show a popup page which allows to specify the details for the textbox to be added to the form.
Key Fields
Label
The caption to be shown for the textbox should be specified here.
Horizontal Align
The horizontal alignment for the textbox can be set from here.
Vertical Align
The vertical alignment for the textbox can be set from here.
Size
The size of textbox can be set from here.
Character Limit
Maximum number of characters allowed for the textbox can be set from here
Error Message
The error message to be displayed to the customer, in case if the field is compulsory.
